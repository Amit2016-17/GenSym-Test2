Gensym Workspace View Control Example

This example demonstrates the use of workspace view controls.


Notes/requirements:

  • In this example, the (invisible) G2 ActiveX Link control connects to host "localhost" at TCP/IP port 1112. The workspace view controls each use the G2Gateway object to help them communicate with G2. Before running this example, you should run G2 on the same computer as your browser. Make sure G2 uses port 1112. You can see which port is being used by running the command Main Menu -> Miscellany -> Network Info. Alternatively, change the host and port settings specified in the VB Script code in this HTML document so that they specify the desired G2 host and port.
  • The WorkspaceView example kb should be loaded into G2 prior to viewing this example web page. That KB contains the two workspaces that are to be called up in this example.
  • This example uses ActiveX, and therefore needs to run on Microsoft's Internet Explorer web browser.
