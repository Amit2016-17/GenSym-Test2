Telewindows ActiveX Control Installation

This HTML page can be used to easily install Gensym's Telewindows ActiveX control on your computer. It will install the control from the cabinet file (TwControl.cab) that you installed as part of Telewindows version 7 or greater. After installing the control, it will attempt to connect to a G2 running on this machine at port 1111.


If you don't see G2 running, click here.
