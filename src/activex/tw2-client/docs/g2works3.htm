Methods

AddWorkspace()

 The AddWorkspace() method initializes selection scroll states, color tables, and the event handler, sets up a listener to the workspace in G2, and displays the workspace view in Visual Basic.

Syntax

    AddWorkspace(Workspace As G2Item,
       [    UpdateWhenNotShown As Boolean])
    Arguments Description
    Workspace As G2Item
    		The workspace in G2 you want to display.
    UpdateWhenNot
    Shown     As Boolean
    		Overides the UpdateNewWorkspaces property for the specified workspace. If true, specifies that the hidden workspace be updated continuously and kept synchronized with the G2 server. If false, specifies that the hidden workspace be updated only when shown.

Example

 In the following code example, the click event passes to the Visual Basic method showWorkspace the name of the workspace the user has selected for display. Inside the Visual Basic method showWorkspace, the GetNamedItem method of G2UiGateway gets the workspace from the G2 server. The AddWorkspace method adds the workspace to a FrmT2Workspace form, a chld form. The ShowWorkspace method makes the workspace visible in the workspace panel. The FrmT2Workspace form's caption is also updated.

    Private Sub Command1_Click()
       Dim WorkspaceName As String
     
       WorkspaceName = List1.Text
       Unload Me
       showWorkspace (WorkspaceName)
    End Sub
     
    Private Sub showWorkspace(name As String)
       Dim NewWorkspace As New FrmT2Workspace
       Dim G2Workspace As g2Item
     
       ' Get the workspace and put it in the panel
     
       Set G2Workspace = G2Connector.GetNamedItem(name, "kb-workspace")
       Call NewWorkspace.Panel.AddWorkspace(G2Workspace, True)
       Call NewWorkspace.Panel.showWorkspace(G2Workspace)
     
       ' Now update the dialog
       NewWorkspace.Caption = name & " (" & G2Connector.G2Location & " )"
       NewWorkspace.Tag = G2Connector.G2Location
       NewWorkspace.Show
    End Sub
For an illustration of displayed workspace, see Viewing and Using Workspaces.

ClearSelection()

The ClearSelection() method clears the selection on the current workspace.

Syntax

    ClearSelection()

DeleteSelection()

 The DeleteSelection() method deletes the items corresponding to each of the elements in the selection on the current workspace.

Syntax

    DeleteSelection()

GetSelection()

 The GetSelection() method gets a list of currently selected workspace elements.

Syntax

    GetSelection() As G2Item

MoveSelection()

 The MoveSelection() method moves the current selection by the current deltas.

Syntax

    MoveSelection(deltaX As Long, deltaY As Long, enlargeWorkspace As Boolean)
    Arguments Description
    deltaX As Long
    		The number of pixels to move along left to right. A negative number causes the selection to move left.
    deltaY As Long
    		The number of pixels to move along up or down. A negative number causes the selection to move down
    enlargeWorkspace
    As Boolean
    		If true, specifies that the workspace resize as it moves. If false, specifies that the workspace move without resizing.

NextWorkspace()

 The NextWorkspace() method displays the next workspace if it is listed in the history buffer.

Syntax

    NextWorkspace()

PreviousWorkspace()

 The PreviousWorkspace() method displays the previous workspace if it is listed in the history buffer.

Syntax

    PreviousWorkspace()

RemoveWorkspace()

 The RemoveWorkspace() method deletes the specified workspace from the workspace view.

Syntax

    RemoveWorkspace(Workspace As G2Item)
    Argument Description
    Workspace As G2Item
    		The workspace you want to remove from the workspace panel.

SelectAll()

 The SelectAll() method adds all of the workspace elements to the current selection.

Syntax

    SelectAll()

ShowWorkspace()

 The ShowWorkspace() method makes the specified KB workspace visible in a workspace panel.

Syntax

    ShowWorkspace(Workspace As G2Item)
    Argument Description
    Workspace As G2Item
    		The name of the workspace you want to want to make visible.

Example

 For an example, see AddWorkspace().
