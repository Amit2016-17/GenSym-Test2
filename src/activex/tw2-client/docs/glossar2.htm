C

 child form: A form that is displayed entirely inside its parent form in a Multiple-Document Interface.

Component Object Model (COM): A standard by which objects access each other's services through well-defined public interfaces.

container: An application that holds an ActiveX control. The control works inside the application, such as a Telewindow2 Toolkit control.
