Introduction

 The G2Item is an object that represents an item in G2. The G2Item object enables an ActiveX application to:

  • Get and set the value of an item in G2

  • Call a method belonging to an item in G2

  • Access the elements of an array if the item is an array or list type in G2

G2Item and G2ItemHolder are similar but have the following differences:

  • G2Item can be passed as an argument to a G2 procedure or returned from G2.

  • G2ItemHolder is an ActiveX control that can be placed in an ActiveX container, such as a Visual Basic form. It can also notify your ActiveX application when an attribute value changes and refer to an item in G2 by name.
