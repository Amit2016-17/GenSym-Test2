Introduction

 Telewindows2 Toolkit for ActiveX maps data types between Visual Basic and G2. Most G2 data types correspond to data types in Visual Basic, except for symbol and structure. To pass data as a symbol or structure to G2, you can use the G2Symbol and G2Structure classes.

The following table lists G2 data types and the corresponding Visual Basic data types.

G2 Data Type Visual Basic Data Type
truth-value
Boolean
quantity
Variant
integer
Long
float
Double
text
String
item-or-value
Variant
value
Variant
sequence
MyData() As Variant
the symbol null
Null

Note: The range of an integer in G2 is -229 to (229 - 1). The range of a long in Visual Basic is-2,147,483,648 to 2,147,483,647. The range of a float in G2 is ±1.79 x 10308 to ±2.22 to 10-308. The range of a double in Visual Basic is ±1.79769313486231308 to ±4.94065645841247-324.
