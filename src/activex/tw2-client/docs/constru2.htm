Adding the Telewindows2 Toolkit Controls to the Toolbox

 You can place Telewindows2 Toolkit controls on Visual Basic forms and reference Telewindows2 Toolkit objects in your code. Before you can do either, you must add the controls to the Visual Basic toolbox.

To add the Telewindows2 Toolkit controls to the toolbox:

  1. Create or open a standard project in Microsoft Visual Basic.

  2. Choose the Components option in the Project menu in Visual Basic to display the Components dialog, as shown in the following figure.


  3. Locate the "Gensym Telewindows2 Toolkit" entry in the list on the Controls page and click the box to select the control, as shown.

  4. Click OK on the Components dialog.

In the Visual Basic toolbox, you see new icons for the Telewindows2 Toolkit controls. The controls are:

  • G2UiGateway


  • G2ItemHolder


  • G2WorkspacePanel


Using a Control in a Form

 You can place a control directly on your form or you can create an instance of the control programmatically in Visual Basic code.

When placed on a form, an instance of a Telewindows2 Toolkit object appears in the Properties window as "G2UiGateway," as shown in the following figure.


If you have more than one of each type Telewindows2 Toolkit control in your project, the number for each of the subsequent controls automatically increments; for example, G2UiGateway2, G2UiGateway3, G2UiGatewayn. You can, however, rename the instances to whatever you want.

To place a Telewindows2 Toolkit control directly on your form:

  1. Click, for example, the G2UiGateway icon in the Visual Basic toolbox.

  2. Place the cursor over the form in the form designer.

  3. Press and hold the mouse button and drag the crosshair cursor down and to the right over the area of the form where you want to place the control.

  4. Release the mouse button and the control appears as a connector icon.


When you run your form, the control is invisible and it may be located under other controls on the form.

Note: To place the control directly on the form, double click on the icon. Visual Basic places it in the middle of the form. For more information on building a form, see Setting the Connection Properties.

Creating a Control Programmatically

You can use a Telewindows2 Toolkit control programmatically by referencing it in your code. The following code fragment shows how you can create and define a variable for the G2UiGateway control, create an instance of the control, and use it to start a G2 procedure in a running G2.

Dim myCTL As G2UiGateway
Set myCTL = New G2UiGateway
myCTL.Start "G2PROC-ONE", StartItem.Text
Note: Make sure that Gensym Telewindows2 Toolkit is referenced by Visual Basic. To bring up the Available References dialog, click References in the Project menu.

Like other controls in Visual Basic, a Telewindows2 Toolkit control has properties you can set. You can set them by clicking on the control in the form to display the Properties window, as shown in the following figure.
