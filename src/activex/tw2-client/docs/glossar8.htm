P

 parent form: In a Multiple-Document Interface, the form that contains the documents, such as windows displaying G2 workspaces.

properties: A set of attributes concerning the internal state of a Java Bean. For example, foreground color or animation rate. By modifying properties, you can customize the appearance and behavior of a Java Bean to meet your application requirements.
