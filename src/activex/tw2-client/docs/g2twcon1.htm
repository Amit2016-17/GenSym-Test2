Introduction

 G2TwConnector enables you to create a connection to G2 and to manage the connection. You can:

  • Connect and disconnect from G2

  • Call or start procedures in G2

  • Get items from G2 and create instances of items in G2

  • Apply changes to the instances of a class

  • Detect when a user logs in and out

  • Detect G2 state changes

  • Detect the current G2 user mode
