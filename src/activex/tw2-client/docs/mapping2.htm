Passing a G2 Symbol

 The G2Symbol() class enables you to pass G2 symbols as arguments in methods in Visual Basic.

Property

Property Description
Value As String
The name of the G2 symbol to be passed as an argument.

Example

The following code fragment creates a new G2 symbol, whose value you can pass to the G2 server.

    Private Sub Command1_Click()
       Dim Status As G2Symbol
     
       Set Status = New G2Symbol
       Status.Value = "on-line"
       Item1.Attribute("Node status") = Status
To create G2 symbols inline, see Symbol().

Method

The String() method enables you to display the text of the G2 symbol in your application.

Syntax

String() As String
 The string you can display is the contents of the Value property.

Example

The following code fragment displays the value of the G2 symbol in a form's caption.

    Private Sub Command1_Click()
       Dim Status As G2Symbol
     
       Set Status = New G2Symbol
       Status.Value = "on-line"
       Caption = Status.String
