Introduction

 A G2WorkspacePanel is an interactive, visual representation of a workspace in G2, displayed much like it would be in classic Telewindows, except that the user interface is Microsoft Windows.

You can respond to data changes in a workspace displayed in a G2WorkspacePanel, including changes to elements inside the workspace.

The following figure shows the G2WorkspacePanel dropped on a form in Visual Basic. The control is resized to an area that can initially hold a significant amount of a G2 workspace, although the panel can be resized once it is displayed.
