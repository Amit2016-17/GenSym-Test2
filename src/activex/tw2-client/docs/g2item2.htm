Properties

Design Time Property

 You can set the CallTimeout property at design time in the properties window or programmatically.

Property Description
CallTimeout As Long
Sets the maximum amount of time for the Telewindows2 Toolkit to wait for G2 to respond. The default is 30 seconds.

Runtime Properties

You can use the Attribute and Element properties at runtime, not design time. Once a connection to G2 is established, you can programmatically access the values of these properties of the connected object

Property Description
Attribute(AttributeName As String)
Enables you to access the value of the named attribute.
AttributeNames As Variant
A read-only array of strings composed of attribute names for the G2 item.
Element(Index As Long)
Access an element in a list or array.
ElementCount As Long
Specifies the number of elements in a list or array. It is read only.
Elements As Variant
Specifies all the elements in a list or array
ElementType As Long
Specifies the enumerated G2 type in a list or array. It is read only. The type is one of the following:
  • Const g2Integer = 1
  • Const g2Symbol = 3
  • Const g2Text = 4
  • Const g2TruthValue = 5
  • Const g2Float = 6
  • Const g2Item = 7
  • Const g2Value = 8
  • Const g2ItemOrValue = 9
  • Const g2Quantity = 10

ManuallyDisabled As Boolean
Informs you whether the item in G2Item has been manually disabled in G2. It is read-only.
Transient As Boolean
Informs you whether the item in G2Item is transient in G2. It is read-only.
