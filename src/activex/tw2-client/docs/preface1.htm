About This Guide

 This guide describes the properties, methods, and events of the Telewindows2 Toolkit for ActiveX product, which enables you to connect to a running G2 and display workspaces in an ActiveX container, such as Microsoft Visual Basic. This guide uses Visual Basic to build an example application that connects to G2.
