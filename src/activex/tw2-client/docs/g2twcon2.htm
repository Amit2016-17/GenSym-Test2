Properties

 G2UiGateway has the following properties, which you can set in a container application. Properties configure the behavior of the G2UiGateway and indicate its status. Most container applications save the state of the properties in your document and restore them when you load the document. The properties can be modified either by using the Property window or programmatically.

Property Description
CallTimeout As Long
Sets the maximum amount of time for the Telewindows2 Toolkit to wait for G2 to respond. The default is 30 seconds.
ConnectAtStart As Boolean
Specifies that the connection to G2 be established at startup. When ConnectAtStart is set to true, Telewindows2 Toolkit does require an explicit call to the Connect() method.
G2InterfaceClassName As String
Specifies the interface class in G2. G2 creates an instance of this class when Telewindows2 Toolkit connects with G2. The instance represents the connection. The default is ui-client-interface. This property must name a class that is defined in G2 and is a subclass of ui-client-interface. You can add attributes to the subclass of ui-client-interface or write rules or procedures that act on the subclass rather than the ui-client-interface class.
G2Location As String
Sets the host machine name and port number of the G2 server to which Telewindows2 Toolkit connects. The format is "hostname:port_number" and the default is "localhost:1111," which is the network address of a G2 server running on the same system as Telewindows2 Toolkit at port 1111, the default G2 address.
G2SoftwareVersion As String
Returns the version number of G2, such "G2 version 5.1 Rev.0." It is read-only.
G2State As G2States
The current G2 state is one of the following constants:
  • Const g2Reset = 0
  • Const g2Paused = 1
  • Const g2Running =2
  • Const g2Resumed = 3

GatewayId As String
The name of the connection to G2.
RemoteInitializationString As String
Identifies the connection and is stored in the Remote Process Initialization String attribute of the G2 Gateway interface object, created in G2 when Telewindows2 Toolkit connects with it. G2 applications use this string to identify the interface, especially when there is more than one client connection to G2.
UserMode As String
Specifies the G2 user mode, for example, administrator.
UserName As String
Specifies the name of the user who is opening a connection with G2.
