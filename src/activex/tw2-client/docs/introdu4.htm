The Controls of Telewindows2 Toolkit for ActiveX

 The Gensym Telewindows2 Toolkit control consists of four constituent controls for:

  • Connecting to G2

  • Managing items between an application shell and G2

  • Displaying workspaces

Once added to your Visual Basic project, three Telewindows2 Toolkit controls appear in the Visual Basic toolbox.

G2UiGateway

The G2UiGateway control enables you to establish a connection between Visual Basic and G2. It also enables you to request login information from users. You can also keep track of the current G2 state, KB, and workspace.

For more information, see Chapter 5, G2UiGateway.

G2Item and G2ItemHolder

The G2ItemHolder control enables you to manipulate G2 items, including workspaces. You can set and change attributes, as well as call or start procedures.

For more information, see Chapter 6, G2Item and Chapter 7, G2ItemHolder.

G2WorkspacePanel

The G2WorkspacePanel control enables you to display workspaces, and to select, move, or resize items on a workspace.

For more information, see Chapter 8, G2WorkspacePanel.
