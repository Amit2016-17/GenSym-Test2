Properties

 G2WorkspacePanel has the following properties, which you can set in a container application. Properties configure the behavior of G2WorkspacePanel and indicate its status. Most container applications save the state of the properties in your document and restore them when you load the document. The properties can be modified either by using the Property window or programmatically.

Property Description
ShowNewWorkspaces As Boolean
If true, specifies that a workspace be shown when it is added or retrieved. If false, specifies that a workspace is not shown when added or retrieved.
UpdateWhileHidden As Boolean
If true, specifies that a hidden workspace be updated continuously and kept synchronized with the G2 server. If false, specifies that a hidden workspace be updated only when shown.
WorkspaceAddress As String
A reference to a particular workspace in named connection to G2 in the format: Workspace@GatewayId The named connection to G2 is the GatewayId property of the G2UiGateway object.
