Introduction

 The G2ItemHolder is an ActiveX control that can be placed in an ActiveX container, such as a Visual Basic form. You can configure it to refer to a specific item in G2 by name. Your ActiveX application can request that G2 notify it when the item's attributes change.

G2ItemHolder contains a G2Item internally. You can assign a new G2Item to a G2ItemHolder at any time.
