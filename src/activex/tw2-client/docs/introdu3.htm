The Telewindows2 Toolkit Default Application Shell

 The Telewindows2 Toolkit default application shell is a sample program in Microsoft Visual Basic with menus that enable you to communicate with G2. The application contains forms for connecting and logging in to a running G2, and for displaying G2 workspaces in the sample application.

The default application shell can be used to access existing G2 applications and serves as an example of how to build Visual Basic applications to access a G2 server by using Telewindows2 Toolkit. You can:

  • Connect to and login to one or more running G2s.

  • Set the run state of a connected G2: Pause, Reset, Restart.

  • View and print G2 workspaces.

  • Perform some operations on an item on a workspace, such as Select and Move.

  • Minimize, maximize, or close the window displaying a G2 workspace.

Choosing Open Connection causes the display of the Open G2 Connection form, in which you can specify the G2 to connect with and login information.


Viewing and Using Workspaces

You can display a window containing a G2 workspace in the default application shell by using the menu choice:

    File > Get Workspace

Inside the application shell appears a new window containing a workspace view. A workspace view is an interactive, visual representation of a KB workspace and behaves like a workspace in G2:

  • Modifications to items in a workspace view are reflected in a KB

  • Changes made to an item in the KB are reflected on the workspace view.
