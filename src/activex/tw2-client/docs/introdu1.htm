Introduction

 Telewindows2 Toolkit for ActiveX is part of the Gensym's Graphical User Interface (GUI) for G2. Telewindows2 Toolkit comprises a large collection of Java classes and Java Beans that you can use to capture and display G2 data and to build Java application shells that give your users access to G2 features and operations. Telewindows2 Toolkit for ActiveX is an Application Programer's Interface (API) to Telewindows2 Toolkit that enables you to tap its power in an ActiveX environment, such as Microsoft Visual Basic or Microsoft Office.

The Telewindows2 Toolkit for ActiveX enables you to establish communication between G2 and a COM-compliant container, such as Microsoft Visual Basic 5.0, running under Windows 95 or Windows NT. You can display a G2 workspace in Visual Basic and make changes to it, such as moving visible items and editing attributes of items. These changes appear in G2. Conversely, any changes made in G2 are reflected in the workspace displayed in Visual Basic.
