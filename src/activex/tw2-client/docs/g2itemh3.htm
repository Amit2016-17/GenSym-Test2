Methods

CallMethod()

 The CallMethod() method calls a procedure in G2 and waits for the G2 procedure to complete.

Syntax

    CallMethod(MethodName As String,
       [    ParamArray InputArguments() As  Variant]) 
       ReturnArguments As  Variant
    Arguments Description
    MethodName As String
    		The name of the method you are calling.
    ParamArray InputArguments()
    As Variant
    		Any of zero or more arguments to be passed to the method.

    Return Values Description
    ReturnArguments As Variant
    		Any of zero or more arguments to be returned from the method. If a method returns more than one value, Visual Basic can access the return values by using array subscripting, such as Ret(1), Ret(2), and so on. If only one value is returned, the array subscript is not necessary.

Description

 The CallMethod() method waits until the G2 procedure completes and returns values before executing the next statement following the call.

Note: The container application can pass any number of arguments to G2 and G2 can return any number of arguments.

Telewindows2 Toolkit returns an exception to the caller under one of the following conditions:

  • If G2 does not respond within the time specified by the CallTimeout property.

  • If an error with the call is detected by Telewindows2 Toolkit.

  • If the G2 procedure signals an error.

Example

 Visual Basic calls the G2 procedure and passes three values: an integer, a string, and a boolean. The call waits for a return value. G2 displays these values on the message board and returns a string to Visual Basic.

Visual Basic 

    Private Sub Form_Load()
       Ret = G2UiGateway1.CallMethod("my-proc", 
          1, "Hello G2", True)
       MsgBox Ret
    End Sub

G2 

    my-proc(arg1: value, arg2: value, arg3: value) = value
     
    begin
       inform the operator that "Value1:[arg1], Value2:[arg2], Value3:[arg3]";
       return "Hello COM";
    end

Download()

 The Download() method downloads a new set of attributes for the G2 item in G2ItemHolder. You can use the Download() method to update attributes if you are not automatically updating them by using the AutoDownload property.

IsInstanceOfClass()

The IsInstanceOfClass() method tests to see if the current item is a member of the specified class.

Syntax

    IsInstanceOfClass(ClassName As String) As Boolean
    Argument Description
    ClassName As String
    		The name of the class that is being checked to see if the current item is a member of the specified class.

StartMethod()

 The StartMethod() method starts a procedure in G2. The effects of starting the procedure are reflected back into your Visual Basic container.

Syntax

    StartMethod(MethodName As String,
       [    ParamArray InputArguments() As  Variant]) 
       ReturnArguments As  Variant
    Arguments Description
    MethodName As String
    		The name of the method you are calling.
    ParamArray InputArguments()
    As Variant
    		Any of zero or more arguments to be passed to the method.

    Return Values Description
    ReturnArguments As Variant
    		Any of zero or more arguments to be returned from the method. If a method returns more than one value, Visual Basic can access the return values by using array subscripting, such as Ret(1), Ret(2), and so on. If only one value is returned, the array subscript is not necessary.

Description

 The started G2 procedure cannot return values to the container application.

Note: The container application can pass any number of arguments to G2.

Structure()

The Structure() method creates a G2 structure from an array of name-value pairs. Structure() serves as a constructor for creating a G2 structure, removing the need to use the Visual Basic keyword new to do create a structure.

Syntax

Structure([ParamArray NameValuePairs() As Variant]) As G2Structure
Argument Description
NameValue
Pairs() As Variant
An array of name-value pairs out of which a G2 structure is created, which can be sent to G2.

SubscribeToAttribute()

The SubscribeToAttribute() method enables you to track the changes to an attribute in G2.

Syntax

    SubscribeToAttribute(AttributeName As String)
    Arguments Description
    AttributeName As String
    		The name of the attribute whose changes you wish to track.

Symbol()

 The Symbol() method creates a G2 symbol from a text string. Symbol() serves as a constructor for creating a G2 symbol, removing the need to use the Visual Basic keyword new to do create a symbol.

Syntax

    Symbol(SymbolName As String) As G2Symbol
    Argument Description
    SymbolName As String
    		A text string out of which a G2 symbol is created, which can be sent to G2.

UnsubscribeToAttribute()

 The UnsubscribeToAttribute() method stops tracking changes to an attribute in G2.

Syntax

    UnsubscribeToAttribute([AttributeName As String])
    Arguments Description
    AttributeName As String
    		The name of the attribute whose changes you have been tracking.

Upload()

 The Upload() method uploads to the G2 server the local changes you have made to a set of attributes for the G2 item in G2ItemHolder. You can use the Upload() method to update attributes if you are not automatically updating them by using the AutoUpload property.
