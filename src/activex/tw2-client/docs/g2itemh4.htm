Event

 G2ItemHolder can raises the event described in this section. In your Visual Basic application, you can specify logic to respond to this event. You do not need to write handlers for events that are not needed by your container application. The event handler for a specific event can freely call other methods of Telewindows2 Toolkit for ActiveX and procedures in G2.

SubscribedAttributeChanged

The SubscribedAttributeChanged() event occurs when the subscribed attribute changes.

Syntax

    SubscribedAttributeChanged(AttributeName As String, NewValue As Variant)
    Arguments Description
    AttributeName As String
    		The name of the attribute whose changes you have been tracking.
    NewValue As Variant
    		The new value of the attribute that changed.
