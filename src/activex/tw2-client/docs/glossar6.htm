M

 methods: An action in one object that can be triggered by an event in another object. For example, a user presses a button that invokes a public method in a calendar object to move ahead one day.

Multiple-Document Interface (MDI): An interface in which multiple documents, such as windows displaying G2 workspaces, are displayed inside a single application window.
