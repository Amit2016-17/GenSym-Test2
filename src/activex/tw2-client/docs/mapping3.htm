Passing a G2 Structure

 The G2Structure() class enables you to pass G2 structures as arguments in methods in Visual Basic. The properties enable you to read a list of all the members of a G2 structure.

Properties

Properties Description
Member(MemberName As String)
Enables you to access the value of the named member. You can use the Member property at runtime, not design time. Once a connection to G2 is established, you can programmatically access the value of this property.
MemberNames As Variant
A read-only list of member names.

Example

At runtime, the following code fragment sets the value of a G2 structure member, which you can pass to the G2 server.

    Private Sub Command1_Click()
       Dim Temperature As G2Structure
     
       Set Temperature = New G2Structure
       Temperature.Member("Cooling") = 40
To create G2 structures inline, see Structure().

Methods

ClearMember()

 The ClearMember() method enables you to erase the value of all the members of a structure or remove the value of a specified member. To erase the values of all structure members, invoke the ClearMember() method without an argument. 

    ClearMember(MemberName As String)
    Argument Description
    MemberName As String
    		The name of the structure member to remove.

Example

 At runtime, the following code fragment clears the value of a G2 structure member, which you can pass to the G2 server.

    Private Sub Command1_Click()
       Dim Temperature As G2Structure
     
       Set Temperature = New G2Structure
       Temperature.ClearMember("MaxHeating")

String()

 The String() method enables you to display the text of the G2 structure in your application.

Syntax

String() As String

Example

 The following code fragment displays the value of a G2 structure in a form's caption.

    Private Sub Command1_Click()
       Dim Temperature As G2Structure
     
       Set Temperature = New G2Structure
       Temperature.Member("Cooling") = 40
       Caption = Temperature.String
