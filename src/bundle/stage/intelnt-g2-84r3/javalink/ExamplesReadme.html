Welcome to G2 JavaLink/Gateway Examples

This release comes with an example KB that utilizes G2 JavaLink/Gateway;

JAVALINK_HOME\kbs\jgidemo.kb

This KB contains tests that allow you to try out G2 JavaLink Gateway features; demonstrating Java-G2 data conversion, RPC calling, Multi threaded access, Java downloading from G2 and much more.

To startup the main G2 JavaLink/Gateway Example client:

  • 1. From G2, load and start jgidemo.kb
  • 2. Startup the main JavaLink example by typing java com.gensym.demos.jgi.ExampleG2Gateway
  • 3. Follow the instructions in jgidemo.kb
    •      This KB can also be used with the other example G2 JavaLink/Gateway examples found in com.gensym.demos.jgi;
     

     Example File

    Description
    ExampleG2Gateway.java           (Note1) This class demonstrates the use of Javalink as a means to communicate with a G2 server via RPC's to G2 from Java and G2 to Java Methods. It shows how complex and simple data and objects are mapped between the  the two systems. It also demonstrates multiple thread accessing a  G2 connection. 
    To use: 
    • 1. make sure jgidemo.kb is loaded and started,
    • 2. run this class: java com.gensym.demos.jgi.ExampleG2Gateway
    • 3. Press the connect button in G2
    • 4. Follow instructions in the G2 Kb
    JavaCallingG2Methods.java (Note1)  This Class simply demonstrates how com.gensym.jgi.G2Gateway can be used to connect to a G2 server and Call a G2 Method of a previously downloaded class.  This class will call a method on generic-r-employee, an instance of a particular G2 class (generic-employee) in order to change an attribute of the instance. 
    To use: see (Note 1)
    JavaCallingG2Procedure.java     (Note1) This Class simply demonstrates how com.gensym.jgi.G2Gateway can be used to connect to a G2 server and Call a G2 Procedure. 

    This method will call the G2 procedure "A-G2-PROCEDURE" across g2Connection and pass the following parameters to G2 via a Procedure call. 

    • INTEGER parameter 
    • TEXT parameter (String converts to G2 TEXT)
    • FLOAT paramter (double [and float] convert to G2 FLOAT)
    • SYMBOL parameter (com.gensym.util.Symbol converts to G2 SYMBOL)
    The G2 Procedure will inform display a message that it has received the parameters and return the current real time to Java., as a double.
    JavaThreadsCallingG2.java        (Note1) This Class simply demonstrates how multiple Java threads can communicate with a G2 server. 
    SendG2Message.java                (Note1) This Class simply demonstrates how com.gensym.jgi.G2Gateway can be used to 
    connect to a G2 server and send a message to its message board.
    ConnectionEventListener.java    (Note1) This Class demostrates how a class can listen for G2ConnectionListener events  that may occur for a particular G2Connection. 
    A G2ConnectionListener is informed of the following; 
    •  That the connected G2 has resumed.
    •  That the connected G2 has paused.
    •  That the connected G2 has been reset.
    •  That the connected G2 has been started.
    •  That the connected G2 has send a message (VIA INFORM THE...)
    •  That the connection to G2 has been closed.
    •  That a connection to G2 has just been established.
    •  That a communicationError has occured while the connection to a G2 is active.
    •  That the network cannot deliver data that Java is attempting to write to G2.
    •  That the network cannot deliver data that G2 is attempting to write to Java.
    A G2ConnectionListener can register interest in these events by calling  G2Gateway.addG2ConnectionListener and can remove interest by calling  G2Gateway.removeG2ConnectionListener
    UnsolicitedConnection.java        (Note2) This Class demonstrates how G2 can initiate an unsolicited connection to a Java VM  running JavaLink. A G2ConnectionHandlerFactory is registered to create a G2Connection to handle incomming network connections from G2 to Java. 
     To use: 
    • 1. make sure jgidemo.kb is loaded and started,
    • 2. run this class: java com.gensym.demos.jgi.UnsolicitedConnection
    • 3. Press the connect button in G2
    • 4. A Message from Java will appear on the G2 message board
    G2CallingJavaMethod.java    (Note2) This Class demonstrates how a Java Method can be registered such that it can be called by G2 over a G2Connection. This example waits for G2 to initiate  a network connection with the Java VM. Once connected G2 can call the method receiveSend. 
    To use: 
    • 1. make sure jgidemo.kb is loaded and started,
    • 2. run this class: java com.gensym.demos.jgi.G2CallingJavaMethod
    • 3. Press the connect button in G2
    • 4. Go to the SENDING-RECEIVING-DATA workspace
    • 5. Send data from G2 by pressing Send buttons
    SimpleSubscription.java (Note 1) A simple class that demonstrates how to connect to a G2, obtain an Item by class and name, and subscribe to that Item 

    After the class is started, any change to a subscribable attribute on the named item will result in a printout to stdout showing the contents of the ItemEvent that was sent because of that change. This makes this class a useful tool in developing applications that depend on subscriptions, as it enables the developer to see the structure of  the data that is sent for that subscription. Pay particular attention to the "new value" part of the printout.
    SimpleVariableSubscription.java (Note 1) A simple class that demonstrates how to connect to a G2, obtain a VARIABLE-OR-PARAMETER by class and name, and subscribe to updates in its LAST-RECORDED-VALUE attribute. 

    After the class is started, any change to the LAST-RECORDED-VALUE attribute on the named VARIABLE-OR-PARAMETER will result in a printout to stdout showing the contents of the VariableValueEvent that was sent because of that change. This makes this class a useful tool in developing applications that depend on subscriptions, as it enables the developer to see the structure of  the data that is sent for that subscription. Pay particular attention to the "new value" part of the printout.
    ExampleDataService.java (Note 2) This example shows how Javalink can be used to provide a Gsi Data Service updates for a G2 connection. 
    To use: To use:
    •  1. make sure jgidemo.kb is loaded and started, 
    •  2. run this class: java com.gensym.demos.jgi.ExampleDataService
    •  3. follow the instructions found by pressing the GSI-DATA-SERVICE-EXAMPLE button in G2.
