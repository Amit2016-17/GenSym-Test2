ShortestPath Demo

The ShortestPath Demo implements a shortest path algorithm for oil movements within a tank farm. A movement consists of an originating tank, an intermediate pump and a destination tank. The search algorithm uses a modified version of the Dykstra shortest path algorithm in combination with a backtrack search algorithm to get the “n” most optimum paths. The “cost” of a movement is calculated as the number and type of valves that have to be opened or closed in order to get an isolated path.
