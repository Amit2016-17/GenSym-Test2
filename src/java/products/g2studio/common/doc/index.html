What is G2Studio?

G2 Studio is a comprehensive development environment for G2, built on Telewindows2 Toolkit, a set of java-based user interface components. G2 Studio is a software product for developing G2 knowledge bases. There are many features that G2 Studio offers knowledge base developers including text and icon editors, workspace navigation tools, Web interaction, toolbars and palettes, and much more.  In addition, try clicking on some of the following links to connect to a particular G2 server or to go to a named item or workspace.   You can even bookmark an unnamed workspace using its uuid attribute. By default, this is your home page, but feel free to edit your preferences and have it point to your most frequented G2, or even a workspace within that G2.
