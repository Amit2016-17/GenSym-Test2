Gensym G2Studio

Version 1.0 Rev. 0

on Windows NT 4.0 or greater with Intel processors


What is G2Studio?

G2 Studio is a comprehensive development environment for G2, built on Telewindows2 Toolkit, a set of java-based user
 interface components. G2 Studio is a software product for developing G2 knowledge bases. There are many features that G2
 Studio offers knowledge base developers including text and icon editors, workspace navigation tools, Web interaction, toolbars
 and palettes, and much more.

How to Run G2Studio

G2Studio can be run from the "Start" menu. It should be installed under "Programs". It can also be run from the g2studio.bat file that can found in %G2STUDIO_HOME%\bin.

Requirements for using G2Studio

In order to use G2Studio, you must have the following software installed:
  • The Java Runtime Environment 1.3  which is also contained within the G2 Studio install.
  • G2 6.0 Alpha rev.0
  • Windows NT 4.0 or greater.

What does this release contain?

This release of G2Studio contains the following software:
  • Java classes that implement G2Studio(Javalink and Telewindows2 Toolkit are embedded within G2Studio)
  • A batch file to run G2Studio, located in the installed bin directory, where g2studio is installed.

System Requirements for running Telewindows2 Toolkit

To use G2Studio on Windows NT on Intel platforms, you will need:
  • A Pentium machine running Windows NT 4.0 (minimum clock speed 133 MHz). A Pentium II CPU and clock speeds higher than 200MHz is recommended.
  • Minimum 64 MB of RAM.

    • Approximately 30 MB of disk space to install the software and documentation.
