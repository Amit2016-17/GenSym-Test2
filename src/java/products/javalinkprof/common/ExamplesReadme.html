Welcome to G2 JavaLink Professional Examples

This release comes with an example KB that utilizes G2 JavaLink Professional;

JAVALINKPROF_HOME\kbs\jlpdemo.kb

This KB contains tests that allow you to try out G2 JavaLink Professional features.

  • 1. Load and start kb
  • 2. Set your current directory to JAVALINKPROF_HOME\classes\com\gensym\demos\jlp
  • 3. Startup the example Java applet by typing  java com.gensym.demos.jlp.ExampleJLPGateway
  • 4. Follow instructions in jlpdemo.kb
