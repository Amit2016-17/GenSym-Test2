

DlgTestApp is an application shell that demonstrates the following:

1. the usage of many of the dialogs in com.gensym.dlg.

The menus for DlgTestApp are generated from DlgCommand. DlgCommand is the
class that actually handles the displaying and dismissing of the dialogs 
in this demo.

******************************************************************************

To start the demo, type the following:  

	java com.gensym.demos.standarddialogs.DlgTestApp

******************************************************************************
