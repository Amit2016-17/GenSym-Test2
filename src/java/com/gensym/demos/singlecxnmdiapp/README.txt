

SingleConnectionApplication demonstrates the following:

1. how to build an application that handles just one TW2 connection at a time.

2. how to build an MDI application by extending com.gensym.shell.util.TW2MDIApplication.

3. how to subclass WorkspaceDocument in order to have an application specific menu for 
workspace views.

4. how to get/set Enable and Disable state information for G2 items on a workspace view

******************************************************************************

To start the demo you should have a G2 already running that you can connect to, 
then type the following:  

	java com.gensym.demos.singlecxnmdiapp.SingleConnectionApplication

******************************************************************************
