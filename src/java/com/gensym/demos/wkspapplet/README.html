Workspace Applet

Workspaces can be displayed in applets through the use of the JavaTM Plug-in. This demonstration shows a fairly simple way of showing any workspace from any G2 in an applet.

Java Plug-in is a Netscape Plug-in or Explorer ActiveX object written by Sun Microsystems. It displays applets using the Java Runtime Environment (or JRE) version 1.1.X, rather than using the browser's own Java Virtual Machine. If you don't already have the Java Plug-in installed, then when you follow the link to the applet, you will be guided through the installation process for the Plug-in.

The Applet

Warning:

If you have installed Netscape's patch for awt1.1, displaying a popup (i.e., clicking the right mouse button anywhere on the workspace), will result in an application error. You do not need the awt1.1 patch to run the Plug-in.
