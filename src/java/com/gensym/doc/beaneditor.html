Nevada Bean Editor

The Nevada Bean Editor is a tool for editing Java Beans.  The
tool allows a user to load Java Beans from jarfiles.  Once
beans have been loaded into the editor, uses can edit their
properties and connect them to each other via events.

In order to run the Bean Editor, type :

    java com.gensym.beaneditor.BeanEditor [jarfile]

The "jarfile" option specifies the path of a jarfile to
open at start.

In order to load a jarfile:

1) Select "Open Jar..." from the "File" menu.
2) A file dialog will be displayed.
3) Select the name a jarfile from the dialog
   or enter a new name.
4) Select "Open" on the file dialog.
5) All the beans in the jarfile will be listed
   under "Controls" on the main menu

In order to add a new bean to the working canvas:

1) Select a bean from the "Controls" menu
2) With the mouse cursor over the canvas, hold the
   right mouse buton down, and start dragging.
3) A rectangle which represents the size of the
   bean you are creating will be displayed as you drag.
4) Release the mouse button.
5) A new bean will be added to the canvas.

In order to resize a bean:

1) Right click on a bean.
2) Eight black resize handles will be displayed around the
   selected bean.
3) Hold the right mouse button down over one of the resize
   handles and drag the mouse.
4) A rectangle which represents the size of the
   bean will be displayed as you drag.
5) Release the mouse button.
6) The bean will be resized to the size of the rectangle.

In order to move a bean:

1) Hold the right mouse button down over a bean, and
   begin dragging.
3) Release the mouse button.
4) The bean will be moved to the new location.

In order to edit the properties of a bean:

1) Left click on a bean.
2) A pop-up menu will be displayed.
3) Select "Edit..." from the pop-up menu
4) A properties dialog will be displayed.
5) The properties of the bean can be changed
   by using the controls on the dialog
   (text fields, lists, etc.)
   (Note: For some properties (Font, Color) a box
   will be displayed next to the name of the property.
   Right clicking on the box will launch an editor for
   the property.

In order to create an output event:

1) Left click on a bean.
2) A pop-up menu will be displayed.
3) Select "New Event Port" from the pop-up menu.
4) A list of "event sets" (mouse, key, etc.) will be
   displayed on a cascaded menu.
5) Select an event set from the cascaded menu.
6) A list of events will be displayed on a cascaded menu.
6) Select an event from the cascaded menu.
7) An output port will be added to the bean.  This port
   will have an event "stub" and round "loose end" attached
   to it.

In order to connect a loose end to another bean:

1) Hold the right mouse button down over a loose end.
2) Drag the loose end on top of another bean.
3) Release the mouse button.
4) A small dialog will be displayed with a list of methods.
5) Select one of the methods.
6) The loose end disapears, and an input port is added to
   the second bean.  The two beans are now connected via
   an "event connection".  When the event corresponding
   to the output port on first bean occurs, the method
   corresponding to the input port on the second bean will
   now be invoked.

In order to save the state of the editor:

1) Select "Save As..." from the "File" menu.
2) A file dialog will be displayed.
3) Select the name a file from the dialog
   or enter a new name.
4) Select "Save" on the file dialog.
5) The state of the editor has now bee saved as a
   file, and can be reloaded via "Open" on the file menu
