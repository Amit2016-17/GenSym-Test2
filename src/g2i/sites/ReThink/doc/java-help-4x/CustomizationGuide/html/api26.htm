Topic - Application Programmer's Interface

bpr-set-attribute-value

Sets the value of an attribute when performing validation on custom properties dialogs.

Synopsis

bpr-set-attribute-value(item: class object, subtable: symbol, attribute: symbol,    data: item-or-value, apply-time-unit: truth-value, time-unit: item-or-value,    notify: truth-value)

Argument
Description
item

The class of object whose attribute values are to be set.
subtable

The subtable of the attribute value to set, as a symbol, for example, cost-subtable.
attribute

The attribute value to set, as a symbol.
data

The value to use for attribute.
apply-time-unit

Whether to apply time-unit to attribute before setting the value.
time-unit

The time unit to apply to the value, for example, seconds, minutes, hours, days, or weeks.
notify

Whether to call bpr-updated-attributes after the attribute value has been set.

Description

This method allows you to perform data validation on attribute values in custom properties dialogs, specified by the bpr-get-attribute-properties method. For details, see Customizing Attribute Properties.

The value to set in data may be a text, for example, while the attribute of the item may be a float, for example. This method performs the conversion.
