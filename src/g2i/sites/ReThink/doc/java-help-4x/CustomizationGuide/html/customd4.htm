Topic - Customizing the User Interface

Creating Custom Tabs

To create custom tabs and to control the order of the attributes that appear on each tab, you implement the bpr-get-dialog-info method for a custom class. This method returns a structure, which sets the dialog title and its label, and the custom tabs. You specify the custom tabs as a sequence of structures, where each structure sets the tab label and the tab attributes as a sequence of symbols.

For details on the signature of this method, see bpr-get-dialog-info.

To customize the tab and order of attributes on properties dialogs:

1 Choose View > Toolbars > New Item and click the Rules and Definitions tab:

2 Create a method and place it on your customization workspace with your custom class definition.
3 Create a method named bpr-get-dialog-info for your custom class, which returns a structure, according to the signature given above.
