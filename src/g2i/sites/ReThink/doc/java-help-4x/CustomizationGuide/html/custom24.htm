Topic - Customizing Blocks

Common Attributes of Duration Subtable

The attributes of the duration-subtable of a block are:

Attribute
Description
mode-type

A symbol that specifies how the block computes duration. The default value is random-normal for all blocks, except the Source block, whose default value is random-exponential. The other options are: fixed-duration, random-uniform, random-triangular, random-erlang, random-weibull, random-lognormal, random-gamma, random-beta, work-object-duration, report-indexed-lookup, report-lookup, duration-file, and custom.
procedure-name

The name of the procedure the block uses to compute duration. The default value for all blocks except the Source block is bpr-random-normal-duration. The default value for the Source block is bpr-random-exponential-duration.
mean

The mean time the block uses to compute the duration of its activities for the following random distributions: Fixed, Normal, Exponential, Erlang, and Lognormal.
standard-deviation

The standard deviation around the mean the block uses to compute duration for the following random distributions: Normal and Lognormal.
min

The minimum parameter of the following random distributions: Uniform, Triangular, and Beta.
max

The maximum parameter of the following random distributions: Uniform, Triangular, and Beta.
mode

The mode parameter of a random Triangular distribution.
number-of-samples

The number of samples to use for a Random Erlang distribution.
shape

The shape of a Random Weibull distribution.
scale

The scale of a Random Weibull distribution.
alpha

The alpha parameter of the Random Gamma and Random Beta distributions.
beta

The beta parameter of the Random Gamma and Random Beta distributions.
duration-file-name

The name of the duration file in Duration File duration mode.
repeat-file-flag

Whether to cycle through the duration file when generating work objects in Duration File duration mode. The default is true.
duration-attribute-name-of-bpr-object

The attribute name that contains the duration in Work Object duration mode.
report-title

The report title in Report Indexed Lookup or Report Lookup duration mode.
lookup-attribute-name

The report attribute column name that contains the duration in Report Indexed Lookup or Report Lookup duration mode.
lookup-label-attribute-name

The report attribute column name containing the block labels to include in the search criteria to find the row in the report containing the duration in Report Indexed Lookup mode.
time-per-unit-attribute-name

The attribute of the work object that determines the number of units the work object represents, which the block multiplies by the duration of the activity to get the duration of the block.
total-work-time

The sum of all the work times for each activity of the block as of the current time.
total-elapsed-time

The amount of time that has elapsed since the simulation began or since you created the block in a running simulation.
creation-time

The simulation time at which the block was created.
average-in-process

The average number of concurrent activities a block has performed since the start of the simulation.
reset-procedure-name

The name of the procedure the block uses to reset the duration subtable when the simulation resets. The default value is bpr-block-duration-subtable-reset.

For an example of how to customize the duration of a block, see Customizing the Duration of a Block.
