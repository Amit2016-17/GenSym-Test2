Topic - Application Programmer's Interface

bpr-get-attribute-properties

Returns a structure that describes the properties of an attribute on a custom tab of a properties dialog.

Synopsis

bpr-get-attribute-properties (object: class object, subtable: symbol,    attribute: symbol, ui-client-item: class ui-client-item)    -> (structure)

Argument
Description
object

The object whose attribute properties the method is describing.
subtable

The subtable in which the attribute appears, if any, for example, duration-subtable.
attribute

The attribute whose properties the method is describing.
ui-client-item

The ui-client-item in which the attribute properties appear, which is the superior class of:

A g2-window, which is a window in the current G2 session or a remote connection to G2 through a Telewindows client.
A ui-client-session, which is a remote connection to G2 through a Telewindows2 Toolkit client, which allows you to create native UI clients for G2 applications.

Return Value
Description
structure

A structure that describes various properties of the attribute.

The returned structure has this format:

structure    (ATTRIBUTE: the symbol FIXED-COST, // attribute name as a symbol     LABEL: "Fixed Costs:", // label in dialogs, reports, and ui     TYPE: the symbol float, // see below for valid types     READ-ONLY: true, // true if it is read-only     DEVELOPMENT-ONLY: false, // true for developer mode only     MIN: 0, // minimum value for quantity values     MAX: 100, // maximum value for quantity values     CHOICES: the symbol none) // procedure name to call to get list of // choices.

The options for the type element of the structure depend on the attribute type specified in the class definition, as follows:

The signature of the procedure to specify for choices has this signature:

choice-procedure-name (item: class object) -> (sequence)

Description

To customize the properties of attributes that appear on custom tabs, you implement the bpr-get-attribute-properties method for a custom class. This method returns a structure, which sets various properties for each attribute, including its label, type, whether it is read-only, whether it is available in developer mode only, and its minimum and maximum values.

The method maps the attribute type specified in the class definition to the type element of the structure.
