Customizing the User Interface

ReThink allows you to customize these aspects of the user interface:

Popup menus for objects on a workspace.
Properties dialogs for blocks, instruments, resources, and work objects.
ReThink toolbar to add custom tabs with custom objects.

ReThink does not currently support customization of these aspects of the user interface:

Top-level menu bar.
Toolbars other than the ReThink toolbar.

For more elaborate customizations or to build a completely new user interface to provide a custom product identification ReThink Online includes extensive libraries to build G2-based or Java-based user interfaces. For details, see the G2 Classic documentation provided with ReThink Online.
