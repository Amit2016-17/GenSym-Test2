Customizing Animation, Duration, and Cost Procedures

To customize the animation, duration, and cost procedures of objects, you follow the steps described in Customizing Reset, Delete, and Update Procedures, except that:

You can copy the default procedure and edit it, as needed.
You edit the procedure names on the Animation, Duration, and Cost tabs of the properties dialog.

For details, see Displaying Default Subobject Procedures.
