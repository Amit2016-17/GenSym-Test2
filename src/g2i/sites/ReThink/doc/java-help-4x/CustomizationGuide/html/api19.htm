Topic - Application Programmer's Interface

bpr-message-to-all-users

Sends a message to connected clients.

Synopsis

bpr-message-to-all-users(msg: text, type: text { "WARNING", "ERROR"})

Argument
Description
msg

The text of the message to send.
type

The type of message to send as a text. A warning sends the message to the Message Board, and an error sends the message to the Logbook.

Description

This procedure sends msg to all connected clients of the server, which includes the Java client application and connected Excel workbooks. The procedure can send the message to the Message Board or Logbook.

Example

bpr-message-to-all-users("You should not be working on Saturday!", "WARNING")
