Topic - Customizing Resources and Work Objects

Attributes of Cost Subtable

The attributes of the cost-subtable of a resource or work object are:

Attribute
Description
cost-per-use

For resources, the fixed cost applied to each resource. This attribute is not relevant for work objects.
cost-per-time-unit

For resources, the variable cost applied to each resource, per time-unit. This attribute is not relevant for work objects.
time-unit

For resources, the time unit the resource uses to compute variable costs. The default value is 1 hour. This attribute is not relevant for work objects.
total-cost

The sum of all the fixed and variable costs of the resource, or the sum of all the activity costs applied to the work object, which includes block and resource costs.
reset-procedure-name

The name of the procedure the resource or work object uses to reset the cost subtable when the simulation resets. The default value is bpr-reset-object-cost-subtable.
procedure-name

The name of the procedure the resource or work object uses to compute total-cost. The default value is bpr-object-cost.
