Customizing the ReThink Toolbar

When subclassing ReThink blocks, instruments, and resources, you might want to make the custom objects available on the ReThink toolbar. You can add any number of user tabs with custom objects.

To customize the ReThink toolbar, you create a Palette Tab object, place custom objects on its detail, and configure the objects on the detail for palette behavior. You configure the order in which the tab appears on the ReThink toolbar, depending on your license, as follows:

Note: Due to item restrictions, you must currently configure the Palette Tab detail label and objects in the server, using Administrator mode. To do this, you must connect to the server, using Telewindows.

To customize the ReThink toolbar:

1 Display the ReThink toolbar and click the Tools tab:

2 Create a Palette Tab and place it on your customization workspace.
3 Configure the Names attribute to be a unique name.
4 Configure the Palette Order to be the order in which the tab appears on the toolbar, according to the table above.
5 Choose Show Detail on the palette tab.
6 Connect to the server, through Telewindows, as follows:
a Navigate to the g2 directory of your ReThink product directory and double-click tw.exe or make a shortcut.
Telewindows automatically connects to the server with your application loaded.
b Enter Ctrl + y and edit the G2 User Mode to be administrator.
c Navigate to the workspace with the Palette Tab object you created and show the detail of the palette tab.
7 Click the label of the detail and edit it to be the label of the tab in the toolbar.
8 Place objects on the detail and enable GFR palette behavior for each object, as follows:
a Display the workspace named gfr-top-level and click the check box next to the palette preparation tools option:

b Place custom objects on the detail to make them appear on the custom tab.
c For each object, choose Add Palette Behavior.
d Once you have added all the objects you want, click the disable the palette preparation tool option on the gfr-top-level workspace.
9 Switch back to Developer mode by choosing Ctrl + y and entering developer.
10 Close the Telewindows connection by clicking on the background and choosing Main Menu > Miscellany > Close Telewindows Connection.
11 Refresh the client user interface by pressing F5 or choosing Go > Refresh.
12 Display the ReThink toolbar.
The custom tab appears on the toolbar with the custom objects.
13 Save you customizations.
