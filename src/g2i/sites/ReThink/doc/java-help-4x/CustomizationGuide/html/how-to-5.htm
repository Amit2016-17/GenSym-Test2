Topic - How to Customize ReThink

Online Modules

If you have licensed ReThink Online, ReThink provides the following additional modules to support online mode:

Module
Description

rethink-online

The top-level ReThink module in which you create new ReThink applications that use online mode.

methods-online

The module you merge to add online capability to existing ReThink applications.

For details, see Chapter 10, "Using ReThink in Online Mode" in the ReThink User's Guide.
