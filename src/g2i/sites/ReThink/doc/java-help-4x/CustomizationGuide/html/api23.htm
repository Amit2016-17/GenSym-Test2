Topic - Application Programmer's Interface

bpr-remove-from-pool

Removes an object from a pool.

Synopsis

bpr-remove-from-pool ( pool: class bpr-object, object: class bpr-object )

Argument
Description
pool

A bpr-object with a subworkspace that contains object.
object

The bpr-object that is removed from pool.

Description

Removes object from pool, by removing it from the list of items in the pool and from the subworkspace of the pool. The method that defines the default behavior of the Retrieve block calls bpr-remove-from-pool to retrieve objects from a pool.

Example

This method is bpr-retrieve::bpr-stop-method, the method that defines the default behavior of the Retrieve block. The method retrieves objects from the pool according to the Procedure-name of the Lookup-subtable of the block. The lookup subtable determines how the block retrieves objects from the pool.
