Common Attributes of Animation Subtable

The attributes of the animation-subtable of a block are:

Attribute
Description
active-color

The color the block uses when it is active processing work objects. The default value is aquamarine.

inactive-color

The color the block uses when it is idle. The default value is thistle.
error-color

The color the block uses when it is in an error state. The default value is yellow.
detail-color

The color a Task block uses when it has detail; not relevant for all other blocks.
reset-procedure-name

The name of the procedure the block uses to reset the animation subtable when the simulation resets. The default value is bpr-animate-block.
procedure-name

The default animation procedure for the block. The default value is bpr-animate-block.
