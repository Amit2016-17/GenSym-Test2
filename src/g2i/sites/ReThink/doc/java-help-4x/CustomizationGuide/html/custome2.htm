Customizing Popup Menus

You can add menu choices to the popup menus for objects. The popup menu appears when you right-click an object. You add additional popup menu choices by defining G2 user-menu choices for objects. You cannot remove menu choices from the popup.

To customize popup menus:

1 Choose View > Toolbars > New Item and click the Rules and Definitions tab:

2 Create a user-menu choice and place it on your customization workspace with your custom class definition.
3 Configure the properties of the user-menu choice, as needed.

For details on how to create user-menu choices, see the G2 Reference Manual.
