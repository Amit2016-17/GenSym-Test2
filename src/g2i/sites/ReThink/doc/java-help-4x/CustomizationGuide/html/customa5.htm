Topic - Customizing Instruments

Common Attributes of Animation Subtable

The attributes of the animation-subtable of an instrument are:

Attribute
Description
active-color

The color the instrument uses when it is active. The default value is sky-blue for probes and magenta for feeds.
inactive-color

The color the instrument uses when it is idle. The default value is blue for probes and violet-red for feeds.
error-color

The color the instrument uses when it is in an error state. The default value is yellow.
procedure-name

The default animation procedure for the instrument. The default value is bpr-animate-instrument.

Customizing the default colors of instruments is similar to customizing the default colors of blocks. Similarly, customizing the animation procedure of instruments is similar to customizing the animation procedure of blocks.
