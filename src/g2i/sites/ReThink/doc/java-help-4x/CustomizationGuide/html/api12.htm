Topic - Application Programmer's Interface

bpr-get-dialog-info

Returns a structure that describes the elements of a custom dialog, including the custom tabs and the order of the attributes on each tab.

Synopsis

bpr-get-dialog-info (object: class my-task, ui-client-item: class ui-client-item)    -> (structure)

Argument
Description
object

The object whose dialog information the method is describing.
ui-client-item

The ui-client-item in which the dialog appears, which is the superior class of:

A g2-window, which is a window in the current G2 session or a remote connection to G2 through a Telewindows client.
A ui-client-session, which is a remote connection to G2 through a Telewindows2 Toolkit client, which allows you to create native UI clients for G2 applications.

Return Value
Description
structure

A structure that describes dialog title, label, and custom tabs.

The returned structure has this format:

structure ( DIALOG-TITLE: title, DIALOG-LABEL: "My label", DIALOG-TABS: sequence ( structure ( TAB-LABEL: "My Tab 1", TAB-ATTRIBUTES: sequence( the symbol CUSTOM-ATTRIBUTE-1, the symbol CUSTOM-ATTRIBUTE-2, the symbol CUSTOM-INT, the symbol CUSTOM-FLOAT, the symbol CUSTOM-BOOLEAN, the symbol SUM-VAL)

To specify the dialog title, you typically refer to the class, replacing hyphens with spaces and capitalizing each word. For example, this code fragment sets the local variable named Title, which is a text:

Title = "[the class of object]"; Title = find-and-replace-pattern("-", " ", title, 1, length-of-text(title)); Title = capitalize-words (title);

Description

To create custom tabs and to control the order of the attributes that appear on each tab, you implement the bpr-get-dialog-info method for a custom class. This method returns a structure, which sets the dialog title and its label, and the custom tabs. You specify the custom tabs as a sequence of structures, where each structure sets the tab label and the tab attributes as a sequence of symbols.

Example

For an example, see Creating Custom Tabs.
