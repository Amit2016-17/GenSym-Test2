Creating Input Reports for Configuring Key Parameters

You will create the following input reports for configuring the key Level 2 SCOR parameters:

Level 2 Input Report, which will contain parameters relating to orders quantity and frequency, delivery time, and planning.
Product Composite Input Report, which will contain parameters relating to product cost, starting inventory, planning.
Tip: Remember, you must include the UUID parameter in all input reports, which provides a unique identifier for each row in the spreadsheet.

To create input reports for configuring key parameters:

1 Create a Level 2 Input Report from the Reports tab on the e-SCOR toolbar and place it on the model detail.
2 Click the Attributes tab and configure the attributes that should appear in the report, according to the table above.
For details, see Creating Input Reports for Configuring Level 2 SCOR Parameters.
When you have configured the dialog, it should look like this:

3 Open the e-SCOR Default Summary Reports in Excel.
4 Choose Create Report on the Level 2 Input Report to create the report template in Excel.
5 Configure the spreadsheet to facilitate viewing.
Tip: Hide the ES, ED, S1, and Mb categories so the spreadsheet includes only the Sc, P2, P3, and D1 categories.
For details, see Creating Input Reports for Configuring Level 2 SCOR Parameters.
6 Create a Product Composite Input Report and configure the attributes to appear, according to the table above.
7 Choose Create Report on the Product Composite Input Report to create the report template in Excel.
