Topic - Configuring Level 1 Roles: ACME Model

Summary: Configuring Level 1 Roles for the ACME Model

To configure Level 1 roles for the ACME model, you:

Set up the model environment.
Create a product hierarchy.
Configure the supply-chain model,.
Assign products to roles.
Configure role parameters.
Configure resources.

You should now be able to configure a more complex model, which contains a product hierarchy, includes a Distributor role and a Manufacturer role, and uses routers and junctions.
