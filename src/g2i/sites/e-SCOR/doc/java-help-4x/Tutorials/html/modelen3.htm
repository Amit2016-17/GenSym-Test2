Creating a Model

A workspace can contain one or more models, such as an "As Is" model and a "To Be" model. You create a model by creating a Model object and placing e-SCOR objects on the model detail.

Most objects in e-SCOR have detail, which you access by choosing Show Detail on the object.

To create a Model:

1 Choose Toolbars from the View menu, choose the e-SCOR toolbar, then click the Tools tab.
2 Select the Model object from the toolbar and click to place it on your workspace.
3 Click right on the Model and choose Properties to display its properties dialog.
4 Edit the Label property to be Simplest Model.
Unlike workspace names, labels can contain spaces. e-SCOR places quotes around the label that appears next to the object.
5 Drag the label to the desired location next to the Model object, such as below.
You can drag objects beyond the workspace borders to expand the workspace size.
6 Drag the Model object slightly on the workspace to add white space around the object.
You can shrink wrap the borders of the workspace and the window to fit just its contents. You can also adjust the window size to be smaller than the workspace, which displays scroll bars on the window.
7 Choose Shrink Wrap from the Workspace menu.
