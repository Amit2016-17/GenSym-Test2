Viewing ED: Enable Delivery Category Metrics

The ED: Enable Delivery category computes:

Total orders received and total product shipments sent since the start of the simulation, which should be the same as for the Deliver category.
The percentage of orders that the role has delivered on or before the customer request date, which is based on the Desired Turnaround parameter of the source product of the downstream buyer role (Delivery Performance)
The percentage of orders that the role has delivered on or before the customer request date and for the requested quantities, which is based on the Completion Preference parameter of the source product of the downstream buyer role (Perfect Order Fulfillment)

To view ED: Enable Deliver category metrics for the Base Manufacturer role:

While the simulation is paused, display the properties dialog for the ED: Enable Delivery category on the Base Manufacturer role detail and view these metrics:
