Mapping Inputs for Base Manufacturer Role

A Base Manufacturer role uses make planning to determine the size and frequency with which it creates and sends build orders to manufacturing. You configure make planning parameters in the P3: Plan Make category and in the delivery product associated with the role.

Similar to the Distributor roles, you must configure the Net Selling price of the delivery product so the downstream central distributor can compute Inventory Days of Supply.
