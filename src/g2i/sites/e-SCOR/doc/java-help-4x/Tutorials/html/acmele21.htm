Configuring Product Composite Parameters: ACME Model

You configure different parameters for different products composites, depending on whether it is a source or delivery product, or both.

Tip: To avoid skewed metrics at the beginning of the simulation when a role has not yet built up any inventory, configure the Starting Inventory to be the same as the Replenishment Threshold.

To configure product composite parameters for the ACME model:

1 Using the Product Parameters Report that you created, configure the parameters for the product composites, according to the topics below, where all time-based values are in hours:
Computer Source Product for Consumer Role
Case Source Product for Computer Manufacturer Role
Electronics Source Product for Computer Manufacturer Role
Computer Delivery Product for Computer Manufacturer Role
Electronics Source and Delivery Product for Electronics Distributor Role
Electronics Delivery Product for Base Manufacturer Electronics Role
Case Delivery Product for Base Manufacturer Cases Role
2 In Excel, choose Report > Apply from the e-SCOR floating toolbar or press Ctrl + A to apply the parameter values to the model.
3 Save the report to the file My-ACME-Summary-Reports.xls.
