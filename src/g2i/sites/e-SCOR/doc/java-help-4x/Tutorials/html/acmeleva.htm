Configuring Level 2 SCOR: ACME Model

In the simplest model, you configured Level 2 SCOR parameters through the dialogs for each category and each product composite in the model. As you build more complex models, you will find that it is easier to configure all Level 2 parameters for categories and product composites in a single location, using input reports.

You can create input reports locally in the e-SCOR client, or you can create them in Excel, in a CSV file, or in a database. In this tutorial, you will create the input reports in Excel.

To configure Level 2 SCOR parameters for the ACME model, you:

Create input reports for configuring category and product composite parameters, using default report templates.
View and edit visible attributes and filters for each type of report.
Configure Level 2 category parameters through an input report.
Configure product composite parameters through an input report.
Create output reports for analyzing the performance of the model.
Configure the simulation to run for a fixed duration.
Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-ACME.kb, which shows the finished model, then, in Excel, open the file named Tutorial-ACME-Summary-Reports. xls, which shows the reports. Follow along in the existing model and reports as you read through the exercises.
