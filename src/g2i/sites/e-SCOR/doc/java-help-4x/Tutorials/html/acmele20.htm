Topic - Configuring Level 2 SCOR: ACME Model

D1 of Electronics Distributor (Deliver Parameters Report)

Parameter
Description

Order Entry Min = .5
Order Entry Mode = 1
Order Entry Max = 1.5

Takes 1 hour, on average, to enter orders for electronics components.

Pick Min = .5
Pick Mode = 1
Pick Max = 1.5

Takes 1 hour, on average, to pick electronics from inventory for delivery.

Packing Min = .5
Packing Mode = 1
Packing Max = 1.5

Takes 1 hour, on average, to pack electronics in containers for shipment.

Shipping Min = 24
Shipping Mode = 48
Shipping Max = 72

Takes 2 days, on average, to ship containers of electronics to buyers.

Order Entry Costs = 2
Order Fulfillment Costs = 2
Distribution Costs =2
Transportation Costs = 5
Invoice Cost =5
Collection Cost = 5

Costs $2 to enter, fulfill, and distribute orders for electronics, and costs $5 to transport containers, send invoices, and collect payments.
