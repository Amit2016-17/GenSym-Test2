Viewing Plan Category Metrics

The P2: Plan Source and P3: Plan Make categories compute the number of planning periods that have passed since that start of the simulation.

To view Plan category metrics for the Base Manufacturer role:

While the simulation is paused, display the properties dialog for the P3: Plan Make category on the Base Manufacturer role detail and view this metric:

Tab
Metric

Planning

Number of Planning Periods
