Creating Output Reports: Simplest Model

You can create reports locally within e-SCOR. You can also send the report data to Excel, to a CSV (Comma Separate Values) file or to a database, which means you have all the capabilities available these programs for analyzing the performance of your model.

To analyze the performance of the simplest model, you:

Generate output report data for viewing static role data.
Generate time-series data for viewing role metrics over time.
Generate output reports in Excel for further analysis.
Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-Simplest-Model.kb, which shows the finished model. To view the Excel report, open the file named Tutorial- Simplest-Model-Summary-Reports.xls. Follow along in the existing model as you read through the exercises.
