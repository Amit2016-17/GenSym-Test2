Configuring the Product Specification: Simplest Model

A supply chain can process one or more products. You model this by creating a product hierarchy, which consists of product specifications for each product that the supply chain sources, makes, and delivers.

In the simplest model, the product hierarchy consists of a single product specification, which represents an entire product family.

In a more realistic model, the product hierarchy consists of a finished product with one or more components, where the components themselves can contain components. You will learn how to create a product hierarchy with components later in this tutorial.

You create the product hierarchy on the detail of a Product Structure.

To create a product specification for the simplest model:

1 Display the e-SCOR toolbar, click the Products tab, select a Product Structure, and place it on the model detail.
2 Display the properties dialog for the Product Structure.
To display the properties dialog for any e-SCOR object on a workspace, you have a number of options:
Right-click the object and choose Properties.
Double-click the object.
Left-click the object to select it and press the F4 key.
Select the object and choose Properties from the Edit menu.
Note: From now on, this tutorial simply instructs you to configure a particular parameter; it does not specifically tell you to display the properties dialog.
3 Configure the Label to be Product Structure, then drag the label to below the object.
4 Right-click the Product Structure to show its popup menu, then choose Show Detail on the Product Structure to display its detail.
You place the product specification on this detail.
Note: From now on, this tutorial simply instructs you to choose a menu choice on an object; it does not specifically instruct you to right-click the object to display its popup menu. Similarly, from now on, this tutorial simply instructs you to show the detail of an object; it does not specifically instruct you to choose Show Detail.
5 Choose a Product Specification from the toolbar and place it on the product structure detail.
6 Configure the Product Name/SKU to be Product Family.
Tip: Shrink wrap the detail and adjust the borders of the window.
