Using Summary Tables and Charts

To facilitate data entry and analysis, you can use Excel macros and Visual Basic to create summary tables that include key parameters and metrics. To provide a graphical interface to the output data, you can use Excel to create charts that plot data values over time.

The file named Tutorial-VMI-Summary-Report.xls provides these summary tables and charts for you.

Note: From now on in this tutorial, use the report named Tutorial-VMI-Summary- Reports.xls for entering values into the model and viewing metrics.

To use these summary tables and chart:

1 In Excel, open Tutorial-VMI-Summary-Reports.xls.
2 Click No to the message that appears about updating links.
This message refers to a static chart that was created, based on data in another spreadsheet.
3 To view the summary tables, click the Summary tab.
You can now configure parameters and view metrics through these summary tables.
4 In the Model Parameters summary table, enter the Replenishment Level estimates you used earlier, as follows:

5 Click the Apply to Model button to apply the values to the model.
Clicking this button runs a Visual Basic macro that copies values from the summary table to the appropriate input reports, then applies the values to the model.
6 Run the simulation.
Observe the Model Metrics as they update once every two weeks of simulation time.
At the end of the simulation, the Model Metrics summary table might have values something like these, based on the replenishment level estimates:

The Model Metrics summary table displays the Supplier On-Time Performance (%) of the three distribution sites and the Inventory Days of Supply for the distribution sites and the central distributor. It also computes the inventory carrying cost and opportunity cost of that inventory, which you will use later in this tutorial.
Notice that the on-time performance is nowhere near 100%, because the replenishment level estimates are too low.
7 Click the Inventory Chart tab and observe the Inventory Level plot as it updates once a day for each of the three distribution sites and for the central distributor.
