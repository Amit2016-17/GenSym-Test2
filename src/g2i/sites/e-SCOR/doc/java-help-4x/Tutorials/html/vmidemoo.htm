Optimizing the Central Distributor Model

Based on the results you obtained by using estimates, you will now fine-tune the estimates to optimize the central distributor model. To optimize the model, you:

Determine optimal replenishment levels.
The goal is to achieve 100% on-time performance, given the lowest possible replenishment levels and the lowest inventory carrying costs, measured in Inventory Days of Supply. To do this, first, you will optimize replenishment levels by configuring the model for the worst case scenario, then you will attempt to minimize inventory days of supply, given normal demand and delivery times. You will find that the replenishment levels need to be significantly higher than you originally estimated to account for variation in order size and delivery times.
Implement a vendor-managed inventory control system in which the central distributor owns and controls all the inventory-its own, as well as that of each of its distribution sites.
By implementing a vendor-managed inventory control system, the central distributor believes it can make deliveries more frequently, which means it can carry less inventory centrally. You will find that the central distributor does not need to carry that much more inventory than it does when it owned only its own inventory. Therefore, the decrease in inventory levels required due to more frequent deliveries to some extent offsets the increase in inventory levels required for each of the distribution sites.
Analyze the opportunity cost of providing different service levels to different customers, based on demand.
By comparing opportunity costs of different levels of on-time performance, you can make key business decisions regarding pricing and contracts.
Using the Scenario Manager with the central distributor tutorial model.
The Scenario Manager allows you to define a script for running multiple simulations.
