Generating Time-Series Data

You might want to compare metric values over time. To do this, you configure the report to keep a history. You can configure any type of output report to keep a history. The output report timestamps each value in the report.

To generate time-series data:

1 Display the properties dialog for the Role Output Report and enable the Keep History option.
When you change the configuration of the report object, you must update the report.
2 Choose Update Report on the report object or select the report object and choose Update from the Reports menu.
3 Run the simulation again.
This time, the report contains separate rows with timestamps for each new value.
4 Reset the simulation.
