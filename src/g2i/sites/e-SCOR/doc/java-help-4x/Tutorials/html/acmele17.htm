D1 of Base Manufacturer Electronics (Deliver Parameters Report)

Parameter
Description

Order Entry Min = 3
Order Entry Mode = 4
Order Entry Max = 5

Takes 4 hours, on average, to enter orders for electronics.

Pick Min = 1
Pick Mode = 2
Pick Max = 3

Takes 2 hours, on average, to pick electronics from inventory for delivery.

Packing Min = 1
Packing Mode = 2
Packing Max = 3

Takes 2 hours, on average, to pack electronics into containers for shipment.

Shipping Min = 48
Shipping Mode = 72
Shipping Max = 96

Takes 3 days, on average, to ship containers of electronics to buyers.

Order Entry Costs = 5
Order Fulfillment Costs = 5
Distribution Costs =5
Transportation Costs = 10
Invoice Cost =10
Collection Cost = 10

Costs $5 to enter, fulfill, and distribute orders for electronics, and costs $10 to transport containers, send invoices, and collect payments.
