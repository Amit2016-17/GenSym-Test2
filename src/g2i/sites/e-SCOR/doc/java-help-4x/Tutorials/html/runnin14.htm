Topic - Running the Simulation: Simplest Model

Summary: Running the Simulation for the Simplest Model

To run the simulation for the simplest model, you:

Run the simulation.
View role metrics.
View Source category metrics.
View ES: Enable Source category metrics.
View Deliver category metrics.
View ED: Enable Delivery category metrics.
View Make category metrics.
View Plan category metrics.
View delivery product metrics.
View source product metrics.

You should now know how to run a simulation and view metrics for roles and Level 2 SCOR categories and product composites for the simplest e-SCOR model.

Next, you will look at the underlying processes that e-SCOR runs when you run a simulation.
