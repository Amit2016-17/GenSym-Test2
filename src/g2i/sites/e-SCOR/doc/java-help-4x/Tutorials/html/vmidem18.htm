Creating Output Reports for Viewing Key Metrics

You will create two output reports for viewing key parameters:

Level 2 Output Report, which will contain the supplier on-time performance metric for the ES: Enable Source category of each Consumer role.
Role Output Report, which will include the asset metrics for each role.

This table summarizes the attributes that should appear in each output report, all of which are in the General group:

Level 2 Output Report

Role Output Report

Cash Flow Period

You will configure the reports to update at regular time intervals, based on simulation time.

To create output reports for viewing key metrics:

1 Create a Level 2 Output Report and a Role Output Report from the Reports tab of the e-SCOR toolbar and place them on the model detail.
2 Configure the attributes to appear in the reports, according to the table above.
For details, see Creating Input Reports for Configuring Key Parameters.
3 Choose Create Report on each report, then configure the spreadsheets to facilitate viewing.
Tip: For the Level 2 Output Report, hide all categories except the ES: Enable Source category.
For details, see Creating Input Reports for Configuring Key Parameters.
4 Configure the Level 2 Output Report to update once a week and configure the Role Output Report to update once every two weeks.
For details, see Generating Output Report Data.
5 Save the reports to the file named My-Tutorial-VMI-Summary-Reports.xls.
