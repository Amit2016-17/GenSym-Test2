Configuring Level 2 Category Parameters: ACME Model

Level 2 category parameters apply to all products that a role sources, makes, and/or delivers. For example, when configuring Source category parameters for the Computer Manufacturer role, you configure the time it takes to receive and place into inventory both cases and electronics, and the cost of receiving both of these components.

If the products use a different planning strategy-stock, make-to-order, or engineer-to-order-you can add categories of the appropriate type to the role detail and configure each category separately.

Because a Manufacturer role both sources components and delivers finished products, it uses both source and make planning. You must coordinate the timing of source and make planning by configuring the initial delays for initiating each type of planning.

Tip: In pull planning mode, configure the Initial Plan Delay for the P3: Plan Make category to be smaller than the Initial Plan Delay for the P2: Plan Source category. That way, make planning occurs before source planning.
Note: When you configure time-based parameters through reports, you must enter all values in the time unit of the report, which is 1 hour, by default. For example, to enter half an hour, enter .5 and to enter 1 day, enter 24. You can configure the time unit of the report to be seconds, minutes, hours, days, or weeks, depending on what is most convenient for your model.

To configure Level 2 category parameters for the ACME model:

1 Using the Level 2 Input Reports that you created, configure the parameters for the Level 2 categories, according to the topics below, where all time-based values are in hours:
Sc of Consumer (Demand Parameters Report)
Sc of Consumer (Source Parameters Report)
S1 of Computer Manufacturer (Source Parameters Report)
S1 of Electronics Distributor (Source Parameters Report)
P3 of Base Manufacturer Cases (Plan Parameters Report)
P3 of Base Manufacturer Electronics (Plan Parameters Report)
P2 of Electronics Distributor (Plan Parameters Report)
P3 of Computer Manufacturer (Plan Parameters Report)
P2 of Computer Manufacturer (Plan Parameters Report)
Mb of Base Manufacturer Cases (Make Parameters Report)
Mb of Base Manufacturer Electronics (Make Parameters Report)
M1 of Computer Manufacturer (Make Parameters Report)
D1 of Base Manufacturer Electronics (Deliver Parameters Report)
D1 of Base Manufacturer Cases (Deliver Parameters Report)
D1 of Computer Manufacturer (Deliver Parameters Report)
D1 of Electronics Distributor (Deliver Parameters Report)
2 In Excel, choose Report > Apply from the e-SCOR floating toolbar or press Ctrl + A to apply the parameter values to the model.
3 Save the report to the file My-ACME-Summary-Reports.xls.
