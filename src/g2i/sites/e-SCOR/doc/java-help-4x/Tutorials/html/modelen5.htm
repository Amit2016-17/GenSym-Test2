Saving a Model

You save models to a .kb file. The first time you save a model, you configure the filename to save. The next time you save the same model, you do not need to enter the filename.

You will save the models you build in these tutorials to the file named My-Tutorials.kb.

To save a model:

Choose File > Save and enter My-Tutorials.kb.
