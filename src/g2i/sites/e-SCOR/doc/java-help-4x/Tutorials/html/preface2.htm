Tutorial Models

At various points in these tutorials, you have the option of loading finished models and summary reports that are solutions to the individual tutorials. If you do not have time to work through the tutorials yourself, you can use these models and reports instead.

To start e-SCOR with a tutorial model open:

Choose Start > Programs > Gensym e-SCOR & ReThink > Examples > e-SCOR, then choose the tutorial model and corresponding report to load.

To open a tutorial model in e-SCOR:

Choose File > Open, then navigate to the kbs directory of your e-SCOR product directory and choose the tutorial model to open.
