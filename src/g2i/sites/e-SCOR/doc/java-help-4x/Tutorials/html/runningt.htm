Running the Simulation: Simplest Model

You run the simulation from the Simulation menu or from the toolbar. When you run the simulation, e-SCOR computes various metrics, which you can view. You can:

View role metrics.
View Source category metrics.
View ES: Enable Source category metrics.
View Deliver category metrics.
View ED: Enable Delivery category metrics.
View Make category metrics.
View Plan category metrics.
View delivery product metrics.
View source product metrics.

You can view these metrics through the properties dialogs, through the Metrics toolbar, or through reports. In this section, you will view these metrics through the dialogs and Metrics toolbar. Later in this tutorial, you will learn how to create reports of e-SCOR metrics.

Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-Simplest-Model.kb, which shows the finished model. Follow along in the existing model as you read through the exercises.
