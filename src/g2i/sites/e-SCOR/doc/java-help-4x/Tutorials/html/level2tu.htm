Configuring Level 2 SCOR: Simplest Model

Each role has detail, which provides the default configuration of Level 2 SCOR for the particular role. Level 2 SCOR includes:

Process categories, which define how the role plans, sources, makes, and delivers products, as appropriate for the role.
Product composites, which define the source and delivery products that a role sources and/or delivers.

Process categories are also referred to simply as "categories."

You configure various parameters for each category and each product composite on the role detail. The parameters you configure for categories apply to all products the role sources and/or delivers, such as the time it takes to manufacture a batch of products. On the other hand, the parameters you configure for product composites apply to individual source and delivery products, such as the starting and maximum inventory for a particular product.

You configure Level 2 SCOR parameters on the various tab pages of the properties dialog for categories and product composites. These dialogs also include the metrics the model computes for each category and product composite.

SCOR metrics appear in bold, both in the dialogs and in the text.

Throughout this tutorial, you can click the name of any e-SCOR parameter or metric to go to its definition in the e-SCOR online help.

For a quick reference to the definition of all parameters and metrics, see the e-SCOR Quick Reference. For the SCOR definition of all SCOR metrics, see Appendix B, SCOR Metrics in the e-SCOR User's Guide.

To configure Level 2 SCOR for the simplest model, you:

View the Consumer role details, then configure:
- Source category parameters
View the Base Manufacturer role details, then configure:
- Make category parameters
- Plan category parameters
- Deliver category parameters
View product composites, then configure:
- Product composite parameters
- Delivery product parameters
- Source product parameters.
Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-Simplest-Model.kb, which shows the finished model. Follow along in the existing model as you read through the exercises.
