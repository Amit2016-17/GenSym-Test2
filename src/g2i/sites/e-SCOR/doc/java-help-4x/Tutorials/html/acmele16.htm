Parameter
Description

Mfg Min = 7
Mfg Mode = 8

Move To Delivery Min = 1
Move To Delivery Mode = 2
Move To Delivery Max = 3

Takes 2 hours, on average, to transfer computers from where they are manufactured to where they are delivered.
