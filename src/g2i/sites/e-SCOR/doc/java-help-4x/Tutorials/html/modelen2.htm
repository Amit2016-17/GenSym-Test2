Creating a Workspace

When you create a top-level workspace, you must name it so you can retrieve it later.

You navigate to named workspaces by using the Workspace menu or the Workspace Browser. You can scale, shrink wrap, print, and edit the colors of workspaces by using the Workspace menu.

To create a workspace:

1 Choose New Workspace from the Workspace menu.
2 Choose Properties from the Workspace menu and edit the workspace Name to be getting-started.
Note: The name cannot contain spaces; use hyphens or underscores instead.
