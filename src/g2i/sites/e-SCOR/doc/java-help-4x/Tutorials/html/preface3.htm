e-SCOR Tutorials
Topic - Welcome

Audience

This tutorial is for new e-SCOR users who want step-by-step instructions that teach you how to build, run, and analyze e-SCOR models. It is also a good place to start to see examples of kinds of models you can build and the kinds of problems you can solve, using e-SCOR.

Users should be familiar with supply-chain modeling and have some familiarity with SCOR.

If you are not familiar with SCOR, visit www.supply-chain.org.
