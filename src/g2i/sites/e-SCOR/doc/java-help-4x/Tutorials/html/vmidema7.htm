Topic - Running the Simulation: Central Distributor Model

Summary: Running the Central Distributor Simulation

To run the central distributor simulation, you:

Estimate initial replenishment levels, based on order size and order frequency.
Run the simulation and view the on-time performance and days of supply.
Create time-series reports for analyzing inventory levels and days of supply over time.
Create summary tables and charts for entering key parameters, viewing key metrics, and viewing inventory levels graphically over time.

You should now have a better understanding of how replenishment levels, order size, and order frequency affect on-time performance.
