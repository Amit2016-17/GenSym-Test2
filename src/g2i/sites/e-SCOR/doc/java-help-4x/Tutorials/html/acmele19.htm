Topic - Configuring Level 2 SCOR: ACME Model

D1 of Computer Manufacturer (Deliver Parameters Report)

Parameter
Description

Order Entry Min = 7
Order Entry Mode = 8
Order Entry Max = 9

Takes 8 hours, on average, to enter orders for computers.

Pick Min = .5
Pick Mode = 1
Pick Max = 1.5

Takes 1 hour, on average, to pick computers from inventory for delivery.

Packing Min = .5
Packing Mode = 1
Packing Max = 1.5

Takes 1 hour, on average, to pack computers in containers for shipment.

Shipping Min = 48
Shipping Mode = 72
Shipping Max = 96

Takes 3 days, on average, to ship containers of computers to buyers.

Order Entry Costs = 5
Order Fulfillment Costs = 5
Distribution Costs =5
Transportation Costs = 10
Invoice Cost =10
Collection Cost = 10

Costs $5 to enter, fulfill, and distribute orders for computers, and costs $10 to transport containers, send invoices, and collect payments.
