Configuring Role Parameters: ACME Model

The ACME model uses the default Financial Period for all roles, 30 days, to determine when the role computes financial, asset, and cost metrics. It also uses the default Financial Payment Terms, also 30 days, to determine when buyer roles make payments to supplier roles.

For the purposes of organizing report data by role, you can configure each different type of role in the model to use a different Sequence Number. You can also configure the Highlight Color of each role to visually distinguish roles of different types.

The ACME model uses a pull planning strategy throughout the supply chain, which means buyers place orders with suppliers, based on source planning. Therefore, you can use the default values for Source Type and Delivery Type, which are both pull.
