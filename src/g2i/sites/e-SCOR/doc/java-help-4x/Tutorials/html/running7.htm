Viewing Make Category Metrics

The Make category computes:

Total build orders started and completed since the start of the simulation.
Average time it takes from when the role receives a build order for delivery products to when the role has finished manufacturing those products (Make Cycle Time).

To view Make category metrics for the Base Manufacturer role:

While the simulation is paused, display the properties dialog for the Mb: Make Base Manufacturer category on the Base Manufacturer role detail and view these metrics:

Tab
Metric

Manufacturer

Build Orders Started
Build Orders Completed
Make Cycle Time (Days)
