Running the Simulation: Central Distributor Model

Before you can run the central distributor model simulation, you must configure the Replenishment Threshold parameters for each of the Distributor roles in the model. The Replenishment Threshold determines the level at which each distribution site, as well as the central distributor replaces its inventory. Each Distributor role uses the Replenishment Threshold and the Minimum Reorder Quantity, along with the current number of products on order, to determine order size when performing its source planning.

To avoid skewed metrics due to lack of initial inventory, you will also configure the Starting Inventory to be the same as the Replenishment Threshold.

To run the central distributor model, you:

Estimate initial replenishment levels to approximate the levels you think will achieve the desired results.
Run the simulation to determine whether the end consumers achieve 100% on-time performance.
Create two time-series reports that report inventory levels and asset metrics over time.
Use summary tables and charts to configure key parameters and view key metrics, numerically and graphically, by opening a pre-existing Excel spreadsheet.
Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-VMI.kb, which shows the finished model, then, in Excel, open the file named Tutorial-VMI-Summary-Reports.xls. Follow along in the existing model and reports as you read through the exercises.
