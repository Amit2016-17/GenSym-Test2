Viewing ES: Enable Source Category Metrics

Each ES: Enable Source category computes:

Total orders sent and total shipments received since the start of the simulation. The total shipments received should be the same as total orders sent and total shipments received for the Source category.
The number of contracts the role has established with upstream suppliers. When no contracts are in effect, the number of contracts established is 1, which means the buyer role has a single supplier for its source products. When contracts are in effect, the buyer might have several suppliers, in which case the number of contracts established could be a number greater than 1.
The on-time performance of the supplier, which is the percentage of shipments that the role has received on or before the customer request date, which is based on the Desired Turnaround of the role's source product (Supplier On-Time Performance).

When you run the simulation, the Supplier On-Time Performance should be around 90%.

To view ES: Enable Source category metrics for the Consumer role:

While the simulation is paused, display the properties dialog for the ES: Enable Source category on the Consumer role detail and view these metrics:
