Using the Scenario Manager with the Central Distributor Tutorial Model

The tutorial KB provides the Scenario Manager script that you can use to run multiple simulations. The tutorial reports provide sample data and graphs generated from that data that plot On-Time Delivery against Inventory Cost and On-Time Delivery against Replenishment Level for each of the three distribution sites.

To view the script:

Open the Tutorials-VMI.kb and show the properties dialog for the Scenario Manager.

The script runs six simulations each for three sets of data values for Replenishment Level and Starting Inventory for each of the three distribution sites and the central distributor. At the end of each simulation, this script writes data to a Free Form Report. The Free Form Report outputs the Supplier On-Time Performance (%) and the Average Inventory of each distribution site and the central distributor.

To view the Free Form Report:

1 Display the properties dialog for the Free Form Report in the model.
2 Click the Attributes tab to view the attributes that the report writes to the various cells in the report.
The report uses UUIDs to add attributes to the report through the dialogs.
3 Click the Excel tab.
The Free Form Report uses the DefaultFormatData macro in Excel to move the row of data down one row after it inserts the data to make room for the next set of data. To do this, the report configures the Excel Macro to Format Data parameter to be DefaultFormatData, which is defined in Visual Basic in the report.
4 In Excel, click the Free Form Summary Report tab.

The Scenario Manager writes data to this sheet at the end of each simulation run.

To view the sample graphs:

In Excel, click the Sample Free Form Reports tab.

The two sample graphs appear. Scroll left in the sheet to view the sample data that was used to generate these graphs. The first nine columns are the raw data for each of the three distribution sites. The next set of columns convert the raw data to the units of thousands of dollars for use in creating the graphs.

To use the Scenario Manager to generate your own data:

Display the properties dialog for the Scenario Manager and click Start.

The simulation starts running. When the first simulation is complete, the Scenario Manager updates the first row of report data in the Free Form Summary Report sheet, then moves the data down one row, according to the macro.

The Simulation Manager runs 18 simulations, writing data to the report at the end of each simulation. This process might take approximately a day, depending on the speed of your computer.
