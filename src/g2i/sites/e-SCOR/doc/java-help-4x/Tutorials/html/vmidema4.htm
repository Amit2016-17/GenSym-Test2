Generating Time-Series Data

You might want to know how inventory levels fluctuate over the time during the simulation. You could then use this data to create an Excel chart that plots the inventory level over time for each distribution site and the central distributor.

To do this, you create an N Dimensional Output Report, which includes metrics for individual objects of any type in the model. The simplest way to create an N Dimensional Output Report is to add attributes to the report directly from the dialogs.

You will create separate N Dimensional Output Reports that generate a history of the Inventory Level metric for the product composites of the central distributor and each distribution site.

To generate time-series data:

1 Create four N Dimensional Output Reports from the Reports tab of the e-SCOR toolbar and place them on the model detail.
2 Configure the Report Title parameters to be Central, A, B, and C, respectively.
3 Configure each report to update once a day.
4 Configure each report to keep a history by enabling the Keep History option.
5 Configure the attributes to appear in the Inventory Level report:
a Display the product composite dialog for the Central Distributor role and click the button next to the Inventory Level metric on the Inventory tab, as follows:

The list of available reports to which you can add this attribute appears.
b Move the report named Central into the left-hand list in the dialog to add the attribute to the report, as follows:

c Repeat steps a and b to add the Inventory Level associated with each distribution site's product composites to the reports named A, B, and C, respectively.
d Display the properties dialog for each report and click the Attributes tab to see the attribute you added, as follows:

In an N Dimensional Output Report, you must configure the Object Label or UUID (unique ID) for each attribute. When you add the attribute through the dialogs, e-SCOR uses the UUID; otherwise, you can configure Object Label associated with each object manually.
6 Choose Create Report on each report to create the report template in Excel.
7 Now, run the simulation again for several days, view the time-series report data for the inventory level and days of supply, then reset the simulation.
