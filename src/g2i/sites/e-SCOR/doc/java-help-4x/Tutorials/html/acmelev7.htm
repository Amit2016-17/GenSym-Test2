Configuring Resources: ACME Model

These roles in the ACME model require manufacturing resources:

Base Manufacturer: Cases
Base Manufacturer: Electronics
Computer Manufacturer

In the ACME model, each manufacturing role uses its own resource.

To configure resources for the ACME model:

1 Create and label three manufacturing resources on a resource map.
2 Assign each resource to each role, as outlined above.
3 Verify the assignment of each resource.

For details, see Configuring Resources: Simplest Model.
