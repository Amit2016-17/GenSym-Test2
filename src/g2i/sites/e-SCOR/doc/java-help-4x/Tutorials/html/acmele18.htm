D1 of Base Manufacturer Cases (Deliver Parameters Report)

Parameter
Description

Order Entry Min = 7
Order Entry Mode = 8
Order Entry Max = 9

Takes 8 hours, on average, to enter orders for cases.

Pick Min = .5
Pick Mode = 1
Pick Max = 1.5

Takes 1 hour, on average, to pick cases from inventory for delivery.

Packing Min = .5
Packing Mode = 1
Packing Max = 1.5

Takes 1 hour, on average, to pack cases into containers for shipment.

Shipping Min = 168
Shipping Mode = 240
Shipping Max = 312

Takes 10 days, plus or minus 3 days, to ship containers of cases to buyers.

Order Entry Costs = 5
Order Fulfillment Costs = 5
Distribution Costs =5
Transportation Costs = 10
Invoice Cost =10
Collection Cost = 10

Costs $5 to enter, fulfill, and distribute orders for cases, and costs $10 to transport containers, send invoices, and collect payments.
