Configuring the Product Hierarchy: ACME Model

In the simplest e-SCOR model, the product hierarchy consisted of a single product specification, which represented an entire product family. In the ACME model, the product hierarchy consists of:

A computer finished product.
A case component.
Three electronics components-mother board, power supply, and drives.

Because the Manufacturer role sources the three electronics components from a single Distributor role, you can treat them as a single product family that consists of three units.

To configure the product hierarchy:

1 Create a product specification named Computer and place it on a Product Structure detail.
For details, see Configuring the Product Specification: Simplest Model.
2 Choose Allow Components on the computer, which creates detail for the product specification.
You create components on this detail.
3 Create a product specification named Case and place it on the computer detail.
4 Create a product specification named Electronics and place it on the computer detail.
5 Configure the Product Quantity of the electronics component to be 3 to represent the three electronics components.
