Configuring the Model Environment: ACME Model

First, you must set up the model environment for the ACME model. To do this, you will create a new model on the Getting Started workspace, which you created earlier in this tutorial.

To set up the ACME model environment:

1 Go to the Getting Started workspace by using one of these two techniques:
Choose Go To on the Getting Started workspace in the Model Browser, which appears on the left-side of the e-SCOR window.
or
Choose Workspace > Get Workspace and select the Getting Started workspace.
2 Create a Model named ACME Model and place it on the Getting Started workspace.
For details, see Creating a Model.
3 Place a Scenario on the model detail.
For details, see Creating a Scenario.
