Configuring Make Category Parameters: Base Manufacturer Role

The Make category is responsible for creating batches from build orders, selecting build orders for manufacture, and manufacturing delivery products. Therefore, you configure timing, batch, and build-selection parameters related to manufacturing delivery products.

For a Manufacturer role, you configure the Production Material Cycle Time and the Production Material Handling cost of the Make category to determine the time from when the role receives components to when it starts manufacturing batches and the cost of handling those components. However, because a Base Manufacturer role does not source components, you do not configure these parameters for this role.

Base Manufacturer and Manufacturer roles use make planning to determine the size of each build order and when to start manufacturing batches, based on orders for its delivery products. Therefore, you configure planning parameters in the P3: Plan Make category and in the delivery products for the role.

Normally, the Order Type of the source and delivery product of a role determines the planning strategy the role uses to source, make, and deliver its products-stock, make-to-order, or engineer-to-order. However, because a Base Manufacturer role has no upstream suppliers, you configure the Order Type for the Mb: Make Base Manufacturer category to determine how the role sources its components.

By default, a Base Manufacturer uses a stock planning strategy to determine build order size, based on orders for its delivery products. If the role uses an engineer-to-order planning strategy, you configure additional parameters for the Base Manufacturer role related to engineering time and costs.

A Base Manufacturer role always chooses build orders based on a first-in first-out (FIFO) selection criteria. A Manufacturer role that makes stock products, on the other hand, can choose build orders, based on a set of build-selection criteria that you configure in the Make category.

Configure the Mb category so that it:

Manufactures stock products, the default.
Takes 1 day, on average, from when the role receives a build order to when it begins manufacturing batches.
Takes 10 hours, on average, to manufacture a batch of product families.
Takes 1 hour, on average, to place the finished products in inventory.
