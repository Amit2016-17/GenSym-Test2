Topic - Configuring Level 2 SCOR: ACME Model

Summary: Configuring Level 2 SCOR for the ACME Model

To configure Level 2 SCOR parameters for the ACME model, you:

Create input reports.
View and edit visible attributes and filters.
Configure Level 2 category parameters.
Configure product composite parameters.
Create role output reports.
Configure the simulation to run for a fixed duration.

You should now know how to configure Level 2 SCOR parameters for categories and product composites through an input report. You are now ready to run the simulation.
