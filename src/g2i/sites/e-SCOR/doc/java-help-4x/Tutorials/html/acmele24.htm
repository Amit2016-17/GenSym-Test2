Topic - Configuring Level 2 SCOR: ACME Model

Electronics Source Product for Computer Manufacturer Role

Parameter
Description

Starting Inventory = 150
Inventory Maximum = 1000

Starts with 150 electronics components in inventory, which is three times the number of cases.

Desired Turnaround = 168

Wants to receive electronics within 7 days of placing an order.

Replenishment Threshold = 150
Minimum Reorder Quantity = 100
Maximum Threshold = 500

Replenishes inventory when fewer than 150 electronics components exist, which is three times as many as the number of cases. Reorders a minimum of 100 electronics components at a time.
