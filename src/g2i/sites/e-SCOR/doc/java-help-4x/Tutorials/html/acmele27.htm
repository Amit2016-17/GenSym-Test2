Topic - Configuring Level 2 SCOR: ACME Model

Electronics Delivery Product for Base Manufacturer Electronics Role

Parameter
Description

Starting Inventory = 200
Inventory Maximum = 10000

Starts with 200 electronics components in inventory.

Maximum Batch Size = 50
Minimum Batch Size = 25
Yield = .80

Builds electronics in batches with a minimum size of 50 units and a maximum size of 25 units. Has an 80% manufacturing yield.

Net Selling Price = 50
Published Delivery Lead Time = 24

Charges $50 to buyers for each electronics component. Publishes that it can deliver electronics components within 1 day of receiving an order. Has an 80% manufacturing yield.

Replenishment Threshold = 200
Minimum Reorder Quantity = 50
Maximum Threshold = 500

Replenishes inventory when fewer than 200 electronics components exist. Creates build orders for a minimum of 50 electronics components at a time.
