Viewing Role Metrics

Each role computes:

Incoming financial metrics related to supplier bookings, outstanding, and collections for products shipped.
Outgoing financial metrics related to buyer obligations and payments for products received.
Days of supply metrics for source and delivery product inventories.
Other standard asset calculations.
Total cost of doing business.

A Consumer role computes only outgoing financial metrics.

You can view role metrics through the role dialog, or you can view financial and asset metrics through the Metrics toolbar, which appears automatically when the model runs.

The role computes total financial collections and payments since the start of the simulation, as well as for each financial period.

Note: The model computes total financial collections and payments with each new event; therefore, these metrics are always current. The model computes financial collections and payments for the period at the end of each financial period; therefore, these metrics refer to collections and payments for the previous financial period.

The financial and asset metrics for roles require that several financial cycles complete before the role begins computing values.

When you view metrics through the dialogs, e-SCOR takes a snapshot at a particular point in time. You must manually update the metrics to view the latest values by clicking the Update button in the dialog.

When you view role metrics through the Metrics toolbar, the metrics update once every half second, automatically.

To view metrics for the Base Manufacturer and Consumer roles:

1 Run the simulation again.
When the simulation first starts running, e-SCOR automatically displays the Metrics toolbar at the bottom of the window. This toolbar contains key metrics for all roles and all product composites in the model. You click the various tabs in the toolbar to view different categories of metrics.
2 Click the Financial tab to view financial metrics for the roles.
The financial collections and payments begin to update after one financial period has completed.
3 Click the Assets tab to view asset metrics for the roles.
Similarly, the asset metrics begin to update after one financial period has completed.
4 Wait until at least four months of simulation time has past, which is four financial cycles for the Base Manufacturer role.
The simulation time should be some time after May 1. Now, you will view the metrics through the dialogs.
5 Display the properties dialog for each role to view these metrics:
