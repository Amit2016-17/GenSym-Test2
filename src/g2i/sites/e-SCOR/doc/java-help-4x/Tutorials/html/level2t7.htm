Configuring Deliver Category Parameters: Base Manufacturer Role

The Deliver category is responsible for entering and choosing orders for shipment, picking delivery products from inventory, packing containers, shipping containers to buyers, and invoicing and receiving payments from buyers. You configure timing and cost parameters related to processing and fulfilling orders for delivery products.

By default, a role chooses orders for shipment at random. You can configure the criteria the role uses to choose orders for shipment for stock products. For example, you can deliver orders on a first-in first-out basis, or you can deliver the most expensive orders first.

You can configure any logistical parameter, based on the carrier. Logistical parameters include all of the timing parameters and some of the cost parameters associated with the Deliver category, as well as the timing and cost parameters associated with receiving source products for the Source category. To do this, you configure the Carrier parameter for the Deliver category of the role, you configure the logistical parameter's Mode Type to use a Report Lookup, then you create a Logistical Lookup Report that specifies values for each of the logistical parameters, based on the carrier.

Configure the Deliver category so that it:

Takes 1 day, on average, to enter an order from when it is received.
Chooses orders for shipment on a first-in first-out basis, by sorting orders based on the smallest Internal Order Number first, the default.
Takes 1 hour, on average, to pick delivery products from inventory and to pack a container.
Takes 3 days, on average, to ship a container.
Costs $10 to enter, fulfill, distribute, ship, invoice, and receive payment for each order.
