Topic - Configuring Level 2 SCOR: ACME Model

Viewing and Editing Visible Attributes and Filters

Each visible attribute in the report belongs to one of a number of groups, depending on the type of report. Most attributes are in the General group. Attributes that allow you to configure the mathematical distribution, such as Order Type, Order Period, and all of the timing parameters, are in a group whose name corresponds to the attribute group in the dialog. For example, the Order Period group defines the Mode Type, Minimum, Mode, and Maximum attributes.

Caution: Each input report contains the UUID internal attribute, which uniquely identifies each object's attribute. Do not remove this attribute from the report; otherwise, editing parameter values in the report does not update the model.

Only the Level 2 Input and Output Reports allow you to define filters; the other report types do not.

Now, you will view the attributes and filters for each report. Because the ACME model uses pull planning, you will remove an attribute from the Product Parameters report that is related to push planning. You will also add parameters to the Demand Parameters report to enable you to configure a random exponential mode type.

To view and edit visible attributes and filters:

1 Display the properties dialog for each report and click the Attributes tab.
Tip: You can select all the reports and choose Properties from the Edit menu to open all the dialogs at once.
The group name appears in the left-hand column, and the visible attribute in each group appears in the right-hand column.
2 For each report, examine the visible attributes and their associated groups.
Notice that each report contains only those attributes that are relevant for the specified template. For example, the Demand Parameters report contains parameters for the Sc: Source Consumer category only.
You can add and remove attributes, as needed. To do this, you add rows to the end of the list, insert rows above a selected row, and/or remove selected rows.
3 In the Product Parameters report dialog, scroll down to the bottom of the list of visible attributes, select the Push Amount attribute, and click Delete Row(s).
4 In the Demand Parameters report dialog, locate the attributes in the Order Period group, select the row above the first attribute in the Order Period group, and click Insert Row.
By default, e-SCOR inserts the Label attribute in the General group.
5 Double-click the General group in the row you just created to display a dropdown list of all the available groups and choose Order Period.
6 Double-click the default attribute in the new row to display a dropdown list of all the available attributes in the Order Period group and choose Mode Type, which is at the bottom of the list.
Now, you will delete the Min and Max parameters in the Order Period group, used for configuring the order period, using a random triangular distribution.
7 Select the Min and Max attributes in the Order Period group and click Delete Row(s).
Tip: You can select several rows at once by holding down the Shift key as you select rows from the list.
The report is now configured to allow you to specify the Mode Type and the Mode of a random exponential function.
8 In each of the Level 2 Input Report dialogs, click the Filters tab.
Notice that each report includes data for the relevant categories only. For example, the Demand Parameters report contains data for the Sc: Source Consumer category only.
9 Save the report to the file My-ACME-Summary-Reports.xls.

You are now ready to configure the Level 2 SCOR parameters for the ACME model, using the input reports.
