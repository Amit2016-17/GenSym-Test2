Running the Simulation: ACME Model

When you run the simulation for the ACME model, you will determine model performance by asking a series of questions about key performance metrics, under two different scenarios:

Normal demand, where the customer orders an average of 25 computers a week.
Increased demand, where the customer orders an average of 50 computers a week.

You will experiment with modifying key parameters to improve the performance of the model under the scenario of increased demand.

Shortcut: If you do not have time to work through the exercises in this part of the tutorial, in e-SCOR, open the file named Tutorial-ACME.kb, then, in Excel, open the file named Tutorial-ACME-Summary-Reports.xls. You will need to edit the parameter values and run the simulation, as described in this tutorial.
