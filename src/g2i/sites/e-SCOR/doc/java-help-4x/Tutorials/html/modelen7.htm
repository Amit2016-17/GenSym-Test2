Topic - Configuring the Model Environment

Summary: Configuring the Model Environment

To configure the e-SCOR model environment, you:

Create a workspace.
Create a model.
Create a Scenario.
Save the model.
Start e-SCOR with your model open.

You should now know how to configure the model environment for the simplest model.

Next, you will configure Level 1 roles to create the supply-chain model.
