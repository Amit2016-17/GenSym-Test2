Configuring Resources: Simplest Model

The last step in configuring Level 1 roles is to configure the resources that each manufacturing role requires to make finished products. You configure resources for Base Manufacturer and Manufacturer roles only.

By default, each resource represents a single unit and takes the entire build time to manufacture a batch of finished products. You can configure the Resource Capacity to model the number of units that each manufacturing resource represents. You can also configure the Efficiency Factor to model variation in the build time for different resources.

Once you have created and configured the resources, you must assign a resource to each manufacturing role in the model. By assigning resources to roles, the role is constrained as to the number of finished products it can manufacture, based on the capacity.

To configure resources for the simplest model:

1 Select a Resource Map from the Resources tab of the e-SCOR toolbar, place it on the model detail, and show its detail.
2 Select a Plant resources from the toolbar and place it anywhere on the resource map detail.
3 Configure this parameter for the plant:

Parameter
Value

Resource Label

 Product Family Base Manufacturer

Tip: Typically, you configure the Resource Label to correspond to the Role Label.
4 Close the resource map detail.
You assign resources to a role through the role's dialog.
5 Display the properties dialog for the Base Manufacturer role, click the Resources tab, and choose the Product Family Base Manufacturer resource from the dropdown list as the value of the Resource parameter.

The resource is now assigned to the role.

You can show the resources associated with a role.

To show the resources associated with a role:

Choose Show Plants on the Base Manufacturer role.

You can also choose Show Roles on a manufacturing resource.
