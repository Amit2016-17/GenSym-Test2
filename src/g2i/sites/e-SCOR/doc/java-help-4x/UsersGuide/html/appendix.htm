Asset Metrics Formulas

This appendix describes the specific formulas e-SCOR uses to compute the following asset metrics for roles:

Days of Supply
- Raw Materials
- Work in Progress
- Finished Goods
- Inventory
Asset Turns
Cash-to-Cash Cycle Time
Days Sales Outstanding
Note: Asset metric values can appear different in dialogs and in Excel reports due to differences in the way in which e-SCOR and Excel interpret binary numbers. For example, a dialog might display a value of 1.9999 when a report displays the same value as 2.0.
