Manufacturer Role

The Manufacturer role includes these specific Level 2 SCOR categories, and one or more Level 2 SCOR categories from these management processes:

ES: Enable Source - Determines how the role chooses its suppliers and manages contracts with suppliers.
Source - Sources stock, make-to-order, and/or engineer-to-order components from upstream suppliers.
P2: Plan Source - Determines the frequency with which the role sources components from upstream suppliers and computes how much to order.
Make - Manufactures stock, make-to-order, and/or engineer-to-order finished products.
P3: Plan Make - Determines the frequency with which the role manufactures finished products and computes how much to make.
Deliver - Delivers stock, make-to-order, and/or engineer-to-order finished products from components, when available.
ED: Enable Delivery - Manages contracts with buyers.
