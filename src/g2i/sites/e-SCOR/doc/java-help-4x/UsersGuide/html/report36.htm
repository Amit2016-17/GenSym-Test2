Creating Demand Input Reports

By default, you determine initial order demand for the overall supply chain by configuring parameters on the Order tab for each Sc: Source Consumer category and for the source products of each Consumer category.

You might have actual order demand data for your supply chain, which you would like to use in your model. To do this, you configure initial order demand through a report by:

Creating a Demand Input Report with the initial order demand data.
Enabling the import of order demand and specify the report to use.

When you run the simulation, the model creates orders for as many products as you configured in the report, then stops.

For detailed information on creating a Demand Input Report, see Configuring Initial Order Demand through a Report.
