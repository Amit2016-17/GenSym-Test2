Source Category Metrics

Group/Parameter
Tab
Description

General

Role Label

Source

The label associated with the role.

See Configuring General Parameters for Roles.

General

Sequence Number

Source

The Sequence Number of the role.

See Configuring General Parameters for Roles.

Metrics

Product Shipments
Received

Source

The total number of product shipments the Source category has received from suppliers since the start of the simulation.

Metrics

Product Shipments Lead Time Delta
(Days)

Source

The average change in time from the Desired Turnaround parameter of source products to the actual time at which the Source category receives product shipments. A positive number indicates product shipments are early, and a negative number indicates product shipments are late.

For a description of the Desired Turnaround parameter, see Source Tab for a source product.

General

Financial Period

Financial

The time period for computing financial metrics, obtained from the role.

General

Financial Payment
Terms

Financial

The time period for making payments to suppliers from invoices, obtained from the role.

Outgoing

Financial Obligations

Financial

Buyer obligations for customer orders that the Source category has placed with suppliers but has not yet received since the start of the simulation. This metric includes orders for source products assigned to the Source category, using the Product Selection parameters.

Outgoing

Financial Payments
Period

Financial

The total amount of money that the Source category has sent to suppliers for product shipments received during the previous collection period. This metric includes product shipments for source products assigned to the Source category, using the Product Selection parameter.

Outgoing

Financial Payments
Total

Financial

The total amount of money that the Source category has sent to suppliers for product shipments received since the start of the simulation. This metric includes product shipments for source products assigned to the Source category, using the Product Selection parameter.

Costs Incurred

Product Acquisition

Cost

The total cost that the Source category has incurred for receiving source products since the start of the simulation. This metric includes costs incurred for receiving source products assigned to the Source category, using the Product Selection parameter.

Costs Incurred

Create Customer
Order

Cost

The total cost that the Source category has incurred for creating customer orders since the start of the simulation. This metric includes costs incurred for creating customer orders for source products assigned to the Source category, using the Product Selection parameter.

Costs Incurred

Supply Chain Finance

Cost

The total cost that the Source category has incurred for paying invoices for product shipments received since the start of the simulation. This metric includes costs incurred for paying invoices for product shipments of source products assigned to the Source category, using the Product Selection parameter.
