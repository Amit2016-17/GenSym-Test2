Configuring Order Parameters for Product Composites

In pull mode, you configure the following parameters related to orders for product composites:

Order Type of source and delivery products.
Desired Turnaround of source products.
Order Fulfillment Preference of source products.

To configure order parameters for product composites:

1 Show the detail of the Products pool for each role in the model.
2 Display the properties dialog for each source and delivery product in the pool, click the Product tab, and configure the Order Type parameter to determine the planning strategy the role uses for sourcing, making, and delivering products.
3 Display the properties dialog for each source product in the pool, click the Source tab, and configure the Desired Turnaround and Order Fulfillment Preference parameters.
