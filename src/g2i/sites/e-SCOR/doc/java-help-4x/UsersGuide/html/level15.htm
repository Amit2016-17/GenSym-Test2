Creating an Organizer

When you create a model, you might want to use an Organizer tool to store miscellaneous objects such as the Scenario Manager or Database Interface object. You use an organizer for any object that the model requires to run but that is not directly connected to the model.

To create an organizer:

1 Display the SCOR toolbar and click the Tools tab:

2 Select an Organizer tool, then click anywhere on the workspace to place it.
3 Display the properties dialog for the organizer.
4 Edit the Label parameter to be any text, then drag the label to the desired location next to the organizer.
5 Show the detail of the organizer.

The organizer detail appears in its own window. You place objects on this detail, as needed.
