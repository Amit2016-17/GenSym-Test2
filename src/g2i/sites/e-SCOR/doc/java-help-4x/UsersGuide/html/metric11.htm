Viewing Metrics for the Make Categories

The properties dialogs for the Make categories have these differences for metrics, by tab:

Manufacturer tab: Only the M3 category includes the Order Entry to Mfg Time (Days) metric.
Cost tab: Only the M3 category includes the ECO metric, and the M1, M2, and M3 categories include the Production Material Handling cost metric.
Engineering tab: Only the M3 category includes this tab.

The Build Order, Build, and Engineering tabs have no computed metrics.
