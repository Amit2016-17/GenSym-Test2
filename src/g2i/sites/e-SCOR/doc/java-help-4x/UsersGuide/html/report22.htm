Creating an Environment Variable for the Server Location

By default, each time you open the default Excel report or a report that is based on the default report, Excel prompts you for the location of the e-SCOR server. To avoid this confirmation dialog, you can define an environment variable that points to the location of the server.

To create an environment variable for the e-SCOR server location:

1 Display the System Properties dialog and click the Environment tab.
Tip: To display the System Properties dialog, choose Properties on the My Computer icon on the desktop or double-click the System icon in the Control Panel. To display the control panel, choose Start > Settings > Control Panel.
2 Create an environment variable named ESCOR_SERVER, whose value is the host and port of the computer on which the e-SCOR server is running.
The default location is localhost:1111. For example, you might be running the server on my-host:1112.
3 Click the Set button, then click Apply.

The next time you open the Excel spreadsheet, Excel automatically connects to the specified server.
