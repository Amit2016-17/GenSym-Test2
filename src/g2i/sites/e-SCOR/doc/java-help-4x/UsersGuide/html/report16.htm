Configuring the Attributes to Appear in a Report

Each type of report includes a default set of parameters and/or metrics, or "attributes," depending on the type of report. You can configure the attributes to appear in any report by using one or both of the following techniques:

Configure the list of visible attributes through the report object.
Add attributes to the report through the dialogs.

When configuring the list of visible attributes through the report object, you can include time-weighted metrics that the model computes for Level 2 categories and product composites.

You can also configure the attributes to appear in reports automatically by using a pre-defined template for the various types of reports. For details, see Configuring Report Templates Automatically.
