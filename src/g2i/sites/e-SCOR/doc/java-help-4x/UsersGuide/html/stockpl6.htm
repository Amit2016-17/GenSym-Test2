Formula for Computing Amount to Order

The amount to order is based on the following formula:

if ( Inventory Level + On Order - Awaiting Order Total) =<
     Replenishment Threshold
then Amount to Order = 0
else Amount to Order = Max [Minimum Reorder Quantity,
                           (Replenishment Threshold -
                            Inventory Level -
                            On Order +
                            Awaiting Order Total ) ]
