Creating Background Images for Workspaces

You can use a .gif or .xmb file as the background image for a workspace. To do this, you load the image and e-SCOR centers it on the background of the workspace. It also creates an image definition, which you typically place on the detail of an organizer.

To create a background image for a workspace:

1 Select the workspace whose background image you want to configure and choose Workspace > Load Background Image.
2 In the file dialog that appears, navigate to a .gif or .xmb image that you want to use as the background and click Select Image.
The image appears centered in the background of the workspace. You will also see an image definition object on the workspace:

Note: You can also create image definition objects from the Displays tab of the ReThink toolbar.
3 Choose Cut on the image definition and paste it on the detail of an organizer.
By default, the image definition is saved as a reference, which makes the KB smaller.
4 To save the image with the KB, display the properties dialog for the image definition and click the Save Image Data with KB? option on.

To delete the background image for a workspace:

Select the workspace whose image you want to delete and choose Workspace > Delete Background Image.
