Topic - Using Pull and Push Planning Modes

Distributor Role in Pull Mode

In pull mode, a Distributor role determines when to source its components and deliver its finished products, and how much to source and deliver, as follows:

Process Category
When
How Much

Source

Sources components, based on source planning cycle (P2: Plan Source).

The amount to order depends indirectly on orders or forecasts for finished products, which the Deliver category sends upstream as replenishment orders for components.

Deliver

Delivers finished products when it can, based on inventory levels for components.

The amount to deliver depends on customer or replenishment orders for finished products.
