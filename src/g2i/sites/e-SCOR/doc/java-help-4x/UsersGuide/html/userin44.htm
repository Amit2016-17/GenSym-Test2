Topic - Working with Models

Selecting, Moving, and Resizing Objects

To select one or more objects:

Click an object to select it, click and drag a rectangular area to select all the objects in the rectangle, or use the Shift key to add objects to an existing selection.

To select all objects on a workspace:

Choose the Edit > Select All or enter Ctrl + A.

To move an object:

Select an object and drag the selected object on the workspace.

To resize an object:

Select an object and drag the handles of the selected object.
