Topic - Creating a Supply-Chain Model

Creating a Model

To create a model, you create a named workspace, place a model on the workspace and label it, then place a scenario on the model detail.

When naming a workspace, you must use a symbol, which is a string of non-case-sensitive, alpha-numeric characters, without spaces. For example, my-workspace is a symbol. The name appears at the top of the workspace.

When labeling a model, you use a text value, which is a string of case-sensitive, alpha-numeric characters that can include spaces. For example, My Model is a text value. The label appears next to the model.

Most ReThink objects define labels, which you can move to the desired location next to the object.

To create a model:

1 Choose Workspace > New Workspace.
e-SCOR creates an empty workspace and places it in its own window. The default name for the workspace is UNNAMED-WORKSPACE-1. Each new workspace you create increments the number appended to the name.
2 Choose Workspace > Properties or choose Properties from the popup menu for the workspace, then enter a new name in the Name field.
The workspace name must be a symbol. If you enter a name with spaces and accept the dialog, the name does not appear.
3 Choose View > Toolbars > e-SCOR or click the Toggle e-SCOR Toolbar button: .
4 Click the Tools tab:

5 Select a Model tool, then click anywhere on the workspace to place the model on the workspace.
6 Choose Properties on the Model tool to display its properties dialog.
7 Edit the Label to be any text value, then drag the label to the desired location next to the model.
Tip: You can configure the Best Practice URL and Model Version for a Model tool, just as you can for a Scenario tool. For details, see Configuring the Scenario.
8 Choose Workspace > Shrink Wrap or click the equivalent toolbar button, located at the bottom of the window:
The workspace and window size adjust to fit the model:

9 Choose Show Detail on the Model tool.
The model detail appears in its own window. You create your e-SCOR model on this detail.
10 Select a Scenario tool from the toolbar, click to place it on the model detail, and adjust the borders of the detail.
