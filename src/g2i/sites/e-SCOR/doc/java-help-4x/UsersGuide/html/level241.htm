Configuring the Source Category

The properties dialogs for the Source categories have these parameter differences, by tab:

Source tab: The S1, S2, and S3 categories do not include the Order Demand Import parameters, and the S1, S2, and S3 categories include the Receive and Putaway Time parameter.
Order tab: The S1, S2, and S3 categories do not include the Source ID, Order Size, Start Time, Stop Time, or Order Frequency parameters and do not include the Order tab at all.
