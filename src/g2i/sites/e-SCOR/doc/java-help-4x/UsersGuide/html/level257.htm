Delivery Tab

Parameter

Description

Product

Delivery Product

"On" if the product is a delivery product of the role and "off" if it is not. The role ignores the parameters on this tab if Delivery Product is "off."

Generally, you do not need to configure this parameter; the model configures it for you, based on the product specification you assign to a role.

Delivery

Completion Preference

For a Distributor role, whether the role should source finished products as complete products (Complete) or assemble finished products from components into kits (Assembled).

For Distributor roles, this parameter is only relevant when the role deliver both finished products and kits. The default value for a Distributor role is Complete. For details, see Configuring a Distributor Role to Outsource Finished Products.

Completion Preference for a Distributor role should always be either Complete or Assembled, never Complete Only.

For a Manufacturer role, the value should always be Complete Only, the default, which indicates that the role should manufacture complete finished products from components.

Delivery

Net Selling Price

The net price suppliers charge customers for each delivery product. The default value is 0.

The model uses this price to compute financial metrics, including Financial Payments Total and Financial Collections Total for the role.

You can use this parameter as the award criteria for choosing suppliers when using contracts. For details, see Modeling a Process with Multiple Suppliers.

Delivery

Published Delivery Lead Time

The typical standard lead time that the role quotes to its customers for a delivery product. The default value is 3 days.

You can use this parameter to choose suppliers when using contracts. For details, see Modeling a Process with Multiple Suppliers.

Multipliers

Order Receipt to
Entry Time

A number that the role multiplies by the Order Receipt to Entry Time parameter of the Deliver category to determine the actual time it takes to enter an order for a delivery product. The default value is 1.0, which means it takes exactly the Order Receipt to Entry Time.

For details, see Configuring Multipliers for Timing Parameters.

Multipliers

Pick Material Time

A number that the role multiplies by the Pick Material Time parameter of the Deliver category to determine the actual time it takes to pick components from inventory for making a delivery product. The default value is 1.0, which means it takes exactly the Pick Material Time.

For details, see Configuring Multipliers for Timing Parameters.

Multipliers

Packing Time

A number that the role multiplies by the Packing Time parameter of the Deliver category to determine the actual time it takes to pack containers for shipment with a delivery product. The default value is 1.0, which means it takes exactly the Packing Time.

For details, see Configuring Multipliers for Timing Parameters.

Multipliers

Order Shipment
to Receipt Time

A number that the role multiplies by the Order Shipment to Receipt Time parameter of the Deliver category to determine the actual time it takes to ship a container of delivery products. The default value is 1.0, which means it takes exactly the Order Shipment to Receipt Time.

For details, see Configuring Multipliers for Timing Parameters.
