Make Process Categories

e-SCOR provides the following Level 2 SCOR process categories to make stock, make-to-order, and engineer-to-order products:

Category
Description

M1: Make-to-Stock

Makes stock products, based on inventory planning, and places them in inventory. The P3: Plan Make category determines how much to make.

M2: Make-to-Order

Makes make-to-order products, based on orders, and places them in inventory. Inventory can go to any downstream Deliver category.

M3: Engineer-to-Order

Makes engineer-to-order products, based on orders, and sends them directly to the downstream Deliver category; the Make category does not maintain an inventory. Products go to the buyer that placed the order.
