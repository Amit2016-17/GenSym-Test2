Configuring Parameters for Source Products

You configure parameters for source products on the following tabs of a product composite dialog:

Product tab
Inventory tab
Source tab
Planning tab

Each topic provides an example of the properties dialog tab page for the source product of a Consumer role and a description of each parameter.
