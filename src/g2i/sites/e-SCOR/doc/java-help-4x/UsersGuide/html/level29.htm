P2: Plan Source

You configure the planning period for ordering source products and the time at which the first plan goes into effect. You can also configure the category to create replenishment orders on demand.

P3: Plan Make

Base Manufacturer
Manufacturer

You configure the planning period for making finished products and the time at which the first plan goes into effect. You can also configure the role to create build orders on demand and to compensate for yield.
