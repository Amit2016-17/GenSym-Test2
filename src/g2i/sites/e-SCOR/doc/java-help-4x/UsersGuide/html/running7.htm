Topic - Running the Simulation

Configuring the Scenario

You can configure various aspects of the scenario including the:

Simulation mode, which determines whether the simulation runs continuously, step-by-step, in real time, or online.
Best practice URL.
Duration of the simulation.
Version of the simulation.
Start time of the simulation.
Simulation speed.
Animation speed and whether or not to animate the model.
Speed at which objects flow along paths.
Indicator arrow behavior.
Computation behavior of the scenario.
