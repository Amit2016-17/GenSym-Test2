Creating Time-Series Role Metrics Reports

You can export to an Excel report time-series data for financial, Days of Supply, and asset metrics for each role, and Inventory Level metrics for each source and delivery product of each role.

You can compare these metrics for each role by creating a pivot table, then sort, analyze, and graph the data in Excel.

To export time-series data, you use these objects:

An e-SCOR Metrics Report, which obtains the time-series data.
An Update Trigger, which determines the frequency with which to obtain the time-series data.
An Excel Export tool, which exports the specified metrics to Excel.
Note: You can generate the same report data by simply using a Free Form Report and configuring the attributes to export. For details, see Creating Free Form Reports.

To configure the model to export time-series role metrics to Excel:

1 Display the e-SCOR palette and click the Tools tab.
2 Select an Update Trigger and place it on the model detail.
3 Click the Reports tab, then select an e-SCOR Metrics Report from the toolbar and place it above the Update Trigger.
4 Connect the e-SCOR Metrics Report to the Update Trigger.
5 Click the Export Tools tab, then select an Excel Export tool and place it above the e-SCOR Metrics Report.
6 Connect the Excel Export tool to the e-SCOR Metrics Report.
7 Configure the Label property for the e-SCOR Metrics Report and the Excel Export tool.
8 Configure the Duration of the Update Trigger to coincide with the financial period of the role.
9 From the popup menu for the Excel Export tool, choose the Choose Update Trigger menu choice, then choose the Select menu choice on the Update Trigger.
The model should look like this:

10 Now, run the simulation.
The Excel Export tool probes the model to obtain the time-series role metrics, based on the Duration of the Update Trigger. Excel creates four new spreadsheets called Financial Metrics, Days of Supply Metrics, Miscellaneous Metrics, and Product Inventory, which contain the time-series data.
11 At the end of the simulation, before you reset the model, choose Create Pivot Tables on the Excel Export tool.
