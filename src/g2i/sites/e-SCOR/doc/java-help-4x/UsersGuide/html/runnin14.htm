e-SCOR User's Guide
Topic - Running the Simulation

Configuring Animation

By default, e-SCOR animates all running models by physically moving objects along paths and animating icons.

If you are using your e-SCOR model as a communication tool to visualize work objects as they flow along paths, you might need to adjust the animation speed, depending on the speed of your computer. In particular, if you have a relatively fast computer, you might need to slow down the animation speed to better visualize the flow of work objects. However, keep in mind that performance degrades the slower the animation speed.

You configure animation speed, based on the number of milliseconds it takes to move an object from the beginning of a path to the end of a path. By default, an object moves along a path 30 milliseconds. To slow down the animation, you might want to move an object along the path in 50 or 100 milliseconds.

When you do not need to visualize the running simulation, you can disable animation altogether to enhance performance.

Note: Once you have configured your model, we recommend that you disable animation, disable updating of the Metrics toolbar, and set the Simulation Speed to 0 for optimal performance.

To adjust animation speed:

Display the properties dialog for the scenario, click the Options tab, and adjust the Animation Speed, where the larger the number, the slower the speed.

To disable animation for a simulation:

Display the properties dialog for the scenario, click the Options tab, and click the Enable Animation option off.
or
Choose Disable Animation on the scenario.

When you run the simulation, objects no longer move along paths and the icons no longer animate, although the simulation clock continues to advance.
