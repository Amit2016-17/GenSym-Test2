Configuring the Resource Map

You place individual resources on the detail of a resource map. To configure the resource map, you create individual manufacturing resources and configure these parameters for each:

Attribute
Description

Resource Label

The label to display with the resource, which must be unique.

Resource Capacity

The capacity of the individual resource, which must be a positive integer. You use this parameter to determine the number of batches of delivery products that a Base Manufacturer or Manufacturer role can make at one time.

Efficiency Factor

A number that the model multiplies by the duration of an activity to determine the amount of time the resource is active manufacturing finished products. The default value is 1.

You use this parameter to determine the relative amount of time it takes to make a batch of delivery products, which the model uses to compute resource utilization metrics.

Similar to assigning product specifications to roles, you have two options for assigning resources to roles:

Choose the resource to assign to the role on the role dialog.
Interactively choose the role to which a resource is assigned.

The results are the same, regardless of the technique you use.

To configure the resource map:

1 Display the e-SCOR toolbar and click the Resources tab.
This toolbar contains a Resource Map and a Manufacturing Resource, as follows:

2 Select a Resource Map and place it on your model detail.
3 Show the detail of the Resource Map.
4 Place individual Manufacturing Resources on the detail, depending on how you wish to configure the model.
5 For each resource, configure the parameters in the table above.
6 For each resource, use one of the following techniques to assign a resource to a role:
Display the properties dialog for the role, click the Resources tab, then choose the resource to assign to the role from the Resource dropdown list.
or
Choose the Choose Role menu choice, then choose the Select menu choice on the role to which to assign the resource.
