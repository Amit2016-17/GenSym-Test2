Topic - Configuring Role Details for Multiple Products

Example of Making and Delivering Multiple Finished Products

This figure shows the Products pool detail and the detail of a Manufacturer role that makes monitors and CPUs. The role detail uses two Make categories and two Deliver categories, each of which uses a stock planning strategy. The role uses separate Make categories for each delivery product, because each product uses a different value for the Batch Preference and Total Build Time parameters. The role uses separate Deliver categories, because each product uses a different value for the Sort By parameter to determine order selection.
