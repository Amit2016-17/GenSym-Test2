Showing Resource Representations for a Role

Each role has a pool that contains representations of the manufacturing resources assigned to the role. The labels for these resources correspond to the default role name.

To show resource representations for a role:

Choose Show Surrogate on a resource on the Resource Map detail.
or
Show the detail of the Manufacturers pool:

The detail contains a representation for each manufacturing resource associated with the role. For example, here is the detail for the Base Manufacturer pool, which contains a single resource representation:

To show the manufacturing resource associated with the representation:

Click the resource representation in the pool and choose Show Original.

A magenta arrow appears next to the corresponding resource on the map. Choose Hide Indicator to remove it.
