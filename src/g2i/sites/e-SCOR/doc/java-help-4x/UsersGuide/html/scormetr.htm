B

SCOR Metrics

The following table alphabetically lists each of the SCOR metrics that e-SCOR supports and provides the SCOR definition for the metric. If the e-SCOR parameter or metric name is different from the SCOR metric name, the e-SCOR name appears in parentheses below the SCOR metric name. If e-SCOR does not fully support the SCOR definition, the part of the definition that is not supported appears with a strike-through.

The table also lists the location in which the SCOR metric is defined in e-SCOR and whether it is an e-SCOR metric or parameter. Click the SCOR metric to go to the e-SCOR definition in the online help.

These definitions are reprinted by permission by the Supply-Chain Council (SCC) (www.supply-chain.org).
