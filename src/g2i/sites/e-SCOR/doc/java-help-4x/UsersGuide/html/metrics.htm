Viewing Metrics

While running the simulation, you can analyze the performance of your supply chain and verify the model by viewing metrics for these objects:

Roles
Categories
Source and delivery products
Resources
Routers

You can view metrics through:

Properties dialogs
The Metrics toolbar
Reports

When viewing metrics through dialogs, you must update metrics manually. You can configure reports to update at regular time intervals. The Metrics toolbar updates automatically.

This topic describes how to view metrics through properties dialogs.
