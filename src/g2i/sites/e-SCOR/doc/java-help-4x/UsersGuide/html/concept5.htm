Topic - e-SCOR Concepts and Terminology

SCOR Metrics

e-SCOR supports numerous SCOR metrics. The model computes these metrics and makes them available through dialogs and reports in the appropriate location in the model.

Some SCOR metrics are what e-SCOR considers parameters, which are values that the user enters into the model as variables.

Other SCOR metrics are what e-SCOR considers metrics, which are values that e-SCOR computes during the simulation.

e-SCOR refers to all inputs to the model as parameters and all outputs from the model as metrics, regardless of whether they are SCOR metrics.

Note: All SCOR metrics appear, both in this online help and in dialogs in the model, in bold typeface to identify them as elements of SCOR.
