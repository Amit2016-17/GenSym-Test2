Editing Workspace Colors and Margins

You can edit the background and foreground colors of any workspace. You can also edit the margins around the items at the edges of the workspace. By default, the background color is white, the foreground color is black, and the margins are 15 pixels.

You can edit the workspace color for a single workspace or for all workspaces in the hierarchy. For example, you might want to use the same workspace color for the model detail and all Task block details in the model.

To edit individual workspace colors and margins:

1 Choose Workspace > Properties or choose Properties on the workspace.
2 Configure the Foreground Color and Background Color by choosing a color.
3 Configure the Margin by entering the number of pixels.

To edit the workspace color of all workspaces in the hierarchy:

1 Configure the color of a workspace, as described above.
2 Choose Format Workspace Hierarchy on the workspace to propagate the workspace color to all workspaces below the current workspace in the hierarchy.
