Configuring the Best Practice URL

You might have a document that explains the overall model, which you can associated with the scenario. The document can be an RTF or HTML file, which displays in its own window when you click the scenario.

Note: Currently, e-SCOR supports the display of HTML 3.2 and basic RTF formats only.

To configure the best practice URL for the scenario:

Display the properties dialog for the scenario, click the Scenario tab, and configure the Best Practice URL to be one of the following:
