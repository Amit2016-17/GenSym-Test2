Applying Input Report Data to the Model

Before you can apply input report data to the model, you must create a report template.

To apply input report data, you enter data into the input report and apply the values to the model. Each time you apply new values, the parameters in the model update.

Configuring parameters through input reports provides an alternative to configuring the same parameters through properties dialogs and has these advantages:

Cutting and pasting spreadsheet values.
Configuring parameters for the same types of items in a single location, for example, all source products or all delivery products.
Running different configurations of the model, using different sets of input parameters and comparing the results.

To apply input report data to the model:

1 Configure the report data for the specified parameters of the objects in the report template.
Double-click a cell to edit the existing value and press Enter to accept the new value.
Tip: You can cut and paste values by editing the cell and using Ctrl + C to copy and Ctrl + P to paste.
2 Apply the report data, using one of these techniques:
Click the Apply button at the top of the report.
or
Select the report object and choose Reports > Apply.

e-SCOR applies the values from the report to the appropriate parameters in the model.
