Configuring the Mode Type

When you configure the Mode Type, the dialog displays the parameters required to configure the mathematical distribution.

To configure the Mode Type:

1 Configure the Mode Type parameter for one of the timing or initial order demand parameter listed above by choosing a mathematical distribution from the dropdown list.
For information about configuring category parameters, in general, see Configuring Category Parameters.
Once you choose the Mode Type, e-SCOR displays the parameters that you must configure for that distribution.
2 Configure the parameters associated with the distribution you choose.
