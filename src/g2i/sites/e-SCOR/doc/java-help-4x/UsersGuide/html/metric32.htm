Viewing Metrics for the Source Categories

The properties dialogs for the Source categories are the same for metrics. The S1, S2, and S3 dialogs do not include the Order tab. The Sc category does include the Order tab, but that tab has no metrics. The Contract tab for the Sc category also has no metrics.
