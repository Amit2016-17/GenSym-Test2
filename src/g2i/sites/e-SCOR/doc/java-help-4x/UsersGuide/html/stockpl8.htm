Topic - Using Stock Planning Strategies

Example: Amount to Order is Zero

Here is an example where the Amount to Order = 0, because the current inventory can cover the number of products on order:

Inventory Level 120
Replenishment Threshold 100
On Order 10
Awaiting Order Total 10
Minimum Reorder Quantity 30

120 - 100 + 10 = 30 (Inventory Level - Replenishment Threshold +
                     On Order)

10 < 30 (Awaiting Order Total < 30) therefore:

Amount to Order = 0
