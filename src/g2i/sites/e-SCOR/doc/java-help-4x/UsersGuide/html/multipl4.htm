Configuring the Source Category of the Buyer to use Contracts

To configure the Source category of the buyer role to use contracts, you configure the Contract Specifications parameters on the Contract tab. These parameters determine whether the role uses contracts, as well as various contract timing parameters, such as the length of the contract and when the first contract goes into effect.

Note: To ensure that the buyer always has a contract in effect when choosing suppliers, configure the Initial Contract Delay parameter to be less than the order frequency with which the role places orders. The order frequency is determined by the Order Period parameters of the Sc: Source Consumer category of a Consumer role or the Planning Period parameter of the P2: Plan Source category of a Distributor or Manufacturer role.

To configure the Source category of the buyer to use contracts:

1 Display the properties dialog for the buyer role that you want to configure to choose its suppliers.
2 Configure the Use Contracts parameter to be Yes and configure the Initial Contract Delay, Contract Duration, and Contract Response Time parameters.
