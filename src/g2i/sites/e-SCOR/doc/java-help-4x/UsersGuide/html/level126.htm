Configuring General Parameters for Roles

You can configure the following general parameters to identify a role:

Parameter
Description

Role Label

The label to display with the role, which must be unique within the model.

Sequence Number

An integer used to sort the role in a report. The default value is 1. Typically, you assign numbers to roles sequentially from upstream to downstream roles, or vice versa.

Highlight Color

Sets a region of color on the role icon, which you can use to identify and organize roles in the model. For example, you can use the same Highlight Color for all roles of a particular type, which is especially useful in models with many roles. The default value is Transparent, which shows no color region.

Upgrade

When installing a new version of e-SCOR, whether to upgrade the role, using the default role template ("on") or whether to leave the role as it is ("off"). The default value is "on".

Caution: Whenever you add categories to the role detail, you should click the Upgrade option "off;" otherwise, when you install a new version of e-SCOR, the new version of e-SCOR will overwrite your changes to the role detail to use the default role template.

To configure general role parameters:

Display the properties dialog for a role, configure the parameters on the Roles tab as described in the table above, and click OK or Apply.
