Activating and Deactivating the Scenario

Before you can run a simulation, you must activate the Scenario. You activate and deactivate the scenario by using menu choices in the Simulation menu or by clicking the equivalent toolbar button, as this table describes:

Menu Choice
Toolbar Button
Status
Description

Active
(toggle on)

stopped

Toggle the Active menu choice or toolbar button on to activate.

Active
(toggle off)

inactivate

Toggle the Active menu choice or toolbar button off to deactivate.

The following figure shows an active and inactive scenario:

Note: Starting the simulation automatically activates the scenario; therefore, typically, you do not need to activate the scenario explicitly.

Certain menu choices, such as Clear Indicators, are only available when the scenario is active.

To activate and deactivate a scenario:

Display a model that contains a scenario, then choose Simulation > Active or click the equivalent toolbar button to toggle activation.
