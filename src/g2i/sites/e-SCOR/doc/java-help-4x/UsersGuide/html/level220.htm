Configuring Logistical Parameters through a Report

By default, you configure timing and cost parameters related to logistics for each Source and Delivery category in the model.

Typically, logistical parameters vary, based on the type of carrier, such as mail, UPS, or Fed Ex. You can configure through a report various logistical parameters, based on the carrier. To do this, you:

Configure the Carrier parameter for one or more Deliver categories on the role detail.
Create a Logistics Lookup Report with the logistical parameter values to use.
Configure the Mode Type to be Report Lookup for each logistical timing parameter you want to lookup in the report.

When you run the simulation, the model looks up in the report the carrier, the supplier role, and the buyer role to determine the parameter value to use. It uses the logistical timing parameter value from the report, as well as the associated cost parameter.

To configure logistical parameters through a report:

1 For each Deliver category in the model whose logistical parameters you want to look up in a report, configure the Carrier parameter to be any text value.
For example, you might configure the Carrier parameter to be UPS for one Deliver category and truck for another.
2 Display the e-SCOR toolbar, click the Reports tab, and create a Logistics Lookup Report on the model detail.
3 Create a report template for the Logistics Lookup Report.
For details, see one of the following, depending on whether you want to create the report locally or in Excel:
Creating Reports.
Creating Reports in Excel.
4 In the report, configure the Supplier and Buyer to be the Role Label of the upstream supplier role and the downstream buyer role, respectively, then configure the Carrier.
For example, suppose you can configure the Carrier of the Deliver category of a Manufacturer role to be UPS or FedEx. Suppose the upstream supplier role is named Base Manufacturer and the downstream buyer role is named Consumer. You would configure the first three columns in the report to be:

5 In the report, configure as many logistical parameters as required for each carrier.
To use the default value for the parameter, leave cells empty.
For example, suppose the UPS carrier takes 3 days to receive components and costs $10 and the FedEx carrier takes 2 days and costs $15. You would configure these columns in the report:

You can configure as many logistical parameters for each carrier as you want.
6 Apply the data to the model.
For details, see one of the following, depending on whether you created the report locally or in Excel:
Applying Input Report Data to the Model.
Applying Input Report Data to the Model from Excel.
7 For each logistical parameter that you configure in the report, configure the Mode Type parameter to be Report Lookup in the dialog for the appropriate category.
In the previous example, you would configure the Receive and Putaway Time parameter of the Source category and the Order Shipment to Receipt Time parameter of the Deliver category to be Report Lookup.
8 Configure the Report Title of the logistical parameter to be the name of the Logistics Lookup Report that contains the logistical data.

e-SCOR displays all Logistics Lookup Reports that exist on the model detail in the Report Title dropdown list.
