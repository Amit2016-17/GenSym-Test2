Topic - Modeling a Process with Multiple Suppliers

Configuring the Buyer to Use Contracts

To use contracts, you:

Configure the Source category of the buyer to use contracts.
Configure the contract amount for the source product of the buyer role.

When you configure the buyer to use contracts, you must also configure the award criteria for choosing suppliers, as described in Configuring the Award Criteria for Choosing Suppliers.
