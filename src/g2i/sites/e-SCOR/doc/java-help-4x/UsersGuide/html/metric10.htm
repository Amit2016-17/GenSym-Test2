Viewing Metrics for Categories

Each category computes a number of metrics, which you can view as the simulation runs. This topic describes the metrics for each category for the simplest e-SCOR model, which consists of a Base Manufacturer role and a Consumer role. Each topic provides sample dialogs for the categories that appear in the simplest model. If the dialogs for other categories are significantly different, sample dialogs for these categories appear, as well. The descriptions of each category metric apply to all categories, unless otherwise indicated in the description. For example, the ECO cost metric is only relevant for the Mb and M3 Make categories.

The following topics show sample dialogs and describe metrics that appear on the following categories of these roles:

Base Manufacturer role:
- Make categories
- Plan categories
- Deliver categories
- ED: Enable Delivery category
Consumer role:
- ES: Enable Source category
- Source categories
