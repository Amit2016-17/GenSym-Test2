Writing Output Report Data to Excel CSV Files

You can create a .csv file for any output report.

Caution: Do not attempt to edit the CSV file while the simulation is running; otherwise, the model cannot write the data to the file.

To write output report data to an Excel CSV file:

1 Create an output report template in Excel.
For details, see Creating a Report Template in Excel.
2 Display the properties dialog for the output report object whose data you want to write to a CSV file and click the Excel tab.
3 Enable the Excel CSV File Reporting Enabled option.
4 Configure the Excel CSV File Name to be the name of the .csv file to create.
5 Configure the report to update at the desired intervals.
For details, see Updating Output Reports at Regular Time Intervals.
6 Configure the report to keep a history, if desired.
For details, see Keeping a History of Data Values.
7 Configure the attributes to appear in the CSV file.
For details, see Configuring the Attributes to Appear in a Report.
8 Configure the time unit, as needed.
For details, see Configuring the Time Unit.
9 Generate the output report data.

For details, see Generating Output Report Data from the Model to Excel.

When you run the simulation and the report updates, e-SCOR writes the report data to the CSV file.
