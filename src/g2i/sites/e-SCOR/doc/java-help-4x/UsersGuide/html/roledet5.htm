Adding Multiple Source Categories to Buyer Role Details

If each source product associated with a Distributor, Manufacturer, or Consumer role has different values for its Source category parameters, you can create multiple Source categories on the role detail for each source product.

Caution: Whenever you add categories to the role detail, you should click the Upgrade option for the role "off;" otherwise, when you install a new version of e-SCOR, the new version of e-SCOR will overwrite your changes to the role detail to use the default role template.

To add multiple Source categories to buyer role details:

1 Configure a buyer role to source multiple products, as follows, depending on the role:
For a Consumer role, assign multiple product specifications to the role.
For details, see Configuring the Products a Role Sources and Delivers.
For a Manufacturer role, create a product specification with two or more components and assign it to the role.
For details, see Configuring a Manufacturer Role.
For a Distributor role that delivers its source products, assign multiple product specifications to the role.
For details, see Configuring a Distributor to Deliver its Source Products.
For a Distributor role that assembles components into kits, create a product specification with two or more components and assign it to the role.
For details, see Configuring a Distributor to Assemble Components into Kits.
2 Display the role detail.
3 Display the e-SCOR toolbar and click the Source tab.
4 For each source product, select from the toolbar a Source category of the same type as the existing category and place it just above the existing category on the role detail.
For example:
To configure a Consumer role to source multiple products, select an Sc: Source Consumer category.
To configure a Distributor or Manufacturer role to source multiple stock products, select an S1: Source Stock Products category.
5 Connect the new category through the category routers on either side of the existing category and using a junction.
A Consumer role has only an upstream category router on its detail.
For details, see Creating and Connecting Roles.
6 For each category, configure the Product Selection parameters, as follows:
a Click the All Products option off.
b Configure the Specific Product Name parameter to be a unique source product by choosing a product from the dropdown list.
7 Configure the parameters of each category, as appropriate for the assigned source product.
