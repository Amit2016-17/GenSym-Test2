Source Tab

The following figure shows the Source tab of the properties dialog for the source product of a Consumer role:

Group/
Parameter

Description

Product

Source Product

"On" if the product is a source product of the role and "off" if it is not. The role ignores the parameters on this tab if Source Product is "off."

Generally, you do not need to configure this parameter; the model configures it for you, based on the product specification you assign to the role.

Orders

Desired Turnaround

The desired amount of time from when a buyer role places customer orders for its source products to when the buyer receives the shipment. The role uses this parameter to compute the Product Shipments Lead Time Delta (Days) of the Source category. The default value is 2 days.

For details, see Viewing Metrics for Categories.

For information on configuring this parameter through a report, see Configuring Initial Order Demand through a Report.

Orders

Order Fulfillment
Preference

Whether the role must receive complete shipments of the source products it orders (Whole) or whether it can receive partial shipments (Partial). The default value is Whole.

If the buyer requires whole shipments, the supplier waits to deliver shipments until it has enough delivery products for a complete order.

If the buyer accepts partial shipments, the supplier delivers what it can when it can and creates back orders for the unfulfilled portion of the order.

For information on configuring this parameter through a report, see Configuring Initial Order Demand through a Report.

Contract/Forecast

Estimated Contract
Amount

When contracts are in effect or when using a stock/forecast planning strategy, an estimate of the number of source products the buyer expects to order over the life of the contract. The default value is 0.

Note: You only need to configure this parameter when using contracts for stock/forecast planning. However, when using contracts to choose suppliers, the Estimated Contract Amount must be greater than zero.

For details, see Using a Stock/Forecast Planning Strategy and see Modeling a Process with Multiple Suppliers.

Multipliers

Receive and
Putaway Time

A number that the role multiplies by the Receive and Putaway Time parameter of the Source category to determine the actual time it takes to place source products in inventory. The default value is 1.0, which means it takes exactly the Receive and Putaway Time.

For details, see Configuring Multipliers for Timing Parameters.
