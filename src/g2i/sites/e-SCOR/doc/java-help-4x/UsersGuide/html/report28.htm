Connecting to and Disconnecting from the Server from Excel

When you first open the default report, e-SCOR automatically attempts to connect to the server. If you have created an environment variable, Excel automatically connects to the server; otherwise, you must configure the Host and Port of the server in the dialog that appears. Exiting Excel automatically disconnects from the server.

You can manually connect and disconnect to the server, as well, for example, if for some reason you need to shut down e-SCOR while Excel is still open.

To connect to and disconnect from the server from Excel:

In Excel, choose Server > Connect or Disconnect from the e-SCOR floating toolbar:

or
Enter Ctrl + Shift + C to connect to the server.
