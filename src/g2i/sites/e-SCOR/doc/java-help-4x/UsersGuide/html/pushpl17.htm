Topic - Using Pull and Push Planning Modes

Determining How Much is Pushed

The model computes the Pushed Amount metric, which is the number of products that an upstream supplier actually pushes onto a downstream buyer.

To determine how much gets pushed:

1 Show the Products pool detail of the buyer role that is downstream of a supplier role whose Push Amount you configured.
2 Display the properties dialog, click the Planning tab, and view the Pushed Amount metric of each source product.
