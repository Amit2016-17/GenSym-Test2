Topic - Working with Models

Using an Annotation Tool

You use an Annotation tool to create icons that display information about the model.

To annotate a model:

1 Select an Annotation tool from the Displays tab of the ReThink toolbar and place it next to the object you want to annotate.
2 Click the Annotation tool to display its detail.
3 Display the properties dialog for the free text that appears and edit the Text to provide a description of the block or process.
