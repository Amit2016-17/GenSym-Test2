Topic - Using Pull and Push Planning Modes

Using Pull Mode Planning

To use pull mode planning, you configure:

Initial order demand for customer orders that originate in the Consumer role.
Planning parameters for the Plan categories, P2: Plan Source or P2: Plan Make categories, depending on the role.
Planning parameters for product composites, depending on the planning strategy: stock/replenishment, stock/forecast, make-to-order, or engineer-to-order.
Order parameters for product composites.
Cost parameters related to orders for Source and Deliver categories.
