Modeling Changes in Order Demand Over Product Life Cycles

Suppose you want to model changes in order demand patterns over a product's life cycle. For example, in the first year, initial order demand might be relatively high with a high degree of variation, which you model by using a triangular distribution. After the first year, initial order demand might decrease around a more stable mean, which you model by using an exponential distribution. An exponential distribution generally has less variation, except very occasionally.

You could create a Simulation Manager that changes the mathematical distribution of the Order Size and Order Period parameters over time, as follows:

select-scenario(scenario-1)
set-order-size-parameter(consumer,Sc: Source Consumer,
   triangular,10,15,20,0)
set-order-period-parameter(consumer,Sc: Source Consumer,
   triangular,6 days,7 days,8 days,0)
set-order-size-parameter(consumer,Sc: Source Consumer,
   exponential,10,52 weeks)
set-order-period-parameter(consumer,Sc: Source Consumer,
   exponential,10 days,52 weeks)
start-simulation(104 weeks)
wait-simulation
set-file-reporting(level 2 output,true,
   Level 2 Output Report 1.csv)
save-report(role-output)
