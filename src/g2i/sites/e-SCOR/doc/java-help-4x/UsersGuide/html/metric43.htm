Group/Metric
Description

Push

Pushed Amount

For a supplier role, the number of delivery products the supplier actually delivers to downstream buyers when the role's Delivery Type is Push.

For a buyer role, the number of source products the buyer actually receives from upstream suppliers when the supplier's Delivery Type is Push.

Metrics

Amount to Order

For delivery products, the number of build orders a role creates per make planning cycle. For source products, the number of orders a role creates per source planning cycle. The model computes this number by comparing the Minimum Reorder Quantity parameter with the Replenishment Threshold parameter, given current orders.

For a description of these parameters for a source product, see Planning Tab.

For a description of these parameters for a delivery product, see Planning Tab.

Metrics

Awaiting Order Total

For the delivery products of a supplier role, the number of products that a downstream buyer has ordered and the supplier has entered, but the supplier has not yet delivered. The Order/Product process increments the Awaiting Order Total metric when the supplier receives an order for delivery products, and it decrements this metric when the supplier sends the product shipment.

Metrics

On Order

For the source products of a buyer role, the number of products that the buyer has ordered but the upstream supplier has not yet delivered. The Order/Product process increments the On Order metric when the buyer creates an order for source products, and it decrements this metric when the buyer receives the product shipment.

For delivery products of a supplier role, the number of products for which the Source or Make Planning process has created replenishment or build orders, respectively, but which the Deliver category of the supplier has not yet received. The Source and Make Planning processes increment this metric when they create replenishment and build orders, respectively.
