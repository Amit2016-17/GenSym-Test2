Updating Metrics

When you open a dialog, e-SCOR updates the metrics so they are current. For performance reasons, e-SCOR does not continue to update metrics as the simulation runs. Once the dialog is open, you must explicitly update the metrics.

Also, note that e-SCOR does not compute all metrics continuously. In particular, e-SCOR only computes financial, asset, and certain cost metrics once per financial period.

To update metrics in dialogs:

Click the Update button in the dialog or choose the Update menu choice on the role.
