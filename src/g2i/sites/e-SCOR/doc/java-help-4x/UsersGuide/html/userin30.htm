Going to an HTML or RTF File on the File System

You can go to an HTML or RTF page on the file system by using the file:\\\ protocol, which has this format:

file:\\\<drive>:\<directory>\<filename>.html
file:\\\<drive>:\<directory>\<filename>.rtf

For example, to go to the e-SCOR-readme.html file in the e-SCOR directory, you would use:

file:\\\C:\Program Files\Gensym\escor_rethink\e-SCOR-readme.html
Note: While it is also possible to go to any URL on the World Wide Web, such as http://www.gensym.com, this capability is not fully supported in the current release.

To go to an HTML or RTF page on the file system, you use the Web toolbar.

To go to an HTML or RTF file on the file system:

Using the file:/// protocol, enter the name of an HTML or RTF file on the file system in the Location text field.
or
Choose an existing location by clicking the arrow to the right of the Location text field.
