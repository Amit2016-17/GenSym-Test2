Configuring the Contract Amount for the Source Product

In addition to configuring the buyer role's Source category to use contracts, you must configure the Estimated Contract Amount of the buyer's source product.

The following figure shows the Source tab of the properties dialog for a source product of a Distributor role that uses contracts:

To configure the contract amount for the source product:

1 Show the detail of the Products pool for the buyer role.
2 Display the properties dialog for the source product that uses contracts.
3 Configure the Estimated Contract Amount parameter.
