Using Free Text

You can label any part of your model by using free text or free text with a border.

To label a model, using free text:

1 Select a Free Text or Borderless Free Text from the Displays tab of the ReThink toolbar and place it anywhere on a workspace.
2 Double-click the ellipses (...), edit the text, and press Ctrl + Return, or choose Properties on the text and edit the Text.

You can change the color of the text or background of the free text.

To change the color of the free text:

1 Choose Color on the free text.
2 Choose the region whose color you want to change, then choose a color from the color palette and click Apply.
You can change the background color of the text or the color of the text itself.
3 Choose a color from the color palette and click Apply.
