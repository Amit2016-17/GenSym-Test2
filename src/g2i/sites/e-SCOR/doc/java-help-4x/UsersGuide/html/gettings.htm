Getting Started

e-SCOR provides a basic model that you can use as a starting point for creating a model. To understand how to use e-SCOR, you can:

View the e-SCOR Window.
View the basic model.
Configure the basic model.
Run the simulation for the basic model.
Configure Level 2 SCOR for the basic model.
View e-SCOR demonstration models.

In addition to working with the basic model and demonstration models, you can view a:

Summary of creating models and running simulations, which provides references to all the chapters in this online help.
Summary of common tasks, which this online help assumes you know how to do.
