Viewing Metrics for Resources

Each resource computes a number of metrics, which you can view for individual manufacturing resources as the model runs.

The following figure shows a sample properties dialog for a manufacturing resource of a Base Manufacturer role:

Metric
Description

Capacity Utilization

Whether the resource is currently allocated. A value of 0 means the resource is not currently allocated, and positive number indicates that the resource is currently allocated.

Average Capacity Utilization

The amount of time the resource is allocated compared to the amount of time it is available, which is the Total Work Time divided by the Total Elapsed Time of the resource. This metric provides a measure of how intensively a role is using the resource to manufacture finished products.

Total Work Time

The total amount of time that the resource has been allocated since the start of the simulation.

Total Elapsed Time

The total amount of time that has elapsed since the start of the simulation.

Total Idle Time

The total amount of simulation time that the resource has not been allocated since the start of the simulation.

You can also view resource animation as a rough measure of resource efficiency. When the model is running and animation is on, the resource highlights when it is allocated to indicate that it is in use, as this figure shows:

You can use this technique to gauge resource efficiency, as follows:

If the resource representation is red...
Then...

Much of the time

The role is using the resource efficiently.

Not much of the time

The role is using the resource inefficiently.

Most of the time

The role is probably overusing the resource and you need to scale back on the amount of product it is making.
