Topic - A Asset Metrics Formulas

Days Sales Outstanding

The value of delivery product sales, which includes Financial Collections, Financial Bookings, and Financial Outstanding metrics for the current financial period, measured in days.

Days Sales Outstanding is relevant for a Base Manufacturer, Distributor, and Manufacturer role. It is not relevant for a Consumer role.
