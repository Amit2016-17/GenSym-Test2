Note: In addition to the default set of attributes, all input report include the UUID attribute, which uniquely identifies the object. When you create an input report in Excel, this attribute must appear; otherwise, you cannot apply values to the model. When you create reports locally, this attribute is only required if the object labels are not unique.

To configure the list of visible attributes in a report:

1 Display the properties dialog for the report object and click the Attributes tab.
The dialog shows the group name in the left column and the visible attributes associated with each group in the right column.
For example, here is the Attributes tab of the properties dialog for a Level 2 Output Report:

2 Add and remove attributes to and from the list, as needed:
Click Insert Row to add a row above the currently selected row.
Click Add Row to add a row at the end of the list.
Click Delete Row to delete the selected row or rows.
Tip: You can use the Shift key to select multiple rows.
3 Configure the attribute group and associated attribute name, as follows:
a Double-click the group name in the left column to display a dropdown list of available groups, then choose a group.
b Double-click the attribute name in the right column to display a dropdown list of available attributes in the selected group, then choose an attribute.

When you create the report template, the report contains the attributes you configured.
