e-SCOR User's Guide
Topic - Working with Models

Interacting with Workspaces

e-SCOR displays each workspace in the model in its own window, which you can open, close, iconify, and expand. You interact with workspaces in these ways:

Choose a workspace to display.
Create a new workspace.
Hide a workspace.
Delete a workspace.
Edit workspace colors and margins.
Scale workspaces.
Shrink wrap workspaces to fit the enclosed elements.
Show the superior item for the workspace.
Create background images for workspaces.
Print workspaces.
Export workspaces as JPEG files.
