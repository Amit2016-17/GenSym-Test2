Finding Items in the Model

You can find all the items in the model that meet some criteria, then go directly to a found item or display its properties dialog.

To find items that meet certain criteria:

1 Choose Tools > Find, enter Ctrl + F, or click the equivalent toolbar button: .
2 Configure the dialog to specify the criteria to use and click the Find Now button.

Enter a named item in the Named field to go to the item. Click the Browse button to display a browser for all classes defined in the model. Expand the tree view to see all the class names, then choose a class name. The other options in this dialog are advanced features, which you use when customizing Level 2 categories.

The found items appear at the bottom of the dialog. For example, this Find Items dialog has found all items of type bpr-model, which is the internal class name for a model:

To interact with found items:

To display a found item, choose Go To on the item in the dialog.
or
To display the properties dialog for the found item, choose Properties on the item.
