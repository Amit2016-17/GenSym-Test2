Configuring the Amount to Push Given Multiple Buyers

If an upstream supplier is pushing a single delivery product to multiple downstream buyers, you must configure the Push Amount parameter of the source product for each buyer role. In this case, the Push Amount is a number that represents the ratio of source products the downstream buyer should accept from the upstream supplier, given multiple buyers. The supplier delivers part of the Push Amount to each buyer.

If a downstream buyer does not want to accept any products from the supplier, use the default value for the Push Amount of the source product, which is 0.

In the following figure, a Distributor role pushes 12 delivery products to two Manufacturer roles. The Push Amount of each buyer's source product is 1 and 2, respectively, which means one buyer gets 1/3 of the Push Amount and the other buyer gets 2/3 of this amount.

To configure the amount to push given multiple buyers:

1 Show the Products pool detail of each buyer that is downstream of the supplier whose Push Amount you configured.
2 Display the properties dialog, click the Planning tab, and configure the Push Amount parameter of the source products being pushed to be a number that represents a ratio.
