Configuring the Award Criteria for Choosing Suppliers

You configure the award criteria for choosing suppliers when sourcing identical products from multiple upstream suppliers, using contracts. To configure the award criteria, you:

Configure the Source category of the buyer to choose suppliers, as follows:
- The criteria the role uses for choosing its suppliers.
- The number of suppliers to which to issue awards.
- Whether the buyer splits the order among the specified number of suppliers.
- When sending the order to a single supplier, the probability that the order will go to each qualified supplier; when splitting the order, the proportion of the order that goes to each qualified supplier.
Configure the supplier preference for the delivery product to rank suppliers, as needed.

In general, we recommend that you configure the Source category to split the order among qualified suppliers. Otherwise, the model chooses a single supplier, which can result in large orders going to secondary suppliers and small orders going to primary suppliers.

When you configure the award criteria for choosing suppliers, you must also configure the buyer to use contracts, as described in Configuring the Buyer to Use Contracts.

When contracts are in effect, the buyer uses the Supplier Information parameters to determine how to split the order among multiple suppliers or how to choose a single supplier. For details, see Examples of Configuring the Source Category for Choosing Suppliers.
