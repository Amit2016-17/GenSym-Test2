Transient Objects that Flow through the Model

In addition to the objects described in Transient Objects in Pools, a number of other objects also flow through the model when the simulation runs; however, the model does not save these objects in pools. This table describes the additional objects that flow through the model:

Object
Description

Invoice
Payment

Suppliers send invoices to buyers for products ordered. Buyers send payments to suppliers for product shipments received.

Product Ordered
Product Container

Buyers send products ordered to suppliers. Suppliers send product containers to buyers, when they are available.

Product Chit
Order Chit

Manufacturer roles generate product chits as part of the Build process when finished products have been manufactured and are available for delivery. Buyer roles generate order chits as part of the Source Planning process when source products arrive from suppliers and are available for manufacture or delivery, depending on the role.
