Configuring Report Templates Automatically

A number of reports provide menu choices for automatically configuring standard report templates. These menu choices configure the attributes to appear in the report, the report filters, and the report title, depending on the type of report. The following menu choices are available:

To configure report templates automatically:

Choose the appropriate menu choice on the report object.

The report title that appears next to the report updates to reflect the new report template.

To verify the visible attributes and report filters for the template:

Display the properties dialog for the report object and click the Attributes tab and/or the Filters tab.
Note: Only the Level 2 Input and Output Reports provide a Filters tab.
