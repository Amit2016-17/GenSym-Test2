Plan Categories

e-SCOR provides two of the four Level 2 SCOR process categories to model planning:

Category
Description

P2: Plan Source

Plans and prioritizes material resource supply and availability. Specifies the frequency with which roles send orders to upstream suppliers.

P3: Plan Make

Plans and prioritizes demand requirements and inventory. Specifies the frequency with which roles make products for delivery to downstream buyers.
