Going to a Location in the Server

You can go to a location in the server by using the g2:// protocol, which has this format:

g2://<host>:port>/<item-name-or-label>

where <item-name-or-label> is the internal name of the item in the server or it's label.

For example, to go to the workspace named my-models, you would use:

g2://localhost:1111/my-models

To go to a block labeled Generate Orders, you would use, where case does not matter:

g2://localhost:1111/generate orders

e-SCOR keeps a history of the locations that you have requested.

To go to a location in the server:

Using the g2:// protocol, enter an item name or label in the Location text field in the toolbar.
or
Choose an existing location by clicking the arrow to the right of the Location text field.
