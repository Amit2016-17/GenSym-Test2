Topic - Creating a Supply-Chain Model

Creating and Connecting Roles

To create your supply-chain model, first, you place roles on the model detail in a configuration that describes your supply chain, given the product hierarchy, then you connect the roles. Each role name must be unique.

Each role has two types of paths that you must connect: input paths and output paths. These paths extend in two directions: upstream and downstream. You can distinguish each type by dragging the stub attached to a role until you see a directional arrow, as this figure shows for each role:

When all the paths are connected, the supply chain forms a circular flow of input and output paths. When the simulation runs, the model create a variety of objects such as purchase requests, purchase responses, purchase awards, orders, invoices, product shipments, and payments.

When creating the supply-chain model, you can:

Create and connect individual roles.
Create and connect an individual role to multiple roles.
