Topic - Working with Models

Editing Icons

You can create custom icons for roles, categories, products, and resources by loading .gif or .xmb images.

To use a GIF or XMB image for an icon:

Select a class definition whose icon you want to edit and choose Edit > Load Image, navigate to a .gif or .xmb file to use as the icon, then click Select Image.

The class definition uses the specified image as its icon when you create an instance.

To show the image associated with the class definition:

Select the class definition and choose Edit > Show Image.

To delete the image associated with the class definition:

Select the class definition and choose Edit > Delete Image.
