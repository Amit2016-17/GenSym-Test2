Configuring the Deliver Category

The properties dialogs for the Deliver categories have these parameter differences, by tab:

Order tab: The D2 and D3 categories do not have the Order Selection parameter.
Order tab: The D3 category does not have the Order Receipt to Entry Time parameter; therefore, the D3 category does not have an Order tab at all.

The Delivery and Financial tabs have no configurable parameters.
