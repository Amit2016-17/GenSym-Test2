Role Parameter Keywords

These keywords set parameters for roles. The name of the keyword corresponds with the role parameter to set.

You specify the <role-label>, a <new-value> for the parameter, and the <time>.

For example, the following keyword sets the Financial Period of the Computer Manufacturer role to be 90 days at the start of the simulation:

SET-FINANCIAL-PERIOD-PARAMETER(computer manufacturer,90 days,0)

SET-FINANCIAL-PERIOD-PARAMETER(<role-label>,<new-value>,<time>)

SET-FINANCIAL-PAYMENT-TERMS-PARAMETER(<role-label>,
   <new-value>,<time>)

SET-LOGGING-ENABLED-PARAMETER(<role-label>,<new-value>,<time>)

SET-LOG-FILE-PARAMETER(<role-label>,<new-value>,<time>)

SET-STATISTICAL-METRICS-PERIOD-PARAMETER(<role-label>,
   <new-value>,<time>)
