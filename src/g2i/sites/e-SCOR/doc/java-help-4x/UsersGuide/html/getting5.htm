Running the Simulation for the Basic Model

When you run the simulation, e-SCOR computes key metrics and displays them in the Metrics toolbar. You can verify that the model is configured correctly and determine the performance of your model by viewing these metrics while the simulation runs.

To run the simulation for the basic model:

1 Click the Start All button on the Modeling toolbar:
Start All activates the Scenario, resets the simulation to time zero, and starts the simulation running.
If you have configured the model correctly, you should see objects begin to flow through the model. The simulation clock advances with each new discrete event. You can monitor the simulation time by viewing the Simulation toolbar clock at the bottom of the window, as well as the simulation clock next to the Scenario.
In addition to advancing the simulation clock, e-SCOR automatically displays the Metrics toolbar and updates the metrics once every half second of real time.
2 Click the Pause button on the Simulation toolbar:
e-SCOR stops the simulation clock, stops computing metrics, and stops processing objects in the model.
3 Click the Orders and Shipments tab of the Metrics toolbar.
This tab shows these metrics:
Quantity Ordered and Quantity Received for the source products of all buyer roles.
Quantity to Deliver and Quantity Shipped for the delivery products of all supplier roles.
Order Fulfillment Lead Time for the delivery products of all supplier roles.
Received Average Fulfillment Time for the source products of all buyer roles.

4 Click the Continue button on the Simulation toolbar:
e-SCOR continues running the simulation and updating metrics.
5 Click the Reset button on the Simulation toolbar:
e-SCOR resets the simulation clock to it's start time, deletes all objects it has created, and resets all metrics back to their default values.
