Viewing the Basic Model

The default e-SCOR model, eSCOR.kb, provides a basic model, which you can use as a starting point for building your own model.

To view the basic e-SCOR model:

1 Click the plus sign next to the Basic Model in the Model Browser to expand the tree view.
The tree view shows the entire contents of the model. You can click the plus sign next to any item in the tree to further expand the tree view. This figure shows the result of expanding the tree to show the basic elements of the model:

2 Right-click the Basic Model icon in the Model Browser and choose Go To.
You can navigate to any item in the model from the tree view. You can also display the properties dialog for any item by choosing Properties on the item.
e-SCOR displays the following window, which contains a named, top-level workspace, and Model, which is a container in which you create a model:

3 Right-click the Basic Model to display its popup menu and choose Show Detail.
Shortcut: To show the detail of any object with detail, press the Ctrl key and click the left mouse button.
