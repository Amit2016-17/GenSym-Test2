Filtering Report Data by Object Class

By default, reports include data for all objects of the specified type. For example, a Role Output Report includes data for all categories of roles, a Level 2 Output Report includes data for Plan, Source, Make, and Deliver categories, and a Product Composite Summary Report includes data for all categories of product composites.

Depending on the number of objects of a particular type you have in your model, you might want filter the data that appears in a report. For example, you might want to create separate Level 2 Output Reports for the Plan, Source, Make, and Deliver categories. You can filter report data for a Level 2 Input or Output Report; you cannot filter data for a Role Input or Output Report or for a Product Composite Input or Output Report.

You can also filter report data automatically by using a pre-defined template for the various types of reports. For details, see Configuring Report Templates Automatically.

To filter Level 2 report data:

1 Create as many Level 2 Input or Output Report objects as needed, depending on the objects to which each report should apply.
2 Display the properties dialog of each report object and click the Filters tab.
By default, Select All Categories is enabled, which includes data for all Level 2 categories in the report.
3 Disable the Select All Categories option, then click one or more categories to which the report should apply.
