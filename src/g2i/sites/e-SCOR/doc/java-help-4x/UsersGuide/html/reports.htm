Using Reports

You determine the performance of your supply-chain model by viewing metrics, which are attributes that the model computes, based on parameters, which are attributes that you configure. You can perform the following reporting tasks:

View simulation metrics in a toolbar.
Create reports, including output reports for viewing metrics and input reports for configuring parameters.
Update output reports at regular time intervals.
Keep a history of data values.
Filter the data to appear in a report.
Configure the attributes to appear in a report.
Configure report templates automatically.
Create reports in Excel.
Write to and read from Excel CSV files.
Write to and read from databases.
Create specialized reports, as follows:
- N-Dimensional Reports
- Free Form Reports
- Demand Input Reports
- Logistics Lookup Reports
- Time-Series Role Metrics Reports
Log transactions that occur between roles
Note: When creating reports in Excel, be sure you have Excel installed on your computer before you attempt to create a report. See the Readme.htm file for the supported versions of Excel.
