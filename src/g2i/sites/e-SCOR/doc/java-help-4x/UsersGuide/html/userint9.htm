Opening a Specific Model at Startup

By default, e-SCOR opens the last model that was open when you shut down the server. You can create a shortcut to the e-SCOR server that opens a specific .kb file at startup. When you connect to the server, the model you opened is loaded.

To create a shortcut that opens a different model at startup:

1 Create a shortcut from the Server.bat file, which is located in the bin directory of your e-SCOR product directory, and place it on your desktop.
2 Display the properties for the shortcut by choosing Properties from its popup menu, then click the Shortcut tab.
3 Configure the Target by appending the name of the KB file you want to load at startup.
For example:
"C:\Program Files\Gensym\escor_rethink\bin\Server.bat"    my-model.kb
4 Exit e-SCOR, start the e-SCOR server by using the shortcut, then start the client.

The model you specify in the shortcut opens automatically.
