Level 1 Roles

e-SCOR provides abstractions of the Level 2 SCOR model by implementing Level 1 "roles." e-SCOR provides these high-level capabilities, using Level 1 roles:

Business-level analysis, from suppliers' suppliers to customers' customers.
Product/product family identification.
High-level resource identification.
Export of metrics to spreadsheet for further analysis.
Dynamic modeling to simulate behavior over time.
Multiple planning and supply chain management modes.
