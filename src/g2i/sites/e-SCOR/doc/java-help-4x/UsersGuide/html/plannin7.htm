Topic - Using Alternative Planning Strategies

Configuring the Role to Use Alternative Planning Strategies

You configure the detail of a Distributor or Manufacturer role to use alternative planning strategies by adding process categories to the role detail. You must add categories to the role detail; you cannot replace existing categories.

To determine which category the role uses for sourcing, making, and delivering its products, you configure the Order Type parameter of the source and delivery products for the role. The category routers send orders, product shipments, invoices, and payments to the appropriate categories, based on the Order Type.

Note: Adding categories to the role detail without configuring the Order Type parameter has no effect; you must configure the Order Type to correspond to the categories on the role detail whose planning strategy you want to use.

To add process categories to a role, you connect input and output paths of the new category to the category routers on either side of the existing category and using a junction.

By adding Level 2 categories to the role detail, you can switch between alternative planning strategies by editing the Order Type parameter of the role's source and/or delivery products.

By default, the category you add applies to all product composites with the same Order Type. You can also configure the role detail to use different categories for product composites with the same Order Type by configuring the Product Selection parameters for the category. For details, see Configuring Role Details for Multiple Products.

Caution: Whenever you add categories to the role detail, you should click the Upgrade option for the role "off;" otherwise, when you install a new version of e-SCOR, the new version will overwrite your changes to the role detail and use the default role template.

To configure the role detail to use alternative planning strategies:

1 Display the e-SCOR toolbar and click the desired category tab, depending on the category you want to add.
2 Select a category from the toolbar and place it on the role detail just above the process category of the same type.
3 Connect the upstream input path (blue) of the new category to the Category Router on the left side of the existing category.
4 Connect the downstream input path (magenta) of the new category to the Category Router on the right side of the existing category.
5 Connect the upstream (magenta) and downstream (blue) output paths of the new category to the corresponding paths, using a junction.
6 Configure the Order Type of the source and delivery products to be Stock, Made-to-order, or Engineered-to-order to determine the planning strategy the role uses for sourcing, making, and/or delivering products.
7 If the Order Type is Engineered-to-order, configure the additional Level 2 SCOR parameters for the M3: Engineer-to-Order category.
For details, see Using an Engineer-to-Order Planning Strategy.
8 Select the shaded box surrounding the categories and drag the top handle up to accommodate the new category, as needed.
