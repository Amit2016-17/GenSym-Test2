Viewing Simulation Metrics in a Toolbar

When you run a simulation, e-SCOR displays key metrics for roles and product composites in the Metrics toolbar. The Metrics toolbar provides a quick way of verifying model performance. The toolbar updates once every half second of clock time.

The Metrics toolbar includes these tab pages:

Product Inventories - Inventory metrics for all product composites in the model.
Inventory metrics - Time-weighted metrics for key inventory metrics.
Orders and Shipments - Order quantity metrics for all product composites in the model, plus key order and shipment performance metrics.
Orders and Shipments metrics - Time-weighted metrics for the number of orders received and product shipments sent.
Assets - Standard asset metrics for all roles in the model.
Financial - Incoming and outgoing financial metrics for all roles in the model.

The time-weighted metrics include averages and standard deviations, which the model computes, based on the Aggregate Metrics Period parameter of each role. For details, see Configuring the Time Period for Computing Time-Weighted Statistics.

For information on disabling Metrics toolbar updating, see Configuring the Computation Behavior.

To hide and show the Metrics toolbar:

Choose View > Toolbars > Metrics, press the F3 key, or click the equivalent toolbar button to toggle the toolbar on and off:

To view simulation metrics in a toolbar:

Run the simulation, display the Metrics toolbar, and click the various tab pages to view metrics for each type of object.

To sort data in the toolbar:

Click the column header.

To configure the position of columns in the toolbar:

Drag the column header left or right.

To configure the width of each column in the toolbar:

Drag the border between each column left or right.
