Using the Tree-View Browsers

e-SCOR provides the following tree-view browsers and hierarchical views, which you can use to navigate models:

Model Browser - Displays all the elements of an e-SCOR model, including models, roles, categories, and other objects in the model environment that you create and configure.
Workspace Hierarchy - Displays the hierarchy of named and unnamed workspaces in the model.
Class Hierarchy - Displays the hierarchy of class definitions for the objects that e-SCOR creates and deletes while the model is running. You use the Class Hierarchy when customizing the implementation of Level 2 categories.
Module Hierarchy - Displays the hierarchy of modules in the current model. You can open other top-level modules to replace the current model, or you can merge modules into the current model and save them with the current model.

To display a tree-view browser:

Choose Tools > Model Browser, Workspace Hierarchy, Class Hierarchy, or Module Hierarchy.
Tip: You can display the Model Browser by clicking the equivalent toolbar button: 

The Workspace, Class, and Module Hierarchies provide additional information and navigation aids.

To expand the tree view:

Click the plus sign next to the item in the tree view, double-click a collapsed item, or click the Expand All button: .

To collapse the tree view:

Click the minus sign next to the item in the tree view, double-click an expanded item, or click the Collapse All button: .

To display the hierarchy with a different top-level node in the tree:

Enter the name of the top-level node in the Root Workspace Name, Root Class Name, or Root Module Name field, press Enter, and expand the hierarchy.

For example, to view the e-SCOR-specific class definitions in the Class Definition Browser, you would enter bpr-object as the Class Name.

To refresh the tree view after making changes:

Click the Refresh button: .

To go to items from a browser:

Choose Go To from the item's popup menu.

To display the properties dialog for an item in a browser:

Choose Properties from the item's popup menu.
