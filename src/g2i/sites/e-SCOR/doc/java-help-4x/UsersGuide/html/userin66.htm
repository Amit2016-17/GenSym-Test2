Viewing Errors and Messages

If an error occurs when running a simulation, you can view the generated error. Depending on the kind of error, you view the error message in either the Logbook or Message Board. The Logbook is for errors that occur in the server, and the Message Board is for errors that occur in the client.

If you are customizing e-SCOR, you might set breaks to pause the simulation at various points. For details, see the ReThink User's Guide.

To view Logbook errors:

Choose View > Logbook.

To view Message Board errors:
