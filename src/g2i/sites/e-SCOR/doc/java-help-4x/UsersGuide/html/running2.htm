Controlling the Simulation

You control the simulation from the Simulation menu or from the Modeling toolbar. These menu choices and toolbar buttons affect the scenario, which is the control center for running the simulation. Every model must contain a scenario.

To control the simulation, you:

Activate and deactivate the scenario.
Start and stop the simulation.
