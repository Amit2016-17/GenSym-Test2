Topic - Using Reports

Creating Specialized Reports

You can create the following specialized reports:

N Dimensional Reports, which include individual parameters or metrics for any number of objects of any type.
Free Form Reports, which include individual metrics for any number of objects of any type, where you configure the location of the metric in the report explicitly.
Demand Input Reports, which include order demand parameters for configuring the Sc: Source Consumer category of a Consumer role, such as Order Size and Order Period, and certain attributes of a product composite.
Logistics Lookup Reports, which include logistics parameters for the Source and Deliver categories of a role, based on the Carrier parameter of the Deliver category.
Time-Series Role Metrics Reports, which include time-series data for the financial, cost, and asset metrics of a role, and the inventory levels for each source and delivery product for the role.
