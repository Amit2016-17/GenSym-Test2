Configuring Process Category Parameters

Category parameters apply to every product that the category sources, makes, or delivers. By default, process category parameters include:

Timing parameters related to sourcing, making, and delivering products, such as the time it takes to enter an order, make a finished product, and pick, pack, and deliver a product shipment.
Cost parameters, such as the cost of acquiring materials, handling components during manufacturing, and entering and fulfilling an order.

To configure process category parameters, you need to understand how to:

Configure category parameters, in general.
Configure the mathematical distribution the category uses to compute variation.
Configure initial order demand through a report.
Configure logistical parameters through a report, based on the type of carrier.

You configure Level 2 SCOR parameters for these categories:

Make categories
Plan categories
Deliver categories
Source categories
Note: The ES: Enable Source category and the ED: Enable Delivery category have no configurable parameters.
Caution: You use the Upgrade parameter for categories to control whether e-SCOR upgrades the category, using the default template, when installing a new version of e-SCOR, or whether to leave the category as it is. Whenever you customize the detail of category, which is an advanced feature, you should click the Upgrade option "off;" otherwise, the new version of e-SCOR will overwrite your changes to the category detail to use the default category template.
