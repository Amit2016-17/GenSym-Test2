Configuring Product Composites for Replenishment Planning

When the Plan Type is Replenishment, the default, a role computes the size of each replenishment order for source products and each build order for delivery products, based on these parameters, which you configure for both source and delivery products:

Parameter
Description

Replenishment
Threshold

The minimum number of products the role needs to maintain in inventory before reordering.

Minimum Reorder
Quantity

The minimum number of products for which the role creates orders, when the inventory dips below the Replenishment Threshold. If necessary, the role can create orders for more than this quantity.

Maximum Threshold

The maximum number of products the role tries to maintain in inventory. When planning how many products to order, the role plans to have no more than this maximum in inventory.

If the Inventory Level metric of the source or delivery product falls below the Replenishment Threshold parameter, the role determines the size of the replenishment or build order, respectively, based on the Minimum Reorder Quantity parameter and the number of current orders. During its source or make planning cycle, the role creates orders such that the Inventory Level does not exceed the Maximum Threshold. If the role requires additional inventory such that the Inventory Level would exceeds this maximum, the role adds the excess amount to the order to the next planning cycle.

You must configure the Maximum Threshold to be less than the Inventory Maximum.

A Consumer role determines order size and frequency, based on order demand parameters that you configure in the Sc: Source Consumer category on the role detail; it does not use source planning. Also, a Consumer role does not keep an inventory; therefore, you do not configure these parameters for the source products of a Consumer role.

To configure a role to use a stock/replenishment planning strategy:

1 Show the detail of each Base Manufacturer, Distributor, and Manufacturer role that you configured to use a stock/replenishment planning strategy.
For details, see Configuring the Stock Planning Strategy.
2 Show the detail of the Products pool for each role to show its source and delivery products.
3 Display the properties dialog for each source and delivery product, click the Planning tab and configure the parameters described in the table above.
