Configuring the Source Category of the Buyer to Choose Suppliers

To configure the Source category of the buyer to choose suppliers, you configure the Award Criteria and Supplier Information parameters on the Contract tab.

For information on configuring the Contract Specification parameters, see Configuring the Source Category of the Buyer to use Contracts.

To configure the Source category of the buyer to choose suppliers:

1 Display the properties dialog for the buyer role that sources identical products from multiple upstream suppliers.
2 Configure the Award Criteria, Number of Suppliers, Split Order, and Proportion parameters.
