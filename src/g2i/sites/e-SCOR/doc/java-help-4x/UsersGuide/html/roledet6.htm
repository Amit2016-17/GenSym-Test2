Topic - Configuring Role Details for Multiple Products

Example of Sourcing Multiple Consumer Products

This figure shows the Products pool detail and the detail of a Consumer role with two Sc: Source Consumer categories, where one sources monitors and the other sources CPUs. The role uses separate Sc: Source Consumer categories for each source product, because each has a different value for the Order Period, Order Size, and Product Acquisition cost parameters.
