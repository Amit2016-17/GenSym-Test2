Topic - Working with Models

Working with Models

You work with e-SCOR models through the menus and toolbars to perform these tasks:

Connect to the server.
Save and open models.
Work with modules.
Navigate e-SCOR models.
Interact with workspaces.
Interact with objects in the model.
Annotate models.
Edit icons.
Use the various toolbars.
Switch user modes.
Viewing errors and messages.
Controlling the server state.
