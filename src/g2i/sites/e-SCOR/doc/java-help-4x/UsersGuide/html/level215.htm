Configuring the Mathematical Distribution

By default, the timing parameters that you can configure for Source, Make, and Deliver categories use a triangular distribution for computing variation. This means you configure the Min, Mode, and Max parameters of a triangular distribution, and the category computes a random value, based on these parameters. Similarly, the parameters relating to initial customer order size and frequency, which you configure for the Sc: Source Consumer category, also use a triangular distribution.

You can configure the mathematical distribution the model uses to compute these parameters. To do this, you configure the Mode Type to be one of a number of mathematical distributions and you configure the specific parameters required for the particular distribution.
