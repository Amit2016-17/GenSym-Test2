Distributor Role

The Distributor role includes these specific Level 2 SCOR categories, and one or more Level 2 SCOR categories from these management processes:

ES: Enable Source - Determines how the role chooses its suppliers.
Source - Sources stock, make-to-order, and/or engineer-to-order components or finished products from upstream suppliers.
P2: Plan Source - Determines the frequency with which the role sources components or finished products from upstream suppliers, and computes how much to order.
Deliver - Delivers stock, made-to-order, and/or engineered-to-order components, kits from components, finished products, and/or components.
ED: Enable Delivery - Manages contracts with buyers.
