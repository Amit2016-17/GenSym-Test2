Viewing e-SCOR Demo Models

e-SCOR provides a number of demonstration models that you can view. These models include:

escor-demos.kb
escor-examples-eto.kb
escor-examples-misc.kb
escor-examples-mto.kb
escor-examples-pull.kb
escor-examples-push.kb

For a description of these models, see the readme.html file.

e-SCOR also provides a number of a tutorial models, which you can load from the Start menu. For details, see the e-SCOR Tutorials.

These models are all located in the kbs directory of your e-SCOR product directory.

Caution: Be sure to save the current model before you open a new model.

To view e-SCOR demo models:

Choose File > Open and click Browse to navigate to the kbs directory and choose the file to open.
