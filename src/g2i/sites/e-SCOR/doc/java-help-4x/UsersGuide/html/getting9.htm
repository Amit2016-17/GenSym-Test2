Summary of Common Tasks

This section summarizes how to perform common tasks in e-SCOR:

To...
Do this...

Display the popup menu for an object on a workspace

Click the right mouse button on the object.

Display the properties dialog for an object on a workspace

Double-click the object, select the object and press the F4 key, or choose Properties from the object's popup menu. You can also select the item, then choose Properties from the Edit menu or click the Properties toolbar button: 

Display the detail for an object, such as a model or a role

Choose Show Detail from the popup menu for the object or Ctrl + click the object.

Display the e-SCOR toolbar

Choose View > Toolbars > e-SCOR, enter Ctrl + E, or click the equivalent toolbar button:

Resize the toolbar

Drag the corners of the toolbar.

Adjust the size of a workspace and its associated window to fit the contents of the workspace

Choose Workspace > Shrink Wrap or click the equivalent toolbar button: Then, adjust the window size to fit the workspace.

Hide a workspace

Ctrl + click the workspace.

Go to the Products pool detail to view and configure product composites for a role

Choose Go To Products Detail from the popup menu for a role.

From this point on, this online help does not give explicit instructions on how to perform these tasks.
