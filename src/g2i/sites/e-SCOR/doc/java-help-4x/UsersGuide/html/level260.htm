Parameter

Description

General

Order Type

Determines whether the role uses a Stock, Made-to-order, or Engineered-to-order planning strategy when creating customer orders for its source products. The default is Stock.

Note: A Consumer role ignores this parameter because it generates orders, based on parameters you configure in the Sc: Source Consumer category, rather than based on source planning.

For details on each of these planning strategies, see Using Alternative Planning Strategies.

Preference

Product Preference

This parameter is only relevant for delivery products.

For details, see Product Tab.

Preference

Supplier Preference

This parameter is only relevant for delivery products.

For details, see Product Tab.

Preference

Customer Preference

A number representing the priority of the customer's source product, where the lower number means a higher priority. The default value is 1.

You can use this number to choose which delivery product to deliver first when configuring the Order Selection parameters of the D1 category.

For details, see Configuring Order Selection Parameters.

For information on configuring this parameter through a report, see Configuring Initial Order Demand through a Report.

Note: This parameter is only relevant for source products.
