Adding Multiple Make and Deliver Categories to Supplier Role Details

If each delivery product associated with a Base Manufacturer, Distributor, or Manufacturer role has different values for its Make and/or Deliver category parameters, you can create multiple Make and/or Deliver categories on the role detail for each delivery product.

Caution: Whenever you add categories to the role detail, you should click the Upgrade option for the role "off;" otherwise, when you install a new version of e-SCOR, the new version will overwrite your changes to the role detail to use the default role template.

To add multiple Make and/or Deliver categories to supplier role details:

1 Configure a Base Manufacturer, Distributor, or Manufacturer role role to deliver multiple products by assigning multiple product specifications to the role.
For details, see Configuring the Products a Role Sources and Delivers.
2 Display the role detail.
3 Display the e-SCOR toolbar and click the appropriate tab-Make or Deliver.
4 For each delivery product, select from the toolbar a Make and/or Deliver category of the same type as the existing category and place it just above the existing category on the role detail.
For example:
To configure a Base Manufacturer to make multiple products, select an Mb: Make Base Manufacturer category.
To configure a Distributor to deliver multiple products, select a D1: Deliver Stocked Product category.
To configure a Manufacturer to make and deliver multiple products, select an M1: Make-to-Stock category and a D1: Deliver Stocked Product category.
5 Connect the new category through the category routers on either side of the existing category of the same type and using a junction.
A Base Manufacturer role has only a downstream category router on its detail.
For details, see Creating and Connecting Roles.
6 For each category, configure the Product Selection parameters, as follows:
a Click the All Products option off.
b Configure the Specific Product Name parameter to be a unique delivery product by choosing a product from the dropdown list.
7 Configure the parameters of each category, as appropriate for the assigned delivery product.
