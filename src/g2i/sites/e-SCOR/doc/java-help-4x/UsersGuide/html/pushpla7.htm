Configuring Cost Parameters Related to Orders

In pull mode, you can configure the following cost parameters related to orders:

Order Entry of the Deliver category.
Create Customer Order of the Source category.

To configure cost parameters related to orders:

1 For each Base Manufacturer, Distributor, and Manufacturer role in the model, show the detail of the role, display the properties dialog of the Deliver category, click the Cost tab, and configure the Order Entry cost parameter.
2 For each Distributor, Manufacturer, and Consumer role in the model, show the detail of the role, display the properties dialog of the Source category, click the Cost tab, and configure the Create Customer Order cost parameter.
