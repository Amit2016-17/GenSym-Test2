Using Attribute Displays

By default, e-SCOR objects include attribute displays that show the Label of the item. For some objects, such as models and product composites, the attribute display is empty (""). For other objects, the attribute display has a default value, such as "Manufacturer". In addition, the product composites that e-SCOR creates when you assign a product specification to a role, include the Product Name and the Inventory Level metrics as attribute displays.

When you edit an object's Label through the properties dialog, the attribute display updates. You can drag the attribute display to a new location next to the object.

You might want to add attribute displays to the layout to show the value of other parameters and/or metrics next to the object. You can also display the name of the parameter or metric with its value.

To edit the layout of attribute displays:

1 Choose Edit Attribute Display Layout on any item with an attribute display.
e-SCOR displays a dialog such as the following for a product composite:

2 Choose the attribute to display from the unselected attributes list and click the right arrow button to move it to the selected attributes list.
3 To include the name of the attribute in the attribute display, click the check box in the Name column next to the attribute name.
The order in which the attributes appear in the dialog is the order in which they appear next to the object. However, you can create additional attribute display groups and drag each group independently.
4 To create an additional attribute display group, click the Insert Display button, which creates a new tab page on the dialog.
By default, each tab page is labeled Display with sequential numbers appended to the name. You can edit these names to make them easier to identify.
5 To edit the tab names, click the tab whose name you want to edit, click the Name Display button, and enter the tab name.
6 To remove an attribute display group and its associated tab, click the tab you want to remove, then click the Remove Display button.
