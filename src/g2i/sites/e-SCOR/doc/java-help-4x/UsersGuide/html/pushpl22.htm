Topic - Using Pull and Push Planning Modes

Combining Pull and Push Mode within a Role

In addition to combining pull and push across roles, as Configuring the Planning Mode for Roles describes, you can combine pull and push planning modes within the same role. You can use these combinations of pull and push for each role:

Distributor role that uses pull-push.
Distributor role that uses push-pull.
Manufacturer role that uses pull-push.
Manufacturer role that uses push-pull.
