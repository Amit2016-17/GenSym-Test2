Creating a New Workspace

You create models on a named workspace. The workspace name must be a symbol; use hyphens or underscores in place of spaces.

To create a new workspace:

1 Choose Workspace > New Workspace.
e-SCOR creates a new workspace, whose default name is unnamed-workspace-1. Each subsequent workspace you create appends a new number to the name.
2 Choose Workspace > Properties or choose Properties on the workspace and configure the Name to be a symbol.
