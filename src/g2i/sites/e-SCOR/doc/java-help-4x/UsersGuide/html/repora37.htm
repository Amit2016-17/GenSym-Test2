Creating Logistics Lookup Reports

By default, you configure logistical parameters related to timing and cost for each Source and Delivery category in the model. Typically, logistical parameters vary, based on the type of carrier, such as mail, UPS, or Fed Ex. You can configure through a report various logistical parameters, based on the carrier or the supplier-buyer route by:

Configuring the Carrier parameter for the Deliver category.
Creating a Logistics Lookup Report with the logistical parameter values.
Configuring the Mode Type for each logistical parameter you want to lookup in the report to be Report Lookup.

When you run the simulation, the model looks up in the report the carrier, the supplier role, and the buyer role to determine the parameter value to use. The model does not use a mathematical distribution to compute a random value.

For detailed information on creating a Demand Input Report, see Configuring Logistical Parameters through a Report.
