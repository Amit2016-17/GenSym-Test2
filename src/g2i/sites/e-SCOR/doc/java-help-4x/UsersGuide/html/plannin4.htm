Topic - Using Alternative Planning Strategies

Understanding the Required Process Categories for Each Role

The role detail can include one or more Source, Make, and/or Deliver categories for each planning strategy, depending on the role. For example, a Manufacturer role can include an S1, M1, and D1, and an S2, M2, and D2.

Certain categories must exist on the role detail and can appear only once. Other categories can appear multiple times. The following table summarizes the required process categories for each role and the number of categories of each type that can appear:

This role...
Must include these categories...

Base Manufacturer

Mb: Make Base Manufacturer (one or more)
P3: Plan Make (one only)
D1, D2, and/or D3 (one or more)
ED: Enable Delivery (one only)

Distributor

ES: Enable Source (one only)
S1, S2, and/or S3 (one or more)
P2: Plan Source (one only)
D1, D2, and/or D3 (one or more)
ED: Enable Delivery (one only)

Manufacturer

ES: Enable Source (one only)
S1, S2, and/or S3 (one or more)
P2: Plan Source (one only)
M1, M2, and/or M3 (one or more)
P3: Plan Make (one only)
D1, D2, and/or D3 (one or more)
ED: Enable Delivery (one only)

Consumer

ES: Enable Source (one only)
Sc: Source Consumer (one or more)
