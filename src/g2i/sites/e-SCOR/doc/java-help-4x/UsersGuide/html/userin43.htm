Topic - Working with Models

Interacting with Objects in the Model

When creating your model, you can interact with objects by using the Edit menu, the object's popup menu, and the Layout menu. Many of the menu choices have shortcuts and/or equivalent toolbar buttons. The toolbar buttons are in the Layout toolbar, which is located at the bottom of the window.

When you create a model, we recommend that first, you place the roles on the workspace, then you align and distribute them, using buttons on the Layout toolbar, then you connect them.

You configure attributes of objects through standard and automatically generated dialogs, using the Text Editor.

This topic describes how to:

Select, move, and resize objects.
Cut, copy, paste, and delete objects.
Control the layout of objects.
Display the properties dialog for objects.
Configure objects through dialogs.
Display details for objects.
