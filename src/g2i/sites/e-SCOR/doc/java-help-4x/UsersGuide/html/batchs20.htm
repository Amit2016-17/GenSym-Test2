Sort-Order Category Parameter Keywords

These keywords set values for Level 2 category parameters that determine sort order, for example, Sort Attribute 1, Sort Attribute 2, etc. The name of the keyword corresponds to the Level 2 category parameter to set.

You specify the <role-label> of the role on whose detail the category exists, the <level-2-label>, the <order-number> of the attribute to set, which is an integer between 1 and 4, the <new-value> for the parameter, and the <time>.

For example, the following keyword sets the Sort Attribute 1 parameter of the S1: Source Stocked Products category on the Computer Manufacturer role detail to Supplier Preference at the start of the simulation:

SET-AWARD-CRITERIA-SORT-ATTRIBUTE-PARAMETER
   computer manufacturer,s1: source stocked product,1,
   supplier preference,0)
