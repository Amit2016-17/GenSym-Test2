Creating N-Dimensional Reports

To facilitate data entry and analysis, you can create an N-Dimensional Input Report that includes the key parameters you need to configure and an N-Dimensional Output Report that includes the key metrics you want to analyze.

An N-Dimensional Input or Output Report includes a single attribute value for any number of objects of any type in the model. You might need to create such a report when the attributes you want to configure or output exist in individual objects of different types, rather than in objects of the same type.

For example, suppose you are creating a model that determines optimal replenishment levels by looking at the Supplier On-Time Performance (%) of a Consumer role and the Inventory Days of Supply of its upstream supplier. To determine optimal replenishment levels, you need to experiment with variation in the Order Size parameters of the Consumer role, the Replenishment Level of the Consumer role's source product, and the Order Shipment to Delivery Time of the upstream supplier's delivery product.

When configuring the attributes to appear in an N-Dimensional Input or Output Report, you must uniquely identify the object by its label or by its UUID. For this reason, it is often easier to add attributes to an N-Dimensional Input or Output report through the dialogs.

Note: If you are entering labels, the labels must be unique and cannot contain carriage returns.

For example, to include the Order Size parameters for two Sc: Source Consumer categories, you might configure the category labels as Sc: Source Consumer 1 and Sc: Source Consumer 2.

Note: Because the labels associated with product composites are not unique across roles, you must choose the attributes of product composites through the dialogs.

To create an N-Dimensional Input or N-Dimensional Output Report:

1 Display the e-SCOR toolbar and click the Reports tab:

2 Select an N-Dimensional Input or Output Report, and place it on the model detail.
3 Display the properties dialog for the report object and configure the parameters on the General tab.

4 Configure the attributes to appear in the report.

5 Create the report template, then generate the output report data or apply input report data to the model.
