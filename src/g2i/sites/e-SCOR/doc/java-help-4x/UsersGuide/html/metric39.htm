Topic - Viewing Metrics

Inventory Tab

The following figure shows the Inventory tab of a sample properties dialog for the delivery product of a Base Manufacturer role:

Group/Metric
Description

Product

Inventory Level

The actual number of source or delivery products in inventory.

This metric is not applicable for a Consumer role, because a Consumer role does not keep an inventory.

Product

Potential Inventory Level

The amount of inventory of source or delivery products that a role could potentially deliver from existing source products.

This metric is not applicable for a Consumer role, because a Consumer does not keep an inventory.

Product

Inventory Status

Whether the inventory for source or delivery products is below, at, or above its threshold, as follows:

below-threshold if the Inventory Level metric is less than the Replenishment Threshold parameter.
above-threshold if the Inventory Level metric is greater than the Maximum Threshold parameter.
within-threshold if the Inventory Level metric is less than or equal to the Replenishment Threshold parameter, and greater than or equal to the Maximum Threshold parameter.

For a description of Replenishment Threshold for a source product, see Planning Tab.

For a description of Replenishment Threshold for a delivery product, see Planning Tab.

This metric is not applicable for a Consumer role, because a Consumer role does not keep an inventory.
