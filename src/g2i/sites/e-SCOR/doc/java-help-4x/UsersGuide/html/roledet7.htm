Topic - Configuring Role Details for Multiple Products

Example of Sourcing Multiple Components

This figure shows the Products pool detail and the detail of a Distributor role that assembles computer kits from monitor and CPU components. The role detail uses two Source categories, each of which uses a stock planning strategy. The role uses separate Source categories for each component, because each has a different value for the Receive and Putaway Time and Product Acquisition cost parameters.
