Choosing a Workspace to Display

You can go to a named workspace by choosing the workspace to display.

To choose the workspace to display:

1 Choose Workspace > Get Workspace to display a list of all named workspaces in the model or click the equivalent toolbar button:
Each named workspace is associated with a KB module and file. For example, the workspace associated with the eSCOR.kb model is basic-model-example.
The Get Workspace dialog shows named workspaces only; it does not show details.
2 Choose the workspace to display and click OK.
