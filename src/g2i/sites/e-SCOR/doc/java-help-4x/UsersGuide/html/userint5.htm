Connecting to a Specific Server at Startup

You might be running the e-SCOR server and client on different computers, or you might be running multiple e-SCOR servers on the same computer. You can create a shortcut to the batch file that starts the e-SCOR client to connect to a specific server at startup. You specify the host and port of the server to which to connect as command-line arguments in the shortcut.

Alternatively, you can create an environment variable for the server to which to connect at startup, which the batch file automatically uses. e-SCOR uses this same environment variable when connecting to the server from Excel when creating reports.

To configure the batch file to connect to a specific server at startup:

1 Create a shortcut from the Client.bat file located in the bin directory of your e-SCOR product directory and place it on your desktop.
2 Display the properties dialog for the shortcut and click the Shortcut tab.
3 Configure the Target by appending the host and port of the e-SCOR server to which to connect.
For example, to connect to my-host at port 1112, enter:
"C:\Program Files\Gensym\escor_rethink\bin\Client.bat"    my-host:1112
4 Start the e-SCOR server from the Start menu, then start the client by double-clicking the shortcut.

To create an environment variable for the e-SCOR server location:

1 Display the System Properties dialog and click the Environment tab.
Tip: To display the System Properties dialog, choose Properties on the My Computer icon on the desktop or double-click the System icon in the Control Panel. To display the control panel, choose Start > Settings > Control Panel.
2 Create an environment variable named ESCOR_SERVER, whose value is the host and port of the computer on which the e-SCOR server is running.
The default location is localhost:1111. For example, you might be running the server on my-host:1112.
3 Click the Set button, then click Apply.
