Topic - Using Reports

Summary of Input and Output Reports

This table summarizes the input and output reports you can create. Click the report name for information on how to create the report.

Report
Description

Role Input Report

Role parameters, which determine the planning mode, the financial payment period and terms, and the time period for computing metrical metrics for each role in the model.

Role Output Report

Role metrics, which compute financial bookings, outstanding, collections, and payments, asset metrics, and total costs for each role in the model.

Level 2 Input Report

Parameters for each Plan, Source, Make, and Deliver category in the model.

Level 2 Output Report

Metrics for each Plan, Source, Make, and Deliver category in the model.

Resource Input Report

Resource parameters, which determine fixed and variable costs, and resource capacity for each resource in the model.

Resource Output
Report

Resource metrics, which compute the number of activities each resource performs during the simulation and the amount of time it spends performing those activities for each resource in the model.

Product Specification
Input Report

Parameters for each product specification in the model, which determine the product name and quantity.

Product Composite Input Report

Parameters for each product composite in the model, which correspond to the parameters on the Product, Inventory, Source, Manufacture, Deliver, and Planning tabs of the product composite dialog.

Product Composite Output Report

Metrics for each product composite in the model, which correspond to the metrics on the Product, Inventory, Source, Manufacture, Deliver, and Planning tabs of the product composite dialog.

N Dimensional
Input Report

Individual parameters for any number of objects of any type in the model.

N Dimensional
Output Report

Individual metrics for any number of objects of any type in the model.

Free Form
Summary Report

Individual parameters and metrics for any number of objects of any type in the model, where you configure explicitly the location of the metric in the report.

Demand
Input Report

Initial order demand parameters for the overall supply chain, such as Order Size and Order Period. You configure the Sc: Source Consumer category of the Consumer role to read order demand parameters from the report.

Logistics Lookup
Report

Logistics parameters for the Source and Deliver categories of a role, based on the Carrier parameter of the Deliver category. You configure logistical parameters of the Source and Deliver categories of a role to read these parameters from the report.

Time-series Role Metrics Report

Time-series data for the financial, cost, and asset metrics of a role, and the inventory levels for each source and delivery product for the role.
