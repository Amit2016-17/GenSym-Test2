Creating a Supply-Chain Model

The basic steps in creating an e-SCOR supply-chain model are:

1 Determine the level of detail to model. The level of detail you model depends on the performance data you wish to analyze.
2 Configure the model environment. The model environment is where you build and run the supply-chain model.
3 Configure the product hierarchy. The product hierarchy describes the products that your supply chain sources, makes, and delivers.
4 Configure the Level 1 roles. Roles are the high-level functions that describe your supply chain. To configure Level 1 roles, you perform these tasks:
a Create the supply-chain model. Select roles from the e-SCOR toolbar and place them on your model workspace, then connect the roles together to form the supply chain.
b Configure the products a role sources and delivers. Assign product specifications to roles to create source and delivery products, as appropriate, which determines the products that each role sources and delivers.
c Configure role parameters. Configure role parameters to determine such things as when the role computes its financial metrics and whether the role uses a pull or push planning mode to source, make, and deliver its products.
5 Configure resources. Resources determine the capacity of each role to manufacture finished products. To configure resources, you create a resource map with manufacturing resources, assign manufacturing resources to manufacturing roles, and configure resource parameters.
