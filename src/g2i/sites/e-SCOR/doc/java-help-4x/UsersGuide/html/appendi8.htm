Cash-to-Cash Cycle Time

The time it takes for money to flow from buyers to suppliers. A positive value indicates the role spends more money than it takes in. A negative value indicates the role takes in more money than it spends. This metric is weighted, based on the Financial Payment Terms of each buyer role and is measured in days.

If any one of the values that make up the Cash-to-Cash Cycle Time is infinite, the value of the metric is also infinite.

Caution: A value of infinite appears in a dialog as the symbol inf. However, a value of infinite appears in a report as 65,536 or 216. Be sure to take into account infinite numbers when interpreting results in Excel reports, for example, when averaging values over time.

Cash-to-Cash Cycle Time is relevant for a Distributor and Manufacturer role. It is not relevant for a Base Manufacturer or Consumer role.
