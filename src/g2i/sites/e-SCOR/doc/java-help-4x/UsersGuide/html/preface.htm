e-SCOR User's Guide
Topic - Welcome

Welcome

The online help provides all the information you need to model, run, and analyze your supply chain, using e-SCOR. The topics include:

Introduction

Concepts and terminology of e-SCOR and SCOR (Supply Chain Operations Reference), upon which e-SCOR is based.
Getting started with e-SCOR, which provides an introduction to creating models and running simulations.

Basics of Using e-SCOR

Creating a supply-chain model by configuring Level 1 roles, which are e-SCOR abstractions of Level 2 SCOR elements.
Configuring Level 2 SCOR parameters for the simplest e-SCOR model, which supplies a single product to a single buyer, based on orders.
Running a simulation and configuring how it runs.
Viewing metrics that the model computes while it runs.
Understanding what happens when a simulation runs by analyzing the processes that e-SCOR runs.
Working with models through the e-SCOR user interface.
Creating reports for viewing static and time-series e-SCOR metrics and for entering e-SCOR parameters.
Accessing databases by writing report data to a database and applying report data from a database.

Building e-SCOR Models

Modeling a distribution process, which delivers products to downstream buyers, either directly or by assembling components into kits.
Modeling a manufacturing process, which makes finished products from components.
Using stock planning strategies, which include replenishment and forecast planning for sourcing and making stock products.
Using alternative planning strategies, which include stock, made-to-order, and engineered-to-order for sourcing, making, and delivering products.
Configuring role details for sourcing, making, and delivering multiple products.
Modeling a process with multiple suppliers to source identical products, using contracts.
Using pull and push planning modes for sourcing and delivering products.
Using the Scenario Manager to run batch simulations and set parameter values during the simulation.

In addition, this online help provides definitions of:

Asset metrics formulas e-SCOR uses to compute asset metrics for Level 1 roles.
SCOR metrics for each e-SCOR metric and parameter that derives its definition from SCOR.
