Topic - Using Reports

Writing to and Reading from Excel CSV Files

For any output or input report in the model, you can:

Write output report data to an Excel CSV file.
Import input report data from an Excel CSV file.

You use CSV (comma separated values) files instead of Excel reports to import report data into a graphics program or to perform further analysis on the data.

You also use CSV files when using the Scenario Manager to run multiple simulations from a script. For more information, see Using the Scenario Manager.
