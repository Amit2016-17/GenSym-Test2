Topic - Using Pull and Push Planning Modes

Using Push Mode Planning

To use push mode planning, you configure:

The number of delivery products to push to downstream buyers and the number of source products that buyers accept.
The amount to push to multiple buyers, where each buyer receives a ratio of the pushed amount.
Which delivery products to push first, given a supplier that pushes multiple delivery products to downstream buyers.
The amount to push when components are shared among multiple delivery products.

The model computes how much is pushed for source and delivery products.

The only planning strategy available in push mode is stock, which behaves differently depending on the finished products being pushed and the number of downstream buyers.

For simplicity, the descriptions that follow use the terms "component" and "finished product" to refer to source and delivery products, respectively.
