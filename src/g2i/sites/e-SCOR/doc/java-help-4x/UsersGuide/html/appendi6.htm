Topic - A Asset Metrics Formulas

Inventory

The sum of the Raw Materials, Work in Progress, and Finished Goods Days of Supply metrics.

If any of these values is infinite due to a zero consumption, the Inventory Days of Supply metric ignores the infinite term. If all three of the values are infinite, the Inventory Days of Supply metric is also infinite.

Inventory Days of Supply is relevant for a Base Manufacturer, Distributor, and Manufacturer role. It is not relevant for a Consumer role.

Description

The value of the Raw Materials Days of Supply metric for the role.
