Topic - Creating a Supply-Chain Model

Configuring the Model Environment

You build and run your e-SCOR model by creating and configuring these objects:

Workspace - A blank area on which you create an e-SCOR model.
Model tool - A container in which you build your e-SCOR model.
Scenario tool - The discrete event simulation engine for controlling the simulation.
Organizer tool - A container in which you place various objects in your model.
Note: Every model must have a Scenario; otherwise, you cannot configure certain aspects of the model and you cannot run the simulation.

Models and organizers both define detail.
