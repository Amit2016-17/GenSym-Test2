Creating Free Form Reports

You might want to create an N Dimensional Output Report in which you configure the labels yourself. To do this, you create a Free Form Report and configure the cell location in the report of each attribute. You use a Free Form Report to create a summary table for viewing key metrics in the model.

When configuring the attributes to appear in a Free Form Report, you configure the Object Label or UUID, as well as the Attribute Group Name and the Attribute Name for each attribute to appear in the report. It is often easier to add attributes to a Free Form Report through the dialogs, in which case, e-SCOR automatically enters the UUID for the object, which uniquely identifies the object.

If you are configuring the list of visible attributes directly, you double-click the Object Label or UUID cell associated with each attribute and enter the label associated with the object whose attribute should appear in the report.

Note: The labels you enter must be unique and cannot contain carriage returns.

For example, to include the Supplier On-Time Performance (%) metric for two ES: Enable Source categories, you might configure the labels as ES: Enable Source 1 and ES: Enable Source 2.

Note: Because the labels associated with product composites are not unique across roles, you must choose the attributes of product composites through the dialogs.

To create a Free Form Report:

1 Display the Reports tab of the e-SCOR toolbar, select a Free Form Report, and place it on the model detail.
2 Display the properties dialog for the report object and configure the parameters on the General and Excel tab.
For details, see:
Updating Output Reports at Regular Time Intervals.
Keeping a History of Data Values.
Creating Reports in Excel.
3 Configure the attributes to appear in the report.
For details, see Configuring the Attributes to Appear in a Report.
Tip: You can use Ctrl + X, Ctrl + C, and Ctrl + V, respectively, to cut, copy, and paste values, as needed.
4 Configure the Sheet Cell for each attribute in the report, where the default cell is labeled A1.
The metric value appears in the report in the specified cell location, with no labels.
5 Create the report template, then configure the report template manually to add labels and other formatting, as desired.
For details, see Creating a Report Template in Excel.
6 Generate the output report data.
For details, see Generating Output Report Data from the Model to Excel.
