Topic - Working with Models

Navigating e-SCOR Models

You have a number of options for navigating e-SCOR models:

Using the tree-view browsers to display hierarchical views of models, workspaces, work objects, and modules.
Finding items in the model that match the specified criteria.
Going to a location in the server.
Going to an HTML page or RTF file on the file system.
