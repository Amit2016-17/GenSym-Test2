Controlling the Layout of Objects

To adjust the order of objects:

Select an object, then choose Layout > Bring to Front or Send to Back or click the equivalent toolbar button:

To reflect or rotate objects:

Select an object, choose Layout > Reflect or Rotate, then choose the desired action from the submenu.

To align objects:

Select two or more objects, choose Layout > Align, then choose the desired action from the submenu, or click the equivalent toolbar buttons:

To distribute objects:

Select three or more objects, then click the desired toolbar button to distribute the items horizontally or vertically:

To nudge an object up, down, right, or left:

Select an object, then hold down the Ctrl key while pressing the up, down, right, and left arrow keys to nudge the item in the desired direction, or click the equivalent toolbar buttons.

For information on the Shrink Wrap toolbar button on the Layout toolbar, see Shrink Wrapping Workspaces.
