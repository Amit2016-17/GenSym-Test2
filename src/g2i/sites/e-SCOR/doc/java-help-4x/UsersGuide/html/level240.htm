Order Selection

Sort Direction
Sort By

Cutoff Value
Cutoff Duration

Order

(D1) The criteria that determines how to sort orders for delivery. The Deliver category sorts orders waiting to be delivered, based on up to four criteria. You can sort orders, based on any numeric or time-based value of an order.

Initially, the Deliver category sorts orders, based on the criteria on the First tab. If two orders have the same value for the specified sort attribute, the Deliver category sorts orders, based on the criteria on the Second tab, and so on.

The D2: Deliver Make-to-Order Product category does not include order selection criteria. Instead, it chooses orders for delivery on a first-in, first-out basis.

The D3: Deliver Engineer-to-Order Product category does not include order selection criteria. Instead, the build selection criteria for the M3: Engineer-to-Order category determines the criteria the role uses to deliver engineer-to-order products.

For a description of the Order Selection parameters and how to use them, see Configuring Order Selection Parameters.

Order Shipment to Receipt Time

Mode Type
Min
Mode
Max

Transportation

The time it takes from when the Deliver category packs a container for shipment to when the buyer receives the container. By default, you specify the Min, Mode, and Max of a triangular function. The default values are 1 day.

For information on how to use other distributions, see Configuring the Mathematical Distribution.

For information on configuring this parameter through a Logistical Lookup Report, see Configuring Logistical Parameters through a Report.

Tip: Assets transfer from supplier to buyer at the end of the Order Shipment to Receipt Time and before the Receive and Putaway Time of the buyer role's Source category. When assets transfer from supplier to buyer, the In Transit metric of the buyer's source product increments. To cause assets to transfer at shipment time, configure the Receive and Putaway Time to be zero and configure the Order Shipment to Receipt Time to include the putaway time. The asset metrics for the supplier and buyer roles reflect this difference, as do the Supplier On-Time Performance and Delivery Performance (%) metrics of the ES: Enable Source and ED: Enable Delivery categories, respectively.
