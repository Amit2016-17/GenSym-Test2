Configuring the Make Category

The properties dialogs for the Make categories have these parameter differences, by tab:

Build Orders tab: The M1 category includes the Build Selection parameters, and the M1 and M2 categories include the Batch Preference parameter.
Build tab: The M1, M2, and M3 categories include the Production Material Cycle Time parameter.
Cost tab: The M1 and M2 categories do not include the ECO parameter, and the M1, M2, and M3 categories include the Production Material Handling cost parameter.
Engineering tab: The M1 and M2 categories do not include the Engineering tab.
