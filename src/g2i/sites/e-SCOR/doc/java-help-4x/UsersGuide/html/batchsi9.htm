Scenario and Model Parameter Keywords

These keywords set parameter values for Scenario and Model tools. The name of the keyword corresponds to the resource parameter to set.

You specify <label>, the <new-value> for the parameter and the <time>.

For example, the following keyword sets the Scenario Version parameter to 2 for the Scenario tool named my-scenario at the start of the simulation:

SET-SCENARIO-VERSION-PARAMETER
   (my-scenario,2,0)

SET-SCENARIO-VERSION-PARAMETER(<scenario-label>,
   <new-value>, <time>)

SET-MODEL-VERSION-PARAMETER(<model-label>, <new-value>, <time>)
