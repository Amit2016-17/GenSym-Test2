Intelligent Objects: Issues Aug 11, 2003

Amine Treaters

Dialogs

  • .In Configuration dialog, don’t use abbreviations for Ovhd.  Use Overhead.  Also spell out Temperature and Pressure.   

  • .In Service properties dialog, spell out Ovhd and Temp. 

  • .Amine Space Velocity properties dialog should say Lean Amine Flow to identify the type of Flow, not just Flow. 

  • .Delta Amine Flow Calculation properties dialog should say Lean Amine Flow and Rich Amine Flow to match the related sensor names. 

  • .Amine Loading Projected High and Low properties dialog should say Lean Amine Concentration, Lean Amine Temperature, and Rich Amine Temperature to match the sensor names. 

  • .Amine Ratio High and Low Limit properties dialog says Amine and Feed for the related sensors, but it should say Lean Amine Flow and Feed Flow to match the sensor names. 

  • .Amine Ratio High and Low Limit dialog title refers to Amine/Feed Ratio.  Should the event refer to this ratio too?  If not, the title should match the event. 

  • .Lean Amine Temperature Projected High dialog should say Lean Amine Temperature not Temperature to match the sensor name. 

  • .Level Flat-Line Detection and Swing Detection dialogs should say Treater Level not Level to match the sensor name. 

  • .Minimum Amine Flow sensor labels should match sensor names, as above.  Plus spell out Min Flow Factor. 

  • .Foaming event dialog should say Overhead KO Pot Level not Level to match the sensor. 

  • .HC Space Velocity should say Feed Flow not Flow. 

  • .Delta T event monitor doesn’t appear when you choose the menu choice. 

  • .Evaporative Loss dialog should say Overhead Temperature, Overhead Pressure, and Lean Amine Concentration to match sensors. 

  • .Feed Temperature High Limit dialog should say Feed Temperature to match the sensor. 

Amine Treater

  • .Many events need to have attributes added to the dialogs like Truth Tolerance, etc. 

  • .Most events don’t have event status values on the event monitor. 

  • .Lean Amine Temperature High Limit event properties dialog shows not-configured for the Temperature even though the sensor is configured.   

  • .Amine treaters have no event selection dialog like other objects.  This needs to be implemented at some point.
