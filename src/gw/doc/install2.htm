Downloading G2 WebLink by FTP

 G2 WebLink files can be downloaded by FTP, in a platform-specific archive format. You must download the files in binary mode.

The files are named following this convention:

<product><revision>.<extension>

where extension varies from platform to platform. Ask Gensym's product and licensing department for the download pathname.

The following table summarizes the formats, extensions, and names of the G2 WebLink files for the platforms where G2 WebLink is available.

Platform Format Extension G2 WebLink File Name
Unix


tar file


.tar


gw10r1.tar


Windows


PKZIP file


.zip


gw10r1.zip


VMS


Backup Save Set


.a


gw10r1.a


To download G2 WebLink, follow these steps:

  1. FTP to the Gensym server, using the following command: 

    > ftp ftp.gensym.com
    Login with the required name and password information.

  2. When the connection to FTP is established, go to the directory on the Gensym server that contains the files that you want to download. This directory will be a platform-specific subdirectory of the main product directory for G2 WebLink.

    Use a command of the following form to go to the product directory: 

    ftp> cd /private/shipping/path
    where path is the pathname of the product directory provided by Gensym.

  3. Set the FTP utility to download files in binary format, as follows: 

    ftp> binary
  4. Download G2 WebLink (the archive file), using the following command:

    ftp> get filename
    where filename is the filename provided by Gensym. The filename will use one of the following platform extensions:

    Platform Filename Extension
    Unix


    		tar


    VMS


    		a


    Windows


    		zip



    For example, on a Unix system: 

    ftp> get gw10r1.tar
    On VMS systems, you must include the /fdl flag, as follows: 

    ftp> get /fdl gw10r1.a
    You will be prompted to enter the name of the file on your local system (the local-file) where you want to store the file that you get.

  5. After you download the file, exit the FTP program: 

    ftp> quit
