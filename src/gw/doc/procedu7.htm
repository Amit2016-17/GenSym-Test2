gw-kill-bridge

Shuts down G2 WebLink.

Synopsis 

    gw-kill-bridge()

Description

The remote procedure gw-kill-bridge() shuts down the bridge. It takes no arguments returns no return value or signal.

Return Values

None.

Signals

None.
