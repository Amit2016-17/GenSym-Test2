Capabilities of G2 WebLink

G2 WebLink enables commonly available Web browsers, such as Netscape or Mosaic, to be used as client front ends for the information available within the G2 knowledge base.

Using Web browsers, a large number of people can access a G2 application on their desk top machines. People who want to make fuller use of the G2 application can convert to a Telewindows interface.

You use G2 WebLink to implement an HTTP server. You can configure G2 WebLink to satisfy requests that it receives from an HTTP client either by fetching a resource from the file system or by passing the request to G2, which can then generate appropriate responses.

Software Components

G2 WebLink consists of the following software components:

  • A G2 WebLink executable

  • G2 Knowledge Bases that provide useful utilities as well as examples that illustrate how G2 WebLink can be used

G2 WebLink also requires a browser such as Netscape Navigator or Mosaic to use for testing.

Note: Netscape is the recommended browser for viewing the online G2 WebLink documentation.
