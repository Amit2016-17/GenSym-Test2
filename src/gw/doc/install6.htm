Running G2 WebLink

 G2 WebLink is launched at the command line. It can take an optional argument specifying the TCP/IP port number where it listens for G2 to connect. If you do not specify a port number argument, it listens on the default port 7736.

G2 WebLink will decline to run if it does not find licensing code specific to the CPU it is running on.
