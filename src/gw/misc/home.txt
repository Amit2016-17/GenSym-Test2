G2 WebLink Home Page
********************

@b(Welcome!)

This page provides a site local home page for the G2 WebLink
installation at your site.  This page links back to
@l("http://www.gensym.com/", Gensym Corporation) and into the
documentation for G2 WebLink

G2 WebLink has these principle components.

- an executable - named gw on unix and gw.exe on VMS and NT).
- gwlow.kb - a required kb module.
- two example kbs
  - gwex1.kb - uses G2 WebLink to fetch resources from the Web.
  - gwex2.kb - uses G2 WebLink to serve resources to the Web.
- a @l("titlepag.htm", user's manual) [@l("booktoc.htm", table of contents)]
- a list of @l("bugs.htm", known bugs).
