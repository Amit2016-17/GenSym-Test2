Simple Updating Data Example

This is a simple example of dynamically updated data feeds between WebLink and Java. The applet is supplied a URL to the data source and a refresh rate.

Note: Due to yet-to-be-determined bugs in certain Java implementations you may need to do a "Reload" to jumpstart the updates. Go figure.

This applet was instantiated with the following HTML code: 

<applet code="SimpleText.class" codebase=applets width=850 height=75>
<param name=refreshRateInMS value="3000">
<param name=dataSource value="/data/random-quote.txt">
<param name=fontname value="Courier">
<param name=fontslant value="italic">
<param name=fontweight value="bold">
<param name=fontsize value="24">
</applet>
